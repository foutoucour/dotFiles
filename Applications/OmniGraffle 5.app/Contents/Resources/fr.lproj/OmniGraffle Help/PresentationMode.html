Mode Présentation

Le mode de présentation est disponible à partir du menu Présentation ; il remplit tout l’écran de votre document pour que vous puissiez le présenter publiquement.
Chaque canevas se comporte comme une diapo ; utilisez les touches fléchées pour vous déplacer d’un canevas à l’autre. Vous pouvez également appuyer sur la touche Retour ou Entrée ou cliquer afin de passer au canevas suivant. Si vos préférences Présentation sont réglées de manière à mettre en surbrillance un objet lorsque vous cliquez dessus, le fait de cliquer ne vous fera donc pas passer au canevas suivant ; vous pouvez toujours cliquer sur une zone vide du canevas pour passer au suivant. Si vous déplacez le curseur de la souris en bas de l’écran, une interface de navigation apparaît, avec des boutons pour aller en avant ou en arrière, quitter ou aller directement à un canevas spécifique.
Vous pouvez mettre des objets en surbrillance pour y attirer l’attention pendant la présentation. Utilisez les préférences Présentation pour définir le mode de surbrillance.
Pour quitter le mode présentation, appuyez sur Escape ou cliquez sur le bouton représentant une croix qui apparaît lorsque vous déplacez le pointeur de la souris dans le bas de l’écran.
Les actions configurées dans l’inspecteur Action fonctionnent lorsque vous êtes en mode de présentation. Lorsque vous cliquez sur un objet auquel aucune action n’est attribuée, il exécute une action comme si vous aviez cliqué dessus à l’aide de l’outil de navigation.
Préférences Présentation
Dans le mode de présentation, vous avez la possibilité de mettre des objets en surbrillance sur le canevas.
Choisissez si un objet doit ou non être mis en surbrillance si vous placez le pointeur dessus, si vous cliquez dessus ou ne jamais le mettre en surbrillance. Vous pouvez également choisir d’afficher un en-tête pour les objets disposant d’actions configurées dans l’inspecteur Action.
Cliquez sur le cadre des couleurs afin de choisir une couleur pour la surbrillance. Une bonne astuce consiste à utiliser le mot clé selectedControlColor à partir de la palette du développeur ; cette couleur correspond toujours à la couleur de surbrillance sélectionnée dans les préférences Apparence de Mac OS X. Utilisez le curseur pour régler l’épaisseur de cette surbrillance.
Si plusieurs fenêtres sont ouvertes, c’est celle du premier plan qui est utilisée pour la présentation. Si vous disposez de deux moniteurs, vous pouvez placer la présentation sur l’un des écrans pendant que vous modifiez le document sur l’autre.
Sélectionnez Nouvelle fenêtre "Document" (où Document correspond au titre de votre document) dans le menu Fenêtre.
Faites glisser une fenêtre sur le second moniteur.
Cliquez sur la fenêtre placée sur le second moniteur pour vous assurer qu’elle est bien au premier plan.
Entrez en mode de présentation. Le second moniteur affiche la présentation alors que le premier conserve la fenêtre habituelle du document.
Vous pouvez désormais modifier le document sur le premier moniteur ; la présentation sur le second est actualisée automatiquement.
