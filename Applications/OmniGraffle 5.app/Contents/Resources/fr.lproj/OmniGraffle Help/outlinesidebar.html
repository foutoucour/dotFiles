Utilisation de la barre latérale de structure pour modifier un schéma hiérarchique

La barre latérale de structure vous offre un moyen rapide de saisir de nombreuses données d’une part, et une autre façon de visualiser des schémas hiérarchiques d’autre part. Pour l’afficher, choisissez Structure dans le menu Présentation. Si la barre latérale de liste est ouverte, vous pouvez également cliquer sur le bouton dans la partie supérieure pour passer à la barre latérale de structure.
Cette présentation affiche toutes les formes que vous avez créées dans le canevas, disposées hiérarchiquement selon les lignes qui les relient. De même, lorsque vous tapez du texte dans la structure, les formes sont automatiquement créées et connectées sur le canevas avec le texte correspondant.
Pour commencer la manipulation dans la structure, il vous suffit de cliquer sur un élément ou de double-cliquer sur une zone vide afin de créer un élément. Dans ce mode, vous pouvez utiliser les touches suivantes pour créer et disposer les éléments entre eux :
Appuyez sur la touche Retour pour créer un élément « pair » de celui sélectionné. Les pairs apparaissent au même niveau d’indentation dans la structure et, s’ils se trouvent au-delà du niveau supérieur, des connexions à partir du même objet parent dans le schéma leur sont établies.
Appuyez sur la touche Tab pour mettre en retrait l’élément sélectionné, le transformant ainsi en élément « enfant » de l’élément se trouvant immédiatement au-dessus dans la structure. Une ligne de connexion est alors tracée dans le schéma à partir de l’objet parent vers l’objet enfant.
Appuyez sur Maj + Tab pour supprimer un niveau de retrait de l’élément sélectionné, le changeant ainsi en pair de ce qui était auparavant son parent.
Appuyez sur Échap pour alterner entre la sélection d’un élément et la modification de son texte.
Il existe également des raccourcis clavier spéciaux pour construire rapidement des structures. Consultez le fichier Raccourcis clavier dans le menu Aide pour obtenir des détails.
Vous pouvez faire glisser un élément dans la présentation Structure en utilisant sa poignée, c’est-à-dire le point ou le triangle situé à gauche de chaque élément. Si vous faites glisser un élément ayant des enfants, ces derniers se déplacent avec. Si vous faites glisser un élément, une ligne noire apparaît pour indiquer l’emplacement et le niveau de retrait que l’élément occupera lorsque vous le déposerez.
Un groupe, un tableau ou un sous-graphique du canevas s’affiche comme un seul élément entre crochets dans la structure. Cliquez sur les flèches à droite de l’élément pour voir apparaître les objets qui y sont inclus et modifier leur texte le cas échéant.
Le menu local Action sous la barre latérale de structure offre un accès rapide aux commandes dans le sous-menu Élaborer la structure du menu Édition, ainsi qu’aux commandes de gestion des styles de schémas.
Si vous avez choisi un style de schéma pour le canevas, des objets dans le schéma sont créés pour correspondre au style de schéma. Un aperçu du style de schéma sélectionné s’affiche au-dessus du menu local. Si vous modifiez le style du schéma, les objets déjà placés sur le canevas se modifient en conséquence pour suivre les propriétés du nouveau style sauf si leur style a été au préalable modifié manuellement. Vous pouvez forcer manuellement les objets disposant d’un style à se conformer au style du schéma en les sélectionnant et en utilisant la commande Format ‣ Réappliquer le style aux objets sélectionnés ou vous pouvez forcer tous les objets du canevas à se conformer au style en choisissant Format ‣ Appliquer de nouveau le style de schéma.
Alors que la présentation Structure est pratique pour entrer rapidement de nombreuses informations, une structure ne peut pas toujours représenter parfaitement les relations d’un schéma complexe. Les objets ayant plusieurs parents, les lignes connectées à d’autres lignes et l’ascendance circulaire (cas de figure de type « Je suis mon propre grand-père ! ») sont autant d’exemples de cas où une structure simple ne peut correctement suffire. Dans ce genre de situation, le mode structure tente d’être le plus précis possible. Si votre schéma n’est pas une arborescence au sens strict du terme, il est alors déconseillé de se fier totalement à la présentation Structure.
