Commandes DOT importables
OmniGraffle peut importer des documents à l'aide d'un sous-ensemble du format de fichier DOT de GraphViz.
Commande Type d'entité
URL (hors graphiques) bord/ligne, nœud/forme
arrowhead bord/ligne
arrowtail bord/ligne
color bord/ligne, nœud/forme
dir bord/ligne
fillcolor nœud/forme
fixedsize nœud/forme
fontcolor bord/ligne, nœud/forme
fontname bord/ligne, nœud/forme
fontsize bord/ligne, nœud/forme
headlabel bord/ligne
label bord/ligne, nœud/forme
labelfontcolor bord/ligne
labelfontname bord/ligne
labelfontsize bord/ligne
orientation nœud/forme
rank nœud/forme
rankdir graphique/page
shape nœud/forme
style bord/ligne, nœud/forme
taillabel bord/ligne
width nœud/forme
Tous les types de pointes de flèche sont pris en charge, mais certains ne correspondent pas exactement aux types de flèches d'OmniGraffle. Les types de flèches inv, invdot, invodot et invempty sont remplacés par la fin de ligne OmniGraffle la plus proche disponible.
