Licences

Sélectionnez l’élément de menu Licences dans le menu OmniGraffle pour afficher la zone de dialogue Licences. Vous pouvez y consulter et y modifier vos licences OmniGraffle qui attestent que vous avez bien acheté le logiciel. (À ce sujet, merci beaucoup !)
Pour acquérir une licence :
Cliquez sur le bouton Acquérir des licences et vous serez renvoyé instantanément sur la page du magasin en ligne d’Omni Group. Naturellement, cela ne fonctionne que si vous êtes connecté à Internet.
Pour ajouter une nouvelle licence :
Cliquez sur le bouton Ajouter une licence.
Si vous possédez une clé de licence reçue dans un courriel, copiez puis collez le nom du propriétaire et la clé de licence dans les champs appropriés. Si vous ne pouvez pas copier et coller les informations, vous devrez les saisir ; assurez-vous de copier toutes les informations comme elles apparaissent dans votre message de licence. Assurez-vous également que la licence que vous saisissez correspond réellement à une licence OmniGraffle 5 et que l’édition est correcte (édition professionnelle ou standard). Les licences destinées à d’autres produits Omni Group ou à des versions antérieures d’OmniGraffle ne sont pas valides.
Utilisez le menu déroulant pour sélectionner le type de licence que vous souhaitez utiliser. (Plus loin, vous retrouverez des informations complémentaires sur les types de licences.)
Cliquez sur le bouton Enregistrer.
OmniGraffle possède deux types différents de licences:
Personnelle : cette licence est destinée à un usage personnel et n’est disponible que si vous ouvrez une session sur cet ordinateur sous l’identité de l’utilisateur qui a installé cette licence. Vous pouvez installer cette licence sur plusieurs ordinateurs mais uniquement dans le but d’un usage rattaché à la personne en question. Vous n’êtes pas autorisé à utiliser le logiciel sur deux ordinateurs différents en même temps par le biais de ce type de licence. Si une licence personnelle est disponible, l’application l’utilise systématiquement.
Ordinateur : vous pouvez utiliser cette licence sur cet ordinateur ; il en va de même pour tous les utilisateurs de cet ordinateur. Cependant, vous ne pouvez pas installer cette licence sur plusieurs ordinateurs.
Pour supprimer une licence :
Sélectionnez la licence que vous voulez supprimer.
Cliquez sur le bouton Retirer une licence.
Pour déplacer une licence d’un Mac vers un autre :
Tout d’abord, recherchez le fichier de licence sur l’ordinateur source. Les fichiers de licence d’OmniGraffle sont situés dans le dossier Bibliothèque/Application Support/Omni Group/Software Licenses, où Bibliothèque se trouve soit au niveau supérieur de votre disque système (pour les licences d’ordinateurs) ou dans votre dossier de départ (dans le cas de licences personnelles). Le fichier que vous recherchez possède un nom se présentant sous la forme omnigraffle-123456.omnilicense. Transférez ce fichier sur un autre ordinateur.
Sur l’autre ordinateur, choisissez Licences dans le menu OmniGraffle puis cliquez sur le bouton Ajouter une licence.
Sur la page qui s’affiche, cliquez sur le bouton Importer fichier de licence.
Sélectionnez le fichier de licence que vous souhaitez importer.
Cliquez sur le bouton Ouvrir.
