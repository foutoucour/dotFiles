Utilisation des styles de schémas

Les styles de schémas sont des documents OmniGraffle qui définissent une apparence que vous pouvez appliquer à une structure ou à un schéma. Il existe une infinité de manières de représenter les mêmes données ; un style de schéma vous permet d’indiquer la manière exacte dont vous voulez représenter les données.
Les styles de schémas se révèlent utiles lorsque vous dessinez une structure à l’aide de la présentation Structure d’OmniGraffle (via la commande Choisir un style de schéma du menu Format) ou lorsque vous importez un fichier OmniOutliner (à travers la zone de dialogue d’importation). C’est dans ces cas que le style de schéma traduit les éléments texte en objets visibles. Plusieurs styles de schémas sont fournis avec OmniGraffle ; testez la présentation Structure avec différents styles de schémas pour en connaître les effets.
Lorsque vous appliquez un style de schéma, les éléments de la structure sont des styles qui correspondent aux réglages de formes, de lignes de connexion et de mise en page automatique du style de schéma. Les éléments à chaque niveau de la structure deviennent des formes correspondant aux objets du même niveau du schéma. Si la structure possède plus de niveaux que le style de schéma, les éléments du niveau inférieur utilisent alors les styles définis au niveau inférieur du style de schéma.
Pour créer votre propre style de schéma, choisissez Nouvelle ressource ▸ Nouveau style de schéma dans le menu Fichier, puis choisissez un modèle sur lequel vous basez. Créez un schéma composé d’un canevas et d’un calque contenant des objets interconnectés, du style de votre choix. Un style de schéma doit se présenter obligatoirement sous la forme d’une structure arborescente, c’est-à-dire que les lignes de connexion doivent créer une hiérarchie sans revenir sur elles-mêmes. Une fois terminé, enregistrez-le. L’emplacement d’enregistrement par défaut est le dossier Application Support qui vous est attribué, dans lequel OmniGraffle recherche vos styles de schémas. Une fois que le modèle est enregistré à cet emplacement, il devient disponible dans le sélecteur de styles de schémas.
Si vous prévoyez d’importer des fichiers OmniOutliner comportant plusieurs colonnes, vous pouvez créer un style de schéma avec un groupe de formes plutôt qu’une seule forme pour représenter chaque élément. Dans ce cas, il est conseillé d’ajouter une étiquette de texte sur tous les objets afin de pouvoir y associer les colonnes lors de l’ouverture du fichier OmniOutliner.
