Insertion de graphiques dans une forme à l’aide de l’inspecteur Image

Vous pouvez utiliser cet inspecteur pour positionner une image à l’intérieur d’une forme. (Au lieu de créer un objet puis d’ajouter une image, vous pouvez aussi tout simplement faire glisser un fichier image à partir d’une autre application directement dans le canevas.)
Utilisez le menu local Image pour définir une image dans la forme sélectionnée. Lorsqu’une image est définie, une commande Supprimer l’image apparaît dans le menu.
Lorsque vous recherchez le fichier d’une image, vous avez la possibilité de créer un alias (ou « lien ») référençant l’image au lieu d’ajouter l’image au fichier OmniGraffle même. Dans ce cas, le document OmniGraffle s’appuie sur le fichier image d’origine ; ainsi, l’alias ne fonctionne pas si vous supprimez le fichier image du disque ou si vous ouvrez le document sur un autre ordinateur. Si l’image d’un objet provient d’un alias, des commandes supplémentaires deviennent disponibles dans le menu local Image : vous pouvez Ouvrir le fichier image d’origine, Afficher le fichier d’origine dans le Finder ou ne plus utiliser l’alias et Intégrer une copie de l’image dans le document OmniGraffle.
Pour définir une image, vous pouvez également faire glisser un fichier image à partir d’une autre application sur le cadre d’image ou sur l’objet de la forme.
Le premier bouton affiche l’image dans ses proportions naturelles, avec un niveau de zoom et une position définis à l’aide des contrôles ci-dessous. Le second bouton permet d’étirer l’image pour qu’elle s’adapte à l’objet. Le troisième bouton permet de créer un nombre indéfini de mosaïques de l’image partout sur l’objet.
Les décalages de positionnement sont activés si vous affichez l’image dans ses proportions naturelles, comme défini par le premier des trois boutons ci-dessus. Vous pouvez saisir un pourcentage positif ou négatif ou simplement faire glisser l’image dans le cadre d’image pour indiquer la zone que vous voulez.
Les contrôles de zoom sont disponibles sauf si vous étirez l’image pour qu’elle s’adapte à la forme. Vous pouvez saisir un pourcentage de zoom ou simplement faire glisser le curseur, pour agrandir ou réduire l’image.
Vous pouvez rendre l’image transparente et permettre au remplissage de l’objet de laisser deviner ce qui se trouve derrière en diminuant la valeur du champ opacité ou en faisant glisser le curseur d’opacité.
Vous pouvez également ajouter une image à un canevas. Pour cela, cliquez sur le canevas dans la barre latérale (pour le mettre en surbrillance selon votre couleur de sélection), puis utilisez l’inspecteur Image normalement.
