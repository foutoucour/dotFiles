Linkback

La technologie LinkBack constitue une méthode d’intégration de contenu entre divers applications de façon à ce que vous puissiez toujours revenir et modifier à volonté le contenu en question directement dans le logiciel qui en est à l’origine.
Pour des informations complètes sur la technologie LinkBack et connaître la liste des applications qui la prennent en charge, rendez-vous sur le site Web à l’adresse www.linkbackproject.org (site en anglais).
Pour intégrer du contenu OmniGraffle à autre application compatible LinkBack, copiez puis collez simplement le contenu comme d’habitude. Il en va de même pour le contenu intégré à des schémas OmniGraffle provenant d’autres applications.
Pour intégrer du contenu OmniGraffle dans un seul objet d’OmniGraffle, choisissez Copier en tant que PDF dans le menu Édition, puis collez normalement. Les objets que vous avez copiés se transforment alors en une seule image ; vous pouvez cliquer deux fois sur l’objet pour modifier le contenu d’origine.
Pour modifier du contenu OmniGraffle intégré à une autre application :
Double-cliquez sur le contenu OmniGraffle placé dans le document de l’application externe. Une fenêtre OmniGraffle s’affiche alors incluant le contenu en question.
Modifiez le contenu dans la fenêtre OmniGraffle, comme vous le feriez pour un schéma habituel.
Choisissez Enregistrer dans le menu Fichier d’OmniGraffle ; le contenu de l’autre application s’actualise automatiquement.
Pour modifier le contenu d’applications compatibles LinkBack intégré à un schéma OmniGraffle, il vous suffit de double-cliquer dessus. Au moment de l’enregistrement de l’élément à partir de l’application externe, le contenu intégré au schéma OmniGraffle s’actualise de la même manière.
