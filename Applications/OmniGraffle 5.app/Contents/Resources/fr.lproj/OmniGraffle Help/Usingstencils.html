Utilisation de patrons pour conserver et partager des objets utilisés couramment

Un patron est un ensemble d’objets utiles que vous pouvez glisser dans vos schémas. Pour afficher vos patrons, sélectionnez Patrons dans le menu Fenêtre.
Cliquez sur ces boutons pour afficher les objets à mesure qu’ils apparaissent sur le patron ou séparés dans un cadre chacun.
Utilisez le menu local Action pour créer ou modifier des patrons, ou encore personnaliser leur liste ; reportez-vous aux paragraphes suivants pour en savoir plus sur ces commandes.
Saisissez des mots dans le champ de recherche pour rechercher des patrons dont les noms correspondent ou des patrons contenant des objets (dont le texte ou les notes correspondent). Vous pouvez même rechercher en ligne pour rechercher des patrons créés par d’autres personnes. Consultez les paragraphes ci-dessous pour en savoir plus sur la recherche de patrons.
Sélectionnez un patron dans la liste pour afficher son contenu ; sélectionnez tout un dossier pour afficher le contenu de tous les patrons qu’il contient.
Les patrons que vous sélectionnez apparaissent ici ; faites glisser les objets apparaissant ici vers votre document. Si un patron est ouvert mais n’est pas encore installé, cliquez sur le bouton Installer qui apparaît dans la barre de titre pour le copier dans votre dossier de départ et le garder en sécurité.
Lorsque vous sélectionnez un objet, certaines informations le concernant, y compris les notes ou les données personnalisées le concernant, apparaissent ici dans la sous-fenêtre des métadonnées.
Si vous ne voulez pas de l’ensemble d’un objet, mais simplement d’un ou de tous ses styles, faites glisser un style du tiroir des styles vers un objet de votre document.
Cliquez sur ce bouton pour afficher ou masquer la sous-fenêtre des métadonnées.
Pour placer un objet sur un canevas, faites-le simplement glisser à partir du patron et déposez-le là où vous voulez. Une copie de l’objet est effectuée et l’original est conservé sur le patron ; vous pouvez donc prendre autant de copies que vous le souhaitez. Les mêmes types d’astuces de sélection que sur un canevas sont possibles, comme cliquer tout en appuyant sur Commande ou faire glisser un rectangle afin de sélectionner plusieurs objets.
La liste des patrons est divisée en plusieurs sections.
Votre Bibliothèque de patrons inclut tous les patrons que vous avez installés, qu’ils soient intégrés à OmniGraffle ou présents dans un dossier Application Support.
La liste Ouvrir le patron récent répertorie les cinq derniers patrons que vous avez affichés. Si vous ouvrez un patron qui n’est pas encore installé, il apparaît dans la liste des patrons récents ; vous pouvez cliquer sur le bouton Installer à côté de son titre pour le copier à droite, dans votre dossier de départ.
La section Patrons favoris est entièrement personnalisable. Vous pouvez y faire glisser tous les patrons installés que vous voulez. Choisissez Nouveau dossier Patrons favoris dans le menu local Action pour créer un dossier pour organiser vos favoris.
Pour rechercher vos patrons, saisissez votre recherche dans le champ correspondant au-dessus de la liste des patrons. OmniGraffle recherche les mots saisis dans vos titres de patrons, dans le texte des objets contenus dans chaque patron, dans les notes et les données personnalisées. Les patrons qui correspondent à la recherche apparaissent dans la section Résultats de la recherche spéciale de la liste des patrons. Normalement, si l’objet d’un patron correspond aux termes de la recherche, tous les objets de ce patron apparaissent dans les résultats. Mais si vous activez le réglage Filtre du menu local du champ de recherche, alors les objets qui ne correspondent pas à votre saisie disparaissent.
Pour rechercher le référentiel de patron OmniGraffle en ligne, cliquez sur le menu local du champ de recherche, puis choisissez Graffletopia. Ce service gratuit proposé par Patrick Crowley permet à tous ceux qui le souhaitent de partager des patrons très pratiques. Lorsque vous effectuez une recherche dans Graffletopia, des patrons dont les noms et les descriptions correspondent apparaissent dans la liste Résultats de la recherche. Cliquez sur un patron pour afficher un aperçu et une description, puis cliquez sur le bouton Télécharger pour en obtenir une copie. Ouvrez le fichier qui s’affiche dans le Finder pour utiliser ou installer le patron normalement. Vous pouvez également accéder à Graffletopia à partir du menu Aide d’OmniGraffle ou en vous rendant directement sur http://graffletopia.com.
Pour créer un patron, choisissez Nouvelle ressource ▸ Patron dans le menu Fichier. Vous pouvez modifier le document qui apparaît comme vous le feriez pour n’importe quel schéma OmniGraffle normal. Un aperçu du patron s’affiche dans la fenêtre du patron tout au long de vos manipulations. Lorsque le patron vous satisfait, choisissez Enregistrer dans le menu Fichier ; l’emplacement par défaut pour l’enregistrer est le dossier Application Support dans votre Bibliothèque.
Pour modifier un patron, sélectionnez-le dans la liste, puis choisissez Modifier une copie du patron (ou Copier et modifier le patron, s’il est intégré) dans le menu Action ci-dessus. Puis, modifiez et enregistrez le patron comme n’importe quel document OmniGraffle normal.
