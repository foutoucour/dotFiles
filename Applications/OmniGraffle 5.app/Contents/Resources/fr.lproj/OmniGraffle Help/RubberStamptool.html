Reproduction des objets à l’aide de l’outil Timbre caoutchouc

Utilisez cet outil pour reproduire des objets. La première fois que vous ouvrez OmniGraffle, l’outil Timbre caoutchouc est vide et le point se trouvant sur sa face est creux. Sélectionnez-le et cliquez sur n’importe quel objet du canevas pour mettre l’objet dans le timbre ; le point se remplit alors. Une fois que le timbre contient un objet, chaque clic sur le canevas crée une copie de l’objet.
Pour réinitialiser le contenu d’un timbre, sélectionnez l’option Effacer le contenu dans le menu local de l’outil ; vous pouvez ensuite saisir un nouvel objet. Vous avez également la possibilité de cliquer sur un autre objet tout en maintenant la touche Option enfoncée pour remplacer le contenu du timbre.
