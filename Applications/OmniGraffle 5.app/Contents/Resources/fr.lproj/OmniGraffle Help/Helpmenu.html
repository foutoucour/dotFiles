Menu Aide

Aide OmniGraffle : cette option appelle l’aide actuellement sur votre écran. Nous vous remercions de votre intérêt pour la documentation.
Raccourcis clavier : retrouvez un document qui répertorie tous les raccourcis clavier disponibles dans OmniGraffle pour éviter le syndrome de la souris.
Nouveautés de la version 5 / Fonctions supplémentaires de la version Pro : ces commandes vous dirigent vers le site Web d’OmniGraffle et vous présenter les avantages qu’offre OmniGraffle Professional 5.
Visitez Graffletopia : consultez le site Web de Graffletopia, site tiers (non sollicité par Omni, mais quelle bonne idée !) de la communauté Omni pour le partage de patrons OmniGraffle de tous types.
Envoyer un commentaire : choisissez cette option pour envoyer un courriel à l’équipe d’assistance OmniGraffle. C’est le meilleur moyen pour nous contacter ; votre numéro de version et vos informations de licence sont entrés automatiquement dans le champ de l’objet du courriel. Par votre geste, vous nous aidez à vous aider à tirer le meilleur parti d’OmniGraffle.
