Menu Édition

Annuler : OmniGraffle mémorise la liste des modifications que vous apportez à un document. Choisissez cette commande pour reculer d’une étape de modification, comme si vous n’aviez pas effectué la dernière action mentionnée dans la liste. Choisissez l’option à plusieurs reprises afin de revenir sur les dernières actions effectuées.
Rétablir : si la dernière action que vous avez effectuée consistait à annuler une action, vous pouvez rétablir cette action en « annulant l’annulation ».
Couper : supprimez la sélection en cours et placez-la dans le presse-papiers pour pouvoir la coller à un autre endroit.
Copier : copier la sélection en cours dans le presse-papiers.
Copier en tant que
PDF : placez une copie de la sélection en cours au format PDF dans le presse-papiers pour la coller dans toute application prenant en charge ce format. Si l’application dans laquelle vous la collez prend également en charge LinkBack (OmniGraffle inclus), vous pouvez même revenir en arrière et modifier le contenu collé dans OmniGraffle.
TIFF : placez une copie de la sélection en cours au format TIFF dans le presse-papiers pour la coller dans toute application prenant en charge ce format.
PICT vectoriel : placez une copie de la sélection en cours au format PICT vectoriel dans le presse-papiers pour la coller dans toute application prenant en charge ce format (particulièrement utile pour coller dans Microsoft Word, qui présente un bogue l’empêchant de récupérer les données correctement lorsque vous collez du contenu OmniGraffle à l’aide de la commande Copier ordinaire).
AppleScript : placez une représentation AppleScript des objets sélectionnés dans le presse-papiers. Vous pouvez coller le résultat dans l’Éditeur de scripts ou dans l’inspecteur Actions d’un autre objet pour élaborer un script qui recrée ces objets. Le résultat permet de se limiter en premier à analyser le mode de fonctionnement de la prise en charge d’AppleScript par OmniGraffle.
Coller : placez le contenu du presse-papiers sur le canevas, au dernier endroit où vous avez cliqué à l’aide de l’outil Sélection. Ou bien insérez du texte si vous modifiez le texte d’un objet grâce au texte inclus dans le presse-papiers. Si ce dernier contient une image ou du texte, une nouvelle forme comportant l’image ou le texte est alors créée.
Coller et appliquer le style actuel : lorsque vous modifiez du texte, vous pouvez utiliser cette commande pour y coller le texte sans l’accompagner du style qui lui était appliqué dans l’application à partir de laquelle vous l’avez copié. Il applique alors le style du texte que vous avez déjà, comme si vous l’aviez saisi à cet emplacement, vous-même.
Dupliquer : créez une copie de la sélection en cours et placez-la sur le canevas sans passer par le presse-papiers.
Cette commande possède une fonctionnalité particulière :
Sélectionnez un élément sur le canevas et dupliquez-le.
Utilisez l’outil Sélection pour déplacer la copie ; OmniGraffle assimile la distance et la direction de la copie depuis l’original.
Dupliquez de nouveau autant de fois que vous le souhaitez ; chacune des nouvelles copies possède la même distance et la même direction que la copie précédente.
Supprimer : éliminez définitivement la sélection en cours (vous pouvez la récupérer via la commande Annuler).
Tout sélectionner : sélectionnez tout ce qui peut l’être sur le canevas. N’oubliez pas que les objets des calques verrouillés ne peuvent pas être sélectionnés. Si vous modifiez le texte d’un objet, cette commande sélectionne à la place tout le texte de cet objet.
Sélectionner
Objets similaires : sélectionnez tous les objets du canevas présentant les mêmes attributs de style que les objets sélectionnés.
Objets connectés : sélectionnez toutes les formes qui sont connectées directement ou indirectement par des lignes aux objets sélectionnés, ainsi que tous les traits entre eux.
Ancêtres : sélectionnez tous les ancêtres des objets sélectionnés, suivant la hiérarchie établie par les lignes de connexion.
Descendants : sélectionnez tous les descendants des objets sélectionnés, suivant la hiérarchie établie par les lignes de connexion.
Élaborer la structure : les commandes de ce sous-menu (en particulier leurs raccourcis clavier) permettent de travailler rapidement en mode de présentation Structure. Elles ne sont disponibles que si vous avez sélectionné un ou des éléments lorsque ce mode d’affichage est activé.
Modification sans souris : ces commandes sont particulièrement utiles lorsque vous vous servez des raccourcis clavier. Vous pouvez effectuer de nombreuses opérations sans toucher la souris.
Sélectionner la forme : ces commandes sélectionnent une forme d’après sa position par rapport à la forme sélectionnée.
Créer une forme : ces commandes créent une forme d’après la forme sélectionnée.
Canevas
Nouveau canevas : ajoutez un nouveau canevas au document.
Dupliquer le canevas : ajoutez un nouveau canevas au document, identique au canevas actif.
Supprimer le canevas : supprimez entièrement le canevas actif.
Calques
Nouveau calque : ajoutez un nouveau calque au canevas actif.
Nouveau calque partagé : ajoutez un nouveau calque partagé au canevas actif.
Dupliquer le calque : ajoutez un nouveau calque au canevas, identique au calque sélectionné.
Fusionner les calques en dessous : déplacez tout le contenu du calque actif sur le calque suivant situé en dessous, puis supprimez le calque sélectionné.
Supprimer le calque : supprimez entièrement le calque actif. Vous ne pouvez pas supprimer le dernier calque dans un canevas.
Supprimer toutes les copies : cette commande est uniquement disponible lorsque vous sélectionnez un calque partagé. Elle permet de supprimer tous les instances du calque partagé sur n’importe quel canevas dans l’ensemble du document.
Réglages de calques : ces commandes sont les répliques des icônes cliquables sur un calque dans la barre latérale des canevas. Visible affiche ou masque les objets du calque, Imprimable contrôle si les objets du calque doivent être inclus lors de l’impression et Déverrouillé contrôle si les objets du calque peuvent être sélectionnés ou modifiés.
Tout sélectionner dans le calque : sélectionnez tous les objets du calque sélectionné.
Déplacer la sélection vers le calque : déplacez tous les objets sélectionnés vers le calque sélectionné.
Tableaux
Insérer une ligne : lorsqu’une cellule d’un tableau est sélectionnée, choisissez cette commande pour ajouter une nouvelle rangée de cellules au tableau. La nouvelle rangée apparaît juste avant celle qui contient la cellule sélectionnée.
Insérer une colonne : lorsqu’une cellule d’un tableau est sélectionnée, choisissez cette commande pour ajouter une nouvelle colonne de cellules au tableau. La nouvelle rangée apparaît juste avant celle qui contient la cellule sélectionnée.
Sélectionner la ligne : lorsqu’une cellule d’un tableau est sélectionnée, choisissez cette commande pour sélectionner toute la rangée qui contient la cellule.
Sélectionner la colonne : lorsqu’une cellule d’un tableau est sélectionnée, choisissez cette commande pour sélectionner toute la colonne qui contient la cellule.
Aimants
Copier les aimants : placez l’aimant de disposition de l’objet sélectionné sur le presse-papiers, pour pouvoir le coller à un autre objet.
Coller les aimants : si le presse-papiers contient une disposition d’aimants, appliquez-la à l’objet sélectionné. Tous les aimants existants sont remplacés par les nouveaux.
Supprimer les aimants : supprimez tous les aimants des objets sélectionnés.
Formes
Formes de soustraction : utilisez les formes sélectionnées se trouvant au premier plan pour « percer un trou » dans la forme voulue se trouvant au dernier plan. Vous pouvez réordonner les formes grâce aux commandes Avancer/Reculer du menu Disposition.
Formes d’union : associez au moins deux formes sélectionnées en supprimant les lignes internes s’entrecoupant pour créer une forme d’une seule couleur.
Formes d’intersection : créez une forme à partir des zones de formes sélectionnées se chevauchant.
Rechercher
Rechercher : cette option ouvre le panneau Rechercher d’OmniGraffle. Il vous permet ainsi de rechercher une chaîne de caractères en particulier ou d’utiliser une expression pour trouver toute une catégorie de chaînes de caractères. Lorsque vous recherchez une chaîne de caractères en particulier, sélectionnez l’option Ignorer les majuscules pour trouver les lettres indépendamment des majuscules et minuscules ; sélectionnez Mot complet pour que le mot recherché ne s’arrête pas aux portions de mots incluant le terme recherché. Si vous utilisez des regroupements dans une expression, vous pouvez être amené à sélectionner (ou remplacer) toute l’expression ou uniquement l’un des groupes ; utilisez le menu local pour indiquer votre choix. Pour les options Précédent et Suivant, OmniGraffle considère les objets dans l’ordre de la gauche vers la droite et de haut en bas.
Rechercher le suivant : choisissez cette option pour rechercher l’occurrence suivante du texte que vous avez saisi en dernier dans le panneau Rechercher comme si vous veniez de cliquer sur son bouton Suivant.
Rechercher le précédent : choisissez cette option pour rechercher l’occurrence précédente de la chaîne de caractères que vous avez saisie en dernier dans le panneau Rechercher comme si vous veniez de cliquer sur son bouton Précédent.
Entrer la sélection : sélectionnez du texte dans un objet, puis choisissez cette commande pour placer le texte sélectionné dans le panneau Rechercher (cette option remplace toute sélection s’y trouvant déjà le cas échéant). Vous pouvez ensuite utiliser les commandes Rechercher le suivant et Rechercher le précédent pour rechercher d’autres occurrences de ce texte.
Aller à la sélection : déplacez la zone d’affichage de sorte que les objets sélectionnés soient visibles.
Orthographe et grammaire
Ce sous-menu contient les utilitaires orthographiques et grammaticaux standard de Mac OS X. Pour les options Précédent et Suivant, OmniGraffle considère les objets dans l’ordre de la gauche vers la droite et de haut en bas.
Insérer une variable
Lorsque vous modifiez le texte d’une forme, choisissez l’une de ces commandes pour insérer une balise spéciale. Une fois modifié le texte, la balise se change en aspect de l’objet en soi, du canevas, de la page ou du document. Si vous changez l’élément auquel se réfère la balise, celle-ci s’actualise automatiquement. Certaines variables, comme le numéro de page, sont utiles à placer sur un calque partagé pour ensuite partager ce calque avec chaque canevas.
Si vous utilisez une des variables de date, vous pouvez personnaliser le format de date grâce aux jetons de formats de date de Cocoa. Par exemple, pour obtenir une date telle que « 6/10/2007 16 h 00 », vous pouvez tapez ce qui suit :
<%date %d/%m/%Y %H h %M %>
Nom du canevas : nom du canevas sur lequel réside l’objet.
Nom du document : nom du document dans lequel réside l’objet.
Date actuelle : date au moment de la manipulation ; cela peut s’avérer pratique pour savoir quand un document a été imprimé.
Date de création du document : date à laquelle le document a été créé, d’après le système de fichiers de Mac OS X.
Date de modification du document : date du dernier enregistrement du document, d’après le système de fichiers de Mac OS X.
Auteur : nom de l’utilisateur qui a créé le fichier d’après le système de fichiers de Mac OS X.
Modificateur : nom de l’utilisateur qui a effectué le dernier enregistrement du fichier, d’après le système de fichiers de Mac OS X.
Numéro de page : numéro de page sur lequel se situe l’objet, en fonction des réglages de l’inspecteur Taille du canevas et du panneau Format d’impression.
Nombre total de pages : nombre de pages total du document. Utilisez cette option avec la balise Numéro de page pour obtenir des résultats du type « Page 2 sur 5 ».
Position X : distance horizontale entre l’objet et le point d’origine, comme indiquée dans l’inspecteur Géométrie.
Position Y : distance verticale entre l’objet et le point d’origine, comme indiquée dans l’inspecteur Géométrie.
Largeur : taille horizontale de l’objet, comme indiquée dans l’inspecteur Géométrie.
Hauteur : taille verticale de l’objet, comme indiquée dans l’inspecteur Géométrie.
Longueur des lignes : cette balise est quelque peu unique. Si la forme contenant la balise est une étiquette de ligne, elle indique la longueur de la ligne. Cette balise n’est pas utile si l’objet de la forme n’est pas attaché à une ligne.
Modifier l’élément LinkBack : si vous avez collé du contenu LinkBack depuis une autre application, sélectionnez l’objet et utilisez cette commande pour ouvrir le contenu dans l’application d’origine.
Caractères spéciaux : ouvrez la palette de caractères Mac OS X standard.
