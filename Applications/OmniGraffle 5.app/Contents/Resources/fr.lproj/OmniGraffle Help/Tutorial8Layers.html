Guide d'initiation 8 : Calques

Les calques contiennent différentes séries d’objets sur le même canevas. Nous allons placer notre image d’arrière-plan sur un nouveau calque pour la garder séparée du schéma des personnages.
Cliquez sur le bouton Canevas de la barre d’outils pour afficher la barre latérale des canevas. Elle contient le seul canevas existant pour le moment ; cliquez sur le triangle d’affichage en regard pour afficher ses calques.
Vous n’avez qu’un seul calque pour le moment, c’est pourquoi nous allons en ajouter un autre. Cliquez sur le bouton Ajouter un calque sous la barre latérale. Vous venez de créer un calque. Vous pouvez le nommer comme bon vous semble, par exemple, « Arrière-plan ».
L’ordre des calques est important ; les éléments présents sur les calques du dessus apparaissent au-dessus des éléments présents sur les calques inférieurs. Nous voulons que l’arrière-plan apparaisse derrière notre schéma de relations. Pour cela, faites glisser le nouveau calque et lâchez-le sous l’autre dans la liste.
Concentrons-nous à présent sur le calque d’arrière-plan. Le calque sur lequel vous travaillez à présent dispose d’une icône représentant un crayon, à gauche de son aperçu dans le tiroir. Si le calque d’arrière-plan ne dispose pas d’une telle icône, cliquez simplement à gauche de l’aperçu du calque pour l’y placer.
Pour votre arrière-plan, recherchez une image de qualité sur vos disques (ou téléchargez-en une depuis le Web). Faites glisser le fichier image depuis le Finder et lâchez-la sur une zone vide du canevas. Une forme contenant l’image est créée sur le calque d’arrière-plan.
Faites glisser les poignées de la sélection sur les coins de la forme pour l’étendre jusqu’à une taille confortable ; maintenez la touche Maj enfoncée tout faisant glisser les poignées pour éviter toute distorsion de l’image. Puis utilisez l’inspecteur Image pour définir l’opacité de l’image sur environ 15 %.
Vous disposez maintenant d’un schéma instructif et attractif. Vous savez comment utiliser la présentation de structure pour créer des objets, comment créer des étiquettes et les connecter et comment ajouter des styles. Vous savez comment propager des styles sur plusieurs objets. Vous savez comment utiliser plusieurs calques. À présent, vous êtes quasiment incollable sur les fonctions de l’application.
