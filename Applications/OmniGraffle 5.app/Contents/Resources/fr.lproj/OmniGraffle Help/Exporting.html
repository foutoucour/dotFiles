Exportation sous différents formats

Vous pouvez créer des fichiers de types différents à partir de vos schémas OmniGraffle. Sélectionnez Exporter dans le menu Fichier. Lors de l’exportation, un nouveau fichier est enregistré sur le disque tout en laissant votre document d’origine tel quel à l’écran. Bien sûr, tous les aspects d’un schéma ne peuvent être représentés dans tous les formats de fichier.
Les formats de fichier disponibles sont les suivants :
Document OmniGraffle Vous pouvez définir le fichier en lecture seule (non modifiable) et choisir d’inclure des images liées dans le fichier pour qu’elles s’affichent correctement sur l’ordinateur d’une autre personne.
Image vectorielle PDF : format PDF (Portable Document Format). Il s’agit d’un format multiplateforme des plus pratiques, qui offre une bonne prise en charge des composants vectoriels et bitmap.
Image bitmap TIFF : format TIFF (Tagged Image File). Ce format prend en charge la transparence.
Image bitmap PNG : format PNG (Portable Network Graphics), un format courant pour le Web. Ce format utilise la compression sans perte pour conserver les détails d’une image tout en diminuant la taille de son fichier. Il prend en charge la transparence.
Image bitmap JPEG : format courant de compression d’image. Vous pouvez diminuer la qualité de l’image afin de réduire la taille d’un fichier.
Image vectorielle EPS : format EPS (Encapsulated PostScript). Il s'agit d'un format courant dans le domaine de l'impression professionnelle.
Carte image HTML : fichier combiné d’image JPEG, PNG ou GIF et hypertexte. Les actions URL du document OmniGraffle d’origine sont codées dans la carte image sous forme de liens permettant de cliquer sur les zones de l’image correspondant aux liens pour suivre ces derniers. (Vous remarquerez que les actions sur les lignes ne sont pas incluses, alors que les actions sur les étiquettes de ligne le sont.)
OmniOutliner 3 : représentez le schéma sous forme de structure de texte, à l’aide des lignes de connexion entre les formes pour créer une hiérarchie. Cela ne s’avère pas le plus adapté aux schémas qui ne possèdent pas de structure strictement arborescente.
Dessin vectoriel SVG : norme Internet Open Source utilisant le langage XML.
Image vectorielle PICT : ancien format graphique Macintosh.
Image bitmap BMP : ancien format graphique.
Style de schéma OmniGraffle, Modèle OmniGraffle, Patron OmniGraffle : ressources d’OmniGraffle.
Document Visio® XML : format de fichier basé sur le XML pour l’application de création de schéma de Microsoft®.
Suivant le format de fichier que vous choisissez, différentes options se présentent. La plupart proposent une option Zone d’exportation grâce à laquelle vous pouvez choisir la zone du document que vous souhaitez exporter.
Sélection actuelle : permet de n’exporter que les objets sélectionnés.
Tous les objets : permet d’exporter tous les objets du canevas, sauf les zones vides.
Région : prévue pour indiquer une zone précise du canevas à exporter.
Canevas actuel : sert à exporter l’ensemble du canevas.
Tout le document : permet d’exporter tout le document ; un fichier image est alors créé pour chaque canevas.
Selon le format que vous choisissez, vous pouvez également choisir une échelle pour exporter le document dans des dimensions plus petites ou plus grandes que la taille réelle, en choisir la résolution, le niveau de compression et en précisant si l’arrière-plan doit être transparent ou non.
