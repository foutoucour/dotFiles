Sous-graphiques

Un sous-graphique est un type de groupe particulier qui peut être étendu pour afficher la hiérarchie qu’il contient, ou réduit pour n’en faire qu’un seul objet et compact.
Pour créer un sous-graphique :
Sélectionnez certains objets du canevas.
Une fois les objets sélectionnés, choisissez Disposition ‣ Regrouper en sous-graphique. Vos objets sont entourés d’un rectangle de sous-graphique.
Appliquez le style et l’étiquette que vous voulez au rectangle.
Pour réduire un sous-graphique, sélectionnez-le puis choisissez Condenser le sous-graphique dans le menu Disposition (ou dans le menu local apparaissant lorsque vous cliquez avec le bouton droit de la souris ou lorsque vous cliquez tout en maintenant la touche Contrôle enfoncée sur le sous-graphique). Les objets contenus dans le sous-graphique sont masqués et le sous-graphique se transforme en un seul objet. Toutes les lignes de connexion entrantes ou sortantes appartenant à des objets du sous-graphique deviennent des lignes entrantes ou sortantes de l’objet condensé.
Les membres d’un sous-graphique peuvent être sélectionnés et modifiés comme des membres d’un groupe : cliquez une fois pour sélectionner le sous-graphique, puis cliquez à nouveau pour sélectionner un membre situé à l’intérieur de celui-ci.
Vous pouvez redimensionner le rectangle contenant l’ensemble, tant qu’il peut contenir tous ses objets membres.
Pour désassembler un sous-graphique, sélectionnez-le puis choisissez Dissocier dans le menu Disposition.
