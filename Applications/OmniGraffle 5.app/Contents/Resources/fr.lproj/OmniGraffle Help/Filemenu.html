Menu Fichier

Nouveau : Si vous disposez d’un modèle par défaut, cette option permet de créer un document s’appuyant sur le modèle. Si vous n’avez pas de modèle par défaut, cette option permet d’ouvrir le sélecteur de modèles.
Nouvelle ressource : créez un patron, un modèle ou un style de schéma OmniGraffle, à partir du modèle de votre choix.
Sélecteur de modèles : ouvrez le sélecteur de modèles, dans lequel vous pouvez choisir quel modèle utiliser pour un nouveau document.
Ouvrir : recherchez un document à ouvrir. Vous pouvez ouvrir un document OmniGraffle ou un fichier dans l’un des formats compatibles avec la fonction d’importation d’OmniGraffle.
Ouvrir récent : ouvrez un document que vous avez utilisé récemment. Choisissez Effacer le menu pour supprimer la liste des documents récents, empêchant ainsi que personne ne sache que vous avez travaillé sur le document « Invitation soirée surprise.graffle ».
Fermer : fermez la fenêtre ouverte au premier plan. Si vous avez apporté des modifications non encore enregistrées, vous êtes alors invité à les enregistrer ou à y renoncer.
Enregistrer : enregistrez sur le disque dur l’ensemble de votre travail effectué dans le document au premier plan, afin de le sauvegarder. Si vous avez déjà enregistré une fois le document, la nouvelle version remplace la précédente. Si c’est la première fois que vous enregistrez le document, vous êtes invité à rechercher un emplacement sur le disque dur et à attribuer un nom de fichier au document.
Enregistrer sous : enregistrez sur le disque dur le document au premier plan, mais sous un nouveau nom. Cela signifie que vous créez une version du fichier portant un autre nom et que l’ancienne version du fichier est conservée telle qu’elle était déjà enregistrée à sa dernière sauvegarde.
Tout enregistrer : enregistrez sur le disque dur l’ensemble de votre travail effectué sur tous les documents ouverts.
Revenir à la version enregistrée : lorsque certaines révisions ne vous satisfont pas, choisissez cette option pour revenir à la version du fichier présente sur le disque ; elle agit comme un grand Annuler. Si vous ouvrez un fichier auquel des modifications ont été apportées lors d’une sauvegarde automatique, vous pouvez choisir cette commande pour revenir à la dernière version enregistrée.
Inclure une image : recherchez un fichier image ; une forme contenant l’image est crée sur le canevas au dernier endroit sur lequel vous avez cliqué avec l’outil Sélection.
Exporter : cette option crée un fichier selon le type de fichier de votre choix à partir d’une partie des données de votre document OmniGraffle. Les fichiers exportés sont des approximations du document OmniGraffle dans un format différent, et non des répliques parfaites. Consultez la rubrique relative à l’exportation pour en savoir plus.
Format d’impression : affichez la zone de dialogue Format d’impression de Mac OS X standard. Cette page comporte plusieurs sous-fenêtres de réglages pour choisir comment représenter votre document sur les pages.
Imprimer : affichez la zone de dialogue Imprimer de Mac OS X standard. En plus des réglages du format d’impression, vous pouvez définir de nombreuses options propres au mode d’impression.
Imprimer le(s) canevas : affichez la zone de dialogue Imprimer, mais seulement pour les canevas sélectionnés.
