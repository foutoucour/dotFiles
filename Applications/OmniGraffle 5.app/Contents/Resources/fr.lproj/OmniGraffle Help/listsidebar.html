Utilisation de la barre latérale des listes pour ordonner des objets

La barre latérale de liste affiche tous les objets de chaque calque du canevas actif, dans l’ordre établi d’avant vers l’arrière. Pour l’ouvrir, choisissez Liste dans le menu Présentation. Si la barre latérale de structure est ouverte, vous pouvez également cliquer sur le bouton dans la partie supérieure pour passer à la barre latérale de liste.
Les calques sont répertoriés dans l’ordre d’avant en arrière, et chaque calque intègre une liste des objets dans l’ordre d’avant en arrière. Vous pouvez faire glisser les objets pour leur faire adopter un ordre différent ou les transférer à un autre calque. Vous pouvez également utiliser les boutons sous la liste pour déplacer les objets sélectionnés d’un cran vers l’avant ou vers l’arrière, ou complètement vers l’avant ou vers l’arrière de leur calque.
Dans les préférences Général, vous pouvez faire en sorte que la barre latérale de liste affiche le numéro d’identification unique de chaque objet, ce qui peut être utile pour la création de scripts.
