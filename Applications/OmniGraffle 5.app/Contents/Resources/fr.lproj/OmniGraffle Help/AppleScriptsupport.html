Prise en charge d’AppleScript

OmniGraffle prend en charge AppleScript de façon étendue. Ouvrez le dictionnaire OmniGraffle dans l’Éditeur de scripts pour obtenir des détails sur le modèle d’objet et les commandes OmniGraffle.
Vous pouvez stocker des scripts AppleScript dans le dossier des scripts d'OmniGraffle pour pouvoir y accéder plus facilement à partir du menu Scripts. Vous pouvez également attacher des scripts à des objets précis d’un canevas à l’aide de l’inspecteur Action. Après avoir activé l’outil de navigation, cliquez sur un objet présentant un script pour que ce dernier s’exécute.
Si vous sélectionnez certains objets, puis que vous choisissez Copier en tant que AppleScript dans le menu Édition, vous obtenez les lignes AppleScript nécessaires pour créer ces objets. Il s’agit d’un moyen efficace pour comprendre comment la prise en charge d’AppleScript par OmniGraffle fonctionne, ou pour ajouter des objets spécifiques à vos scripts.
