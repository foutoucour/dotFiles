Élaboration d’ombres d’objets projetées à l’aide de l’inspecteur Ombre

Utilisez l’inspecteur Ombre pour appliquer une ombre derrière les objets sélectionnés.
La case Ombre définit si l’objet doit projeter une ombre. Lorsqu’elle est cochée, les commandes deviennent disponibles pour modifier l’apparence de l’ombre.
Faites glisser le curseur d’application de flou pour régler la netteté ou la diffusion de l’ombre.
Cliquez sur un bouton pour choisir si l’ombre apparaît devant les objets ou derrière eux sur le même calque. Si vous choisissez de laisser les ombres apparaître devant les objets d’un même calque, l’ordre des objets devient particulièrement important ; vous pouvez réordonner les objets à l’aide des commandes Avancer... et Reculer... du menu Disposition.
Faites glisser le réticule dans la commande du décalage ou saisissez des nombres dans les champs de décalage afin de régler la distance de l’objet où l’ombre doit tomber.
Cliquez sur le cadre de couleurs dans la partie supérieure gauche afin de choisir une couleur d’ombre. Ce sont celles qui sont partiellement sombres qui ont tendance à rendre le mieux.
