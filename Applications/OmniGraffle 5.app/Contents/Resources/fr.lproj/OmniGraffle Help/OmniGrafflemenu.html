Menu OmniGraffle

À propos d’OmniGraffle : découvrez les fonctionnalités propres à la version d’OmniGraffle que vous utilisez.
Rechercher des mises à jour : normalement, OmniGraffle recherche les versions mises à jour régulièrement selon les préférences de mise à jour. Utilisez cette commande pour rechercher les mises à jour immédiatement. S’il existe une version mise à jour du logiciel, vous êtes alors invité à la télécharger.
Préférences : ouvrez la fenêtre des préférences d’OmniGraffle ; elle peut vous servir à personnaliser le comportement de l’application.
Licences : affichez la zone de dialogue Licences, dans laquelle vous pouvez retrouver les licences de votre logiciel OmniGraffle. Pour en savoir plus sur le fonctionnement des licences, consultez la rubrique Licences.
Services : le menu Services reprend l’ensemble des services pour tout le système, proposés par les applications installées sur votre ordinateur. OmniGraffle fonctionne plutôt bien avec Mac OS X, donc vous devriez être en mesure d’utiliser vos services standard à partir de l’application même.
Masquer OmniGraffle : laissez OmniGraffle ouvert, mais écartez temporairement ses fenêtres lorsque vous travaillez dans une autre application.
Masquer les autres : masquez toutes les fenêtres des autres applications ouvertes, pour pouvoir vous concentrer sur OmniGraffle.
Tout afficher : affichez les fenêtres de toutes les applications que vous avez masquées à l’aide des commandes Masquer.
Quitter OmniGraffle : choisissez cette option si vous souhaitez vraiment quitter OmniGraffle pour l’instant. Si vous avez des documents ouverts non enregistrés, vous êtes invité à enregistrer vos modifications ou à y renoncer.
