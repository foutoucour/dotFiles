Contrôle des réglages du document dans l’inspecteur Document

Le menu local sous les options de format de fichier détermine si vous voulez enregistrer votre document en tant que fichier plat ou paquet de fichiers. Un fichier plat correspond à un seul fichier sur le disque, dans lequel toutes les images attachées sont intégrées. Un paquet de fichiers est en fait un dossier apparaissant comme un seul fichier et où toutes les images attachées sont placées en désordre dans le dossier. Suivant les cas, il peut s’avérer plus intéressant d’utiliser un type de fichier plutôt qu’un autre ; si vous n’en voyez pas l’utilité, optez pour le réglage Automatique.
Généralement les documents OmniGraffle sont des fichiers texte de type « property list ». Si la case Compresser sur le disque est cochée, votre fichier est enregistré dans un format binaire occupant moins d’espace sur le disque mais qui ne peut être lu par les applications d’élaboration de scripts ou d’édition de texte. Si vous pensez que personne n’aurait à ouvrir leurs documents OmniGraffle dans un éditeur de texte, vous pouvez compresser vos fichiers.
Vous pouvez spécifier des marges de page ou utiliser simplement les marges d’impression définies par votre gestionnaire d’imprimante (ou par vos réglages personnalisés de mise en page).
