Ajout de données à des objets à l’aide de l’inspecteur Note

L’inspecteur Note contient un champ de saisie de texte à associer à l’objet sélectionné et un tableau de données personnalisées.
Cette note peut être mise au format RTF, ce qui signifie que vous pourrez utiliser tous les styles et couleurs de police pouvant être employés dans d’autres modules d’OmniGraffle. Lorsque vous placez le pointeur sur un objet présentant une note, le texte de la note apparaît dans une balise d’aide placée au-dessus de l’objet.
Vous pouvez utiliser le tableau de données personnalisées pour conserver vos propres informations sur l'objet. Les données personnalisées sont stockées sous forme de paires clé/valeur : la clé s’apparente à une étiquette sur le type d’informations stockées et la valeur est l’information en soi.
Prenons l’exemple du schéma d’un réseau informatique dans lequel vous voulez attribuer un numéro de modèle à chaque composant. Cliquez sur la rangée de paramètre fictif ou sur le bouton plus pour créer une paire clé/valeur. Dans la colonne Clé, vous pouvez saisir Numéro de pièce et dans le champ Valeur, par exemple A1181. Ces données n’ont aucun effet sur le fonctionnement d’OmniGraffle ; il s’agit simplement d’un moyen de stocker des données arbitraires se rapportant aux objets de votre schéma. Pour supprimer une rangée de données, cliquez sur le bouton représentant une croix à droite de la rangée.
Vous pouvez également modifier la note d’un canevas. Pour cela, cliquez sur le canevas dans la barre latérale (pour qu’il soit mis en surbrillance dans votre couleur de sélection), puis utilisez l’inspecteur Note normalement.
La fonction de recherche Spotlight de Mac OS X vous permet d’accéder aux notes, si vous recherchez des schémas OmniGraffle contenant certains mots.
