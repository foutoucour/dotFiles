Coloriage des formes à l’aide de l’inspecteur Remplissage

Utilisez l’inspecteur Remplissage pour choisir une couleur ou un dégradé pour remplir l’intérieur des formes sélectionnées.
La case Remplissage définit si la forme doit avoir un remplissage ou non. Lorsqu’elle est cochée, les commandes permettant de choisir le type de remplissage deviennent disponibles.
Utilisez le menu local situé à gauche de l’inspecteur pour choisir parmi les cinq sortes de remplissage : Uni, Mélange linéaire, Mélange radial, Mélange linéaire double et Mélange radial double.
La commande de rotation du mélange s’active lorsque vous choisissez un mélange linéaire ; saisissez un nombre de degrés dans le champ ou faites glisser la commande circulaire afin de régler l’angle du dégradé. La commande de position du mélange s’active, elle, lorsque vous choisissez un dégradé radial ; faites glisser son réticule pour déplacer le centre du dégradé.
Cliquez sur le cadre des couleurs afin de choisir une couleur pour le remplissage.
Si vous avez choisi un mélange à deux couleurs, vous pouvez cliquer sur le bouton d’inversion afin d’inverser leurs couleurs.
Si vous choisissez un mélange, deux ou trois cadres de couleurs deviennent disponibles. Cliquez à l’endroit approprié pour choisir les couleurs destinées à former le mélange.
Faites glisser le curseur de déviation se trouvant complètement à droite pour dévier le mélange vers l’un des côtés.
Vous pouvez également modifier la couleur de remplissage d’un canevas. Pour cela, cliquez sur le canevas dans la barre latérale (pour qu’il soit mis en surbrillance dans votre couleur de sélection), puis utilisez l’inspecteur Remplissage normalement.
OmniGraffle Pro offre une section supplémentaire dans le panneau Couleur, permettant de créer des motifs de remplissage.
