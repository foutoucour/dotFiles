Guide d'initiation 7 : Ajout de sélection et de style

Nous allons à présent ajouter des styles aux objets restants. Pour sélectionner simultanément plusieurs formes, utilisez l’inspecteur Sélection qui fait partie de la fenêtre de l’inspecteur Canevas. Si vous ne le voyez pas à l’écran, sélectionnez-le dans le menu Inspecteurs.
L’inspecteur Sélection répertorie tous les types d’objets sur le canevas, classés selon leur style. Pour le moment, il doit mettre en évidence qu’un de vos trois rectangles de couleur est sélectionné. Un élément représentant les quatre rectangles en blanc uni avec des contours noirs doit également y être présent. Cliquez dessus pour sélectionner tous les objets similaires simultanément ; l’élément reflète alors que vous avez sélectionné les quatre objets semblables.
Vous pouvez utiliser les inspecteurs Propriétés pour modifier les aspects d’un objet qui ne sont pas strictement liés à son style visuel. Nous allons utiliser l’inspecteur Géométrie pour modifier la taille des quatre objets sélectionnés. Si l’inspecteur Géométrie n’apparaît pas à l’écran, sélectionnez-le dans le menu Inspecteurs. Dans les champs de largeur et de hauteur, saisissez de nouvelles valeurs pour rendre ces formes plus grandes que les autres.
Utilisez les inspecteurs Style pour appliquer simultanément un style aux quatre formes de façon à les différencier des autres. Dans cet exemple, nous avons choisi une couleur différente, un contour légèrement plus épais et une police plus grande.
