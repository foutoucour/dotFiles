Menu Présentation

Lancer la présentation : permet de passer en mode Présentation avec le document en arrière-plan.
Zoom
Zoom avant : permet de passer au niveau suivant du zoom pour que le canevas apparaisse plus gros.
Zoom arrière : permet de passer au niveau précédent du zoom pour que le canevas apparaisse plus petit.
Revenir à la taille réelle : réactive le niveau initial du zoom pour que le canevas apparaisse aussi grand qu’il l’est réellement d’après les unités du canevas.
Zoom sur la sélection : permet de passer au niveau et à la position afin de remplir la présentation à l’aide des objets sélectionnés.
Zoom précédent : permet de revenir au niveau de zoom précédant le niveau actif.
Faire tenir dans la fenêtre : permet de passer au niveau et à la position adéquats de façon à pouvoir visualiser tous les objets du canevas sur lequel vous travaillez.
Configuration d’essai : utilisez ce sous-menu pour définir un périphérique d’impression ou de vérification ainsi que le profil qui lui est attribué. Le profil de vérification par défaut est celui de Mac OS X (généralement le « Profil Générique CMJN »), mais vous pouvez utiliser le profil de n’importe quel périphérique d’impression ou de vérification disponible sur votre système.
Couleurs d’essai : activez la vérification à l’écran, en utilisant le périphérique sélectionné dans le sous-menu Configuration d’essai. Cela ajoute une étape de plus au chemin d’affichage des images et des couleurs. Chaque couleur est successivement convertie en profil de vérification, à partir du profil de document le plus adéquat parmi les trois disponibles, puis en profil d’affichage. Cette opération vous permet théoriquement de visualiser à l’écran la sortie prévue à l’impression, car le chemin d’impression de Mac OS X est ainsi dupliqué. Cependant, cela dépend fortement de la qualité du profil de couleurs propre à votre imprimante. Pour de nombreuses imprimantes à jet d’encre grand public par exemple, le profil d’imprimante prétend pouvoir reproduire n’importe quelle couleur (afin d’éviter le passage par la fonctionnalité ColorSync pour pouvoir substituer leur propre amélioration des couleurs dans le pilote d’imprimante). Pour ces imprimantes, il se peut que la vérification à l’écran ne change rien d’un point de vue visuel.
Bonus : hormis les objets, le canevas contient d’autres éléments dont l’affichage peut s’avérer utile. Choisissez un élément dans cette liste pour l’afficher ou le masquer.
Actions : tout objet auquel une action a été attribuée dans l’inspecteur Action affiche une balise d’action dans son coin supérieur droit.
Grille : il s’agit de la grille définie dans l’inspecteur Grille. Vous pouvez activer la fonction d’alignement sur la grille, que celle-ci soit visible ou pas.
Guides : il s’agit de guides manuels que vous configurez pour aligner les objets sur le canevas.
Aimants : sélectionnez cette option pour visualiser en permanence les aimants de tous les objets.
Notes : tout objet auquel une note de l’inspecteur Note affiche une balise de note dans son coin supérieur droit.
Origine : il s’agit du réticule qui représente l’origine du canevas.
Sauts de page : ce sont les lignes qui délimitent les différentes pages.
Règles : les règles vous aident à redimensionner les objets sur le canevas et à inclure une barre des inspecteurs pour modifier des objets.
Barres latérales : sélectionnez le nom d’une barre latérale pour l’afficher ou la masquer. Les canevas ont leur propre barre latérale, tandis que la présentation de structure et la présentation de liste partagent la même barre latérale.
Canevas : liste de canevas et de calques du document.
Structure : la barre latérale de structure, qui montre la hiérarchie des objets sur le canevas.
Liste : la barre latérale de liste, qui montre l’ordre des objets sur chaque calque.
Affichage du canevas
Modifier le canevas suivant : permet de passer au canevas suivant à celui que vous êtes en train de modifier.
Modifier le canevas précédent : permet de passer au canevas précédant celui que vous êtes en train de modifier.
Personnaliser la barre d’outils : permet d’afficher une page de commandes que vous pouvez faire glisser vers votre barre d’outils, comme c’est le cas pour d’autres applications de Mac OS X.
Afficher/Masquer la barre d’outils : définissez la visibilité de la barre d’outils dans la fenêtre du document.
