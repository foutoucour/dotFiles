Menu Disposition

Placer au premier plan/Avancer d’un plan : les objets placés sur le même calque suivent un ordre spécifique : les objets récemment créés apparaissent devant les objets déjà en place. Choisissez Placer au premier plan pour déplacer les objets sélectionnés devant les autres objets du calque. Maintenez la touche Option enfoncée pour changer cette commande en Avancer d’un plan, déplaçant alors les objets sélectionnés d’un plan vers l’avant dans l’ordre de leur placement.
Placer à l’arrière-plan/Reculer d’un plan : ces options fonctionnent comme Placer au premier plan, à la différence qu’elles permettent de déplacer les objets sélectionnés derrière tous les autres objets du calque. Maintenez la touche Option enfoncée pour changer cette commande en Reculer d’un plan, déplaçant ainsi les objets sélectionnés d’un plan vers l’arrière dans l’ordre de leur placement.
Verrouiller : cette fonction permet de verrouiller les objets sélectionnés. Un objet verrouillé ne peut être ni déplacé, ni modifié. Lorsque vous en sélectionnez un, il présente alors des croix au lieu des poignées de sélection. Généralement, la seule chose que vous soyez autorisé à faire avec un objet verrouillé est de le déverrouiller. Vous pouvez néanmoins modifier son étiquette et sa position hiérarchique à partir de la présentation Structure. Verrouiller peut se révéler utile pour vous assurer de ne pas modifier accidentellement un objet parfaitement établit.
Déverrouiller : cette fonction permet de déverrouiller les objets sélectionnés pour les déplacer ou les modifier.
Grouper : utilisez cette commande sur plusieurs objets sélectionnés afin de les réunir en un seul groupe, pour que ce dernier se comporte comme un seul et même objet.
Dissocier : utilisez cette commande sur un groupe sélectionné pour séparer ses composants en objets distincts.
Élaborer un tableau : cette commande, après avoir sélectionné un objet, vous permet de le transformer en un tableau d’une colonne sur une rangée (1 x 1) auquel vous pouvez ajouter des rangées et des colonnes.
Regrouper en sous-graphique : vous pouvez transformer les objets sélectionnés en un sous-graphique que vous pouvez ensuite condenser en un seul objet.
Condenser le sous-graphique : l’option permet de modifier le sous-graphique sélectionné en un seul objet, en conservant toutes les lignes de connexion cibles et sources.
Développer le sous-graphique : redonnez à un sous-graphique condensé sa structure entièrement développée.
Guides
OmniGraffle inclut des guides automatiques et des guides manuels. Les guides automatiques sont de petites lignes bleues qui apparaissent lorsque vous déplacez, redimensionnez ou faites pivoter des objets. Ils vous permettent de garder les objets toujours alignés et répartis sur l’espace de travail.
Les guides manuels sont des lignes magenta que vous placez vous-même et qui s’étendent sur l’ensemble du canevas. Ils vous permettent d’aligner des objets, et vous pouvez les afficher ou les masquer depuis le menu Présentation.
Activer/Désactiver les guides d’alignement automatiques : il s’agit de guides automatiques qui apparaissent pour vous aider à aligner les bords ou le centre des objets. Ils vous aident également à accompagner la rotation des objets lorsque vous faites glisser les poignées de sélection d’un objet tout en appuyant sur la touche Commande.
Activer/Désactiver les guides de distance automatiques : il s’agit de guides automatiques qui vous permettent d’espacer régulièrement les objets.
Verrouiller/Déverrouiller les guides : tant que les guides manuels sont verrouillés, vous ne pouvez pas les déplacer.
Nouveau guide horizontal/vertical : cela place un guide en travers du canevas ; si les guides ne sont pas verrouillés, vous pouvez utiliser l’outil Sélection pour faire glisser un guide et le positionner où vous le souhaitez. Lorsque vous faites glisser un objet à proximité d’un guide, ses bords ou son centre s’aligne sur le guide. Vous pouvez également faire glisser les nouveaux guides en dehors des règles verticale et horizontale.
Effacer les guides manuels : la fonction permet d’effacer tous les guides manuels du canevas. Si vous voulez simplement les masquer temporairement, utilisez le menu Présentation.
Aligner : ces commandes reprennent la fonction de chaque bouton de l’inspecteur Alignement pour aligner les bords et le centre des objets ou en espaçant les objets de façon égale.
Grille : les options Aligner à la grille et Aligner les objets sur la grille fonctionnent de la même manière que dans l’inspecteur Grille ; Caler bords sur grille redimensionne les objets pour que leurs bords suivent les lignes les plus proches de la grille.
Mise en page des schémas : ces commandes disposent les objets, soit tous les objets du canevas, soit les objets sélectionnés uniquement, selon les réglages de l’inspecteur Mise en page des schémas.
Taille : vous pouvez ordonner que tous les objets sélectionnés soient redimensionnés à la même hauteur, à la même largeur, ou bien aux deux. L’option Taille normale égalise la hauteur et la largeur d’un objet. Dans le cas des objets possédant une image définie à partir de l’inspecteur Image, Ajuster à l’image redimensionne l’objet pour qu’il atteigne les dimensions d’origine de l’image entière.
