Menu Format

Afficher les polices : permet d’ouvrir le panneau des polices Mac standard.
Afficher les couleurs : permet d’ouvrir le panneau des couleurs Mac standard.
Choisir un style de schéma : permet d’ouvrir le sélecteur de style de schéma, pour appliquer un style cohérent à tous les objets du canevas.
Appliquer de nouveau le style de schéma : permet d’appliquer à nouveau le style de schéma actif, pour tout objet d’un autre style ayant été ajouté après la dernière application.
Réappliquer le style aux objets sélectionnés : permet d’appliquer à nouveau le style de schéma actif aux seuls objets sélectionnés.
Police : il s’agit du sous-menu Police standard de Mac OS X ; il contient les commandes de définition d’une police et d’un style pour le texte sélectionné.
Texte : ce sous-menu contient les commandes pour modifier la position et le comportement du texte sélectionné. Cette fonctionnalité est également disponible dans l’inspecteur Texte.
Copier le style : placez les styles de l’objet sélectionné dans le presse-papiers à partir duquel ils peuvent être collés sur d’autres objets.
Coller le style : si des styles d’objet figurent dans le presse-papiers, appliquez-les aux objets sélectionnés.
Profils de couleur : si vous avez activé la gestion des couleurs dans les Préférences ColorSync, vous pouvez attribuer ou convertir les profils de couleur du document.
Attribuer des profils : l’attribution d’un profil de couleur ne modifie pas les données de couleur du document ; cela change uniquement la manière dont celles-ci sont interprétées lorsqu’elles sont envoyées vers un moniteur ou une imprimante. Cela signifie que les couleurs que vous voyez à l’écran ou sur la page imprimée sont susceptibles de varier lorsque vous attribuez un autre profil.
Convertir en profils : la conversion vers un autre profil modifie les données des couleurs dans le document de sorte qu’il garde la même apparence même avec un profil différent. Vous ne devriez pas observer de différence de couleur grâce à cette opération ; le but est en effet de conserver la même apparence des couleurs quelque soit le périphérique.
