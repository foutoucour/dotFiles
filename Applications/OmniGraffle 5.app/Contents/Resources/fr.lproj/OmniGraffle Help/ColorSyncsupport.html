Prise en charge de ColorSync

OmniGraffle Professional prend en charge la technologie ColorSync d’Apple, qui tente d’assurer des couleurs fidèles autant que possible entre leur visualisation sur un moniteur compatible ColorSync et leur impression sur une imprimante compatible ColorSync.
ColorSync est la quatrième sous-fenêtre de préférences d’OmniGraffle ; vous pouvez l’afficher en choisissant Préférences dans le menu OmniGraffle.
Cochez la case Utiliser les profils de couleur pour activer la prise en charge de ColorSync par OmniGraffle. OmniGraffle intègre les profils de couleur dans les documents que vous enregistrez et tente de lire les profils inclus dans les documents que vous ouvrez. Un profil de couleur aide les logiciels et le matériel compatibles ColorSync à afficher les couleurs du document de manière fidèle.
Il se peut que vous receviez des documents enregistrés avec des profils de couleur différents de celui que vous utilisez. Utilisez dans ce cas le menu local pour décider du comportement d’OmniGraffle relatif à ce genre de situation. Sélectionnez l’option conserver l’original pour garder le profil de couleur tel quel dans le document. Sélectionnez conversion automatique pour accorder le profil de couleur à vos réglages par défaut. Sélectionnez demander à l’ouverture pour qu’OmniGraffle vous propose de choisir parmi des options chaque fois que vous ouvrez un document présentant un profil de couleur différent.
L’option Profils par défaut permet d’afficher les profils de couleur que vous avez configurés dans les préférences ColorSync et s’appliquant au système tout entier. Cliquez sur ce bouton pour ouvrir ces préférences afin de modifier ces profils de sorte qu’ils répondent à votre matériel. Ces profils sont ceux qui seront intégrés dans les documents que vous créez.
Vous pouvez définir et convertir manuellement les profils de couleur à partir du menu Format et ainsi remplacer les profils définis par défaut.
