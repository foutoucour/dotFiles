Menu Inspecteurs

Afficher/Masquer les inspecteurs : permet de rendre la disposition de l’inspecteur actif visible ou invisible.
Réinitialiser l’emplacement des inspecteurs : permet de replacer tous les inspecteurs à leur emplacement d’origine, comme si vous vous serviez d’OmniGraffle pour la toute première fois.
Style : permet d’afficher ou de masquer l’inspecteur Style de votre choix.
Propriétés : permet d’afficher et de masquer l’inspecteur Propriétés de votre choix.
Canevas : permet d’afficher ou de masquer l’inspecteur Canevas de votre choix.
Document : permet d’afficher ou de masquer l’inspecteur Document de votre choix.
Afficher/Masquer les couleurs : permet d’appeler ou de refermer le panneau Couleurs de Mac OS X ; vous pouvez également cliquer sur le cadre de couleurs ouvert par un des inspecteurs. Vous pouvez faire glisser des échantillons de couleurs de ce panneau vers les remplissages des objets, leurs contours, leur texte ou bien vers les cadres de couleurs des inspecteurs.
Afficher/Masquer les couleurs : permet d’appeler ou de refermer le panneau Couleurs de Mac OS X ; vous pouvez également cliquer sur le bouton Police de l’inspecteur Texte. Si le canevas comporte un élément sélectionné, la police que vous choisissez s’y applique.
