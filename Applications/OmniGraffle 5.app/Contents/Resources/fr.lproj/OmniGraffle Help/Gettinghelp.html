Assistance par courriel
Si vous êtes coincé, si vous avez une bonne idée pour la prochaine version d’OmniGraffle ou si vous voulez simplement prendre de nos nouvelles, n’hésitez pas à nous envoyer un courriel. Choisissez Envoyer un commentaire dans le menu Aide pour nous envoyer un message ou adressez-nous un courriel directement à omnigraffle@omnigroup.com. Pour nous, l’assistance est primordiale, vous êtes donc assuré d’une réponse rapide rédigée par de véritables spécialistes.
