Personnalisation du comportement d’OmniGraffle à l’aide des préférences Général

Les préférences Général sont accessibles en choisissant Préférences dans le menu OmniGraffle.
Démarrage
Sélectionnez Ouvrir les documents ouverts lors de la dernière utilisation si vous souhaitez qu’OmniGraffle se souvienne de vos documents ouverts.
Sélectionnez Créer un document si rien n’est ouvert pour commencer avec un nouveau document lorsque vous n’activez pas les fonctions d’OmniGraffle de mémorisation des documents ouverts, ou si aucun document n’était ouvert la dernière fois que vous avez quitté l’application.
Nouveaux documents
Sélectionnez Afficher le sélecteur de modèles pour activer l’ouverture d’une fenêtre afin de choisir un modèle à chaque fois que vous créez un document. Sélectionnez Utiliser le modèle et choisissez votre modèle par défaut, si vous préférez ne pas passer par le sélecteur. D’une façon ou d’une autre, vous pouvez toujours ouvrir le sélecteur de modèles à l’aide de la commande Sélecteur de modèles du menu Fichier.
Modifier le texte
Déterminez le comportement des touches Retour et Tab lorsque vous modifiez le texte d’un objet. Quel que soit le comportement choisi, vous pouvez maintenir la touche Option enfoncée tout en appuyant sur la touche pour effectuer l’action inverse.
Sélection
Les petits objets présentent des poignées de plus petite taille lorsque vous les sélectionnez ; si vous préférez les plus petites poignées, vous pouvez cliquer sur Toujours utiliser les petites poignées de sélection pour les afficher à tout moment.
Lorsque vous cliquez au milieu de la forme d’un objet non rempli par une image ou une couleur, la sélection que vous souhaitez faire peut être de l’objet lui-même ou bien de tout autre élément se trouvant derrière celui-ci. Utilisez la case Cliquer sur les objets non remplis pour indiquer votre choix.
Balises d’aide
Lorsque vous déplacez ou redimensionnez un objet, une petite balise d’aide apparaît pour vous indiquer précisément le nouvel emplacement de l’objet et sa nouvelle taille. Vous êtes néanmoins libre de désactiver cette fonctionnalité si elle vous gène.
Sauvegarder
OmniGraffle peut périodiquement enregistrer une sauvegarde de vos fichiers quand vous travaillez dessus ; si un incident fâcheux survient, une partie de votre travail peut alors être récupérée.
Barre latérale de liste
Si vous accédez aux objets OmniGraffle via AppleScript ou si vous préférez utiliser des identifiants uniques, vous pouvez sélectionner l’option Afficher les numéros d’identification des objets pour ajouter une colonne à la barre latérale de liste.
Bouton Réinitialiser dans les préférences
Chaque sous-fenêtre des préférences est dotée d’un bouton Réinitialiser en bas à gauche. Si vous cliquez sur ce bouton, tous les réglages par défaut de cette sous-fenêtre sont rétablis. Pour réinitialiser les réglages recouvrant toutes les sous-fenêtres de préférences, maintenez la touche Option enfoncée tout en cliquant sur le bouton Réinitialiser.
