Création et gestion des modèles

Un modèle est un document qui agit en tant que point de départ pour les nouveaux documents. Par exemple, le modèle « 1 cm = 1 m » fourni avec OmniGraffle comporte des unités définies pour un dessin de grande échelle. D’autres modèles sont configurés avec des styles de schémas correspondant à certains types de dessin. OmniGraffle est fourni avec plusieurs modèles mais vous pouvez aussi créer votre propre modèle si vous élaborez souvent le même type de documents.
Dans le sélecteur de modèles, vous pouvez choisir un modèle pour votre nouveau document. Vous pouvez également créer un modèle, modifier un modèle vous appartenant ou définir un modèle par défaut. Une fois le modèle défini par défaut, OmniGraffle contourne le sélecteur de modèles et utilise ce modèle pour tout document que vous créez. Si vous voulez utiliser un autre modèle, vous pouvez également utiliser la commande Sélecteur de modèles du menu Fichier.
Lorsque vous créez un modèle OmniGraffle, toutes ses caractéristiques le concernant sont conservées, de la taille du canevas et la disposition de la grille, aux calques, en passant par les visibilité des suppléments et les réglages de disposition automatique. Pour en créer un, choisissez Nouvelle ressource ▸ Nouveau modèle de document dans le menu Fichier, puis choisissez un modèle existant à partir duquel commencer. Configurez le document comme vous le souhaitez, puis enregistrez-le. L’emplacement d’enregistrement par défaut est le dossier Application Support qui vous est attribué, dans lequel OmniGraffle recherche vos modèles. Une fois que le modèle est enregistré à cet emplacement, il devient disponible dans le sélecteur de modèles.
