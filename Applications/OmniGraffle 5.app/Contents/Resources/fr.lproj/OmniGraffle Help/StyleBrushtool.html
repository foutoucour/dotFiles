Reproduction de styles à l’aide de l’outil Brosse de style

Utilisez cet outil pour reproduire des styles. La première fois que vous ouvrez OmniGraffle, l’outil Brosse de style est vide et son curseur est doté de flèches indiquant que vous pouvez absorber un style. Sélectionnez-le et cliquez sur n’importe quel objet du canevas pour absorber le style d’un objet dans la brosse. Une fois que la brosse contient un style, vous pouvez cliquer sur n’importe quel objet afin de remplacer le style de cet objet par celui de la brosse.
Son menu local comporte une liste de styles pouvant être absorbés ; sélectionnez ceux qui vous conviennent afin que la brosse n’absorbe ou n’applique que ces styles. Par exemple, si vous sélectionnez uniquement Remplissage, vous pouvez appliquer un certain style de remplissage à des objets sans affecter aucune de leurs autres propriétés de style.
Pour réinitialiser le contenu de la brosse, sélectionnez l’option Effacer le contenu dans le menu local de l’outil ; vous pouvez ensuite absorber un nouveau style. Il est par ailleurs possible de cliquer sur un autre objet tout en maintenant la touche Option enfoncée pour remplacer le contenu de la brosse.
