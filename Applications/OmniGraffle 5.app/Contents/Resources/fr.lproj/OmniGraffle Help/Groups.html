Groupes

Il est possible de regrouper plusieurs objets ensemble. Sélectionnez au moins deux objets, puis utilisez la commande de menu Disposition ‣ Grouper pour les transformer en groupe. Un groupe se comporte comme un seul objet ; vous pouvez le sélectionner, le déplacer, le redimensionner, utiliser des inspecteurs pour en modifier le style et les propriétés, y connecter des lignes, etc. Tous les objets peuvent être groupés, y compris les groupes d’objets entre eux.
Cliquez une fois pour sélectionner le groupe en tant qu’ensemble, puis cliquez de nouveau pour sélectionner un membre précis du groupe. Vous pouvez ensuite continuer à cliquer pour sélectionner individuellement des membres à l’intérieur du groupe.
Pour désassembler un groupe, sélectionnez-le et utilisez la commande Disposition ‣ Dissocier.
