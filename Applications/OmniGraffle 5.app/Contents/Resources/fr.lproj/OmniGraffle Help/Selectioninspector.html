Recherche d’objets à l’aide de l’inspecteur Sélection

Cet inspecteur comporte des représentations de tous les différents styles d’objets sur le canevas.
Chacun des styles d’objet contient une représentation de l’objet ainsi que deux nombres. Le premier nombre correspond au nombre d’objets sélectionnés qui sont de ce style ; le second est le nombre d’objets de ce style se trouvant sur le canevas.
Cliquez sur une case pour sélectionner tous les objets correspondants. Vous pouvez ensuite utiliser les inspecteurs Style et Propriétés pour modifier les objets sur le canevas comme si vous aviez cliqué dessus individuellement. Cette méthode est des plus pratiques, si vous souhaitez, par exemple, changer tous les losanges bleus en cœurs roses.
Vous pouvez également envoyer une note de style depuis le tiroir de style vers la case de cet inspecteur pour appliquer le style à tous les objets associés.
Le menu local Filtrer contient des attributs de style que vous pouvez utiliser pour différencier les types d’objets. Par exemple, si vous voulez que les objets dotés de remplissages différents soient considérés comme des types différents, mais qu’il importe peu que leurs contours soient différents, sélectionnez l’option Remplissage et désélectionnez Contour.
Saisissez des mots dans le champ Rechercher pour sélectionner des objets dont le texte, la note ou les données personnalisées correspondent à un mot ou à un groupe de mots particuliers. À mesure que vous tapez, les objets sont sélectionnés ou désélectionnés selon qu’ils correspondent ou non au texte saisi. Le menu local de la loupe stocke vos recherches les plus récentes.
