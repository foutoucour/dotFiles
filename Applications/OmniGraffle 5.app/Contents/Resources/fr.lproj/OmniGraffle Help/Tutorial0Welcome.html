Didacticiel 0 : Bienvenue

OmniGraffle excèle dans la clarification de relations complexes ; or, il n’existe pas de relations plus complexes que dans les comédies romantiques de Shakespeare. Essayez donc d’y voir clair entre les personnages de Beaucoup de bruit pour rien, et vous découvrirez vite les atouts de l’utilisation d’OmniGraffle. Bien sûr, il ne s’agit là que d’un exemple ; si vous êtes plus familier avec une autre histoire, n’hésitez pas ! La guerre des étoiles ou Le cinquième élément, à vous de choisir !
Pour commencer, créez un document en sélectionnant Sélecteur de modèles dans le menu Fichier ; pour ce didacticiel, nous allons utiliser le modèle "Basic".
