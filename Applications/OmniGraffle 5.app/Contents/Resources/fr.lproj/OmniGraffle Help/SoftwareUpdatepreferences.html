Préférences Mise à jour de logiciels

OmniGraffle peut automatiquement utiliser votre connexion à Internet pour vérifier la disponibilité de nouvelles versions et de versions mises à jour du logiciel ; vous pouvez accéder à ces options en choisissant Préférences dans le menu OmniGraffle, puis en cliquant sur le bouton Mettre à jour.
Si la case Rechercher des mises à jour est cochée, vous serez avisé lorsqu’une nouvelle version d’OmniGraffle est disponible. Cliquez sur le bouton Rechercher des mises à jour pour vérifier manuellement s'il existe une nouvelle version du logiciel.
Vous pouvez cocher la case Inclure les informations système pour communiquer certaines informations relatives à la configuration de votre ordinateur lorsque vous recherchez des mises à jour. Ces informations aident Omni à gérer les questions de compatibilité matérielle dans ses logiciels. Pour plus de détails au sujet du type d’informations que nous recueillons et de l’utilisation que nous en faisons, cliquez sur le bouton En savoir plus dans les préférences du logiciel.
