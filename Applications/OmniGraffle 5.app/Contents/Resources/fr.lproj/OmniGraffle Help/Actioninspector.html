Interactivité des objets à l’aide de l’inspecteur Action

Cet inspecteur détermine ce qui doit se passer lorsque vous cliquez sur l’objet par le biais de l’outil Navigation.
L’action par défaut ne fait rien. Vous pouvez donc cliquer sur l’objet autant de fois que vous le voulez, rien ne se passe.
Si vous choisissez ouvre une URL ou ouvre un fichier, un champ de texte et deux boutons apparaissent. Vous pouvez alors taper une URL ou un chemin de fichier dans le champ, faire glisser un fichier dans le panneau, ou encore cliquer sur le bouton Fichier afin de rechercher un fichier sur vos disques. Il est à noter que les chemins d’accès aux fichiers sont relatifs : ils s’expriment donc en partant du dossier contenant le document que vous manipulez et non pas du dossier de départ de votre disque dur. Cliquez sur le bouton Ouvrir pour tenter d’ouvrir le fichier ou l’URL spécifié dans le champ. Lorsque vous cliquez sur l’objet, le fichier ou l’URL s’ouvre.
Si vous choisissez exécute un script, un champ de texte apparaît pour entrer un script AppleScript. Le script que vous saisissez est exécuté de manière à ce que l’argument « self » se réfère à l’objet sur lequel vous avez cliqué. Cliquez sur le bouton Vérifier la syntaxe pour vous assurer que le script AppleScript est correct, puis cliquez sur Exécuter le script pour le tester. Lorsque vous cliquez sur l’objet, le script est exécuté.
Si vous choisissez passe à un autre endroit, un autre menu local apparaît pour choisir où se rendre l’emplacement dans le document actif. Vous pouvez passer à un canevas en particulier, au canevas suivant ou au précédent, ou encore à un point ou un objet spécifique sur n’importe quel canevas. Certaines de ces options offrent un tout petit aperçu du canevas sur lequel vous pouvez cliquer ou faire glisser le pointeur de la souris pour indiquer quel objet mettre en surbrillance, sur quel point centrer ou encore où zoomer l’affichage.
Si vous choisissez Afficher ou Masquer les calques, vous pouvez indiquer si vous voulez afficher, masquer ou basculer l’affichage d’un calque du canevas en cours.
