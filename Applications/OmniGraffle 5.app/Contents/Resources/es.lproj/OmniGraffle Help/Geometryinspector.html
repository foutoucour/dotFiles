Cambio de posición y tamaño de objetos mediante el inspector Geometría

En este inspector se establece la posición y la orientación de los objetos.
Los campos X e Y contienen las coordenadas horizontal y vertical de los objetos seleccionados; estos valores representan la distancia entre el origen del lienzo y la esquina superior izquierda del objeto (que puede no ser la que parece si el objeto está girado o invertido). Si hay varios objetos seleccionados, aparecerán dos guiones (--) en los campos que poseen distintos valores. Si introduce un nuevo valor de X e Y para los objetos seleccionados, estos se desplazarán a la posición indicada en el lienzo.
Los campos Anchura y Altura indican el tamaño horizontal y vertical de los objetos seleccionados. Si hay varios objetos seleccionados, aparecerán dos guiones (--) en los campos que poseen distintos valores. Si introduce un nuevo valor de anchura y altura para los objetos seleccionados, estos adoptarán el nuevo tamaño especificado. Si la opción Mantener proporciones está seleccionada, al cambiar la altura o la anchura el otro valor cambia proporcionalmente.
Para girar el objeto, introduzca un valor en grados en el campo de rotación o arrastre el control circular.
Haga clic en los botones de inversión para invertir el objeto horizontal o verticalmente.
Bajo la línea de separación hay controles para ajustar las etiquetas de línea; estos controles se activan si selecciona una etiqueta de línea. Arrastre el regulador Posición etiquetas para desplazar la etiqueta a lo largo de toda la línea. Arrastre el regulador Separación para mover la etiqueta de forma perpendicular a la línea. Utilice el menú local para seleccionar la rotación de las etiquetas: Paralelas y Perpendiculares definen la rotación con respecto a la línea, y el resto define la rotación independientemente.
