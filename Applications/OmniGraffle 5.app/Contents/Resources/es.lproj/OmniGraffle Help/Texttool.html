Creación y edición de texto con la herramienta Texto

Utilice la herramienta Texto para añadir o editar la etiqueta de un objeto, o para crear un nuevo objeto de figura con texto en su interior.
Haga clic en una figura para empezar a editar su texto. Mientras esté en el modo de edición de texto, la regla superior se convertirá en una regla de texto y podrá utilizar los comandos habituales de edición de texto de Mac OS X (por ejemplo, puede usar las teclas de flecha para desplazar el punto de inserción).
Haga clic o arrastre el puntero por una zona vacía del lienzo para crear una nueva figura especialmente configurada para contener texto. Si hace un solo clic, se creará un objeto que se expande horizontalmente para albergar el texto que introduzca; si arrastra el cursor, se creará un objeto de una anchura determinada. Recuerde que los objetos que cree de esta manera no difieren en nada esencial de las demás figuras, simplemente comienzan con determinados ajustes en el inspector Estilo: Texto, lo que los hace adecuados para contener texto.
Haga clic en una línea de conexión para añadir una etiqueta. Esta etiqueta es como el objeto creado al hacer clic en una zona vacía del lienzo, salvo por el hecho de que está ligada a una línea en forma de etiqueta de línea.
Las notas del interior de los objetos pueden encontrarse mediante la función de búsqueda Spotlight de Mac OS X, en el caso de que necesite encontrar aquellos diagramas de OmniGraffle que contengan determinadas palabras.
