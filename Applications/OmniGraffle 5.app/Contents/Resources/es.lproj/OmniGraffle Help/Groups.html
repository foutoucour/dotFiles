Grupos

Varios objetos pueden unirse para formar un grupo. Seleccione dos o más objetos y utilice el comando Disposición ‣ Agrupar para convertirlos en un grupo. Un grupo se comporta como un objeto simple: puede seleccionarlo, moverlo, cambiar su tamaño, utilizar los inspectores para modificar su estilo y sus propiedades, conectarle líneas, etc. Puede agruparse cualquier tipo de objeto, incluso grupos de objetos.
Haga clic una vez para seleccionar el grupo y vuelva a hacer clic para seleccionar un componente del grupo. Puede seguir haciendo clic para seleccionar componentes de grupos situados dentro de grupos.
Para dividir un grupo, selecciónelo y utilice el comando Disposición ‣ Desagrupar.
