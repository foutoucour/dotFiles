Exportación a diferentes formatos

Puede crear archivos de distintos tipos a partir de los diagramas de OmniGraffle. Seleccione Exportar en el menú Archivo. Al exportar, se crea un nuevo archivo en el disco y se deja intacto el documento original tal como aparece en pantalla. Por supuesto, no todos los elementos de un diagrama pueden convertirse a cualquier formato de archivo.
Los formatos disponibles son:
Documento de OmniGraffle: es un documento de OmniGraffle. Puede hacer que el archivo sea de sólo lectura (no editable), así como incluir en él imágenes enlazadas para que se muestren correctamente en el ordenador de otra persona.
Imagen vectorial PDF: el conocido formato PDF. Este formato multiplataforma es muy utilizado y resulta útil para componentes vectoriales y mapas de bits.
Imagen de mapa de bits TIFF: formato TIFF. Este formato admite transparencias.
Imagen de mapa de bits PNG: un formato muy utilizado para archivos web. Este formato emplea un sistema de compresión sin pérdidas que conserva los detalles de la imagen y reduce su tamaño. También acepta transparencias.
Imagen de mapa de bits JPEG: un conocido formato de compresión de imágenes. Reduce la calidad de la imagen para obtener un archivo de menor tamaño.
Imagen vectorial EPS: PostScript encapsulado. Este es un formato muy habitual en la impresión profesional.
Mapa de imagen HTML: un archivo de hipertexto más una imagen JPEG, PNG o GIF. Las acciones URL del documento original de OmniGraffle se codifican en el mapa de imagen en forma de enlaces de manera que puede hacerse clic en las imágenes para verlas (no se incluyen las acciones asignadas a líneas, pero sí las acciones asignadas a etiquetas de línea).
OmniOutliner 3: representa el diagrama como un esquema de texto, usando las líneas de conexión entre figuras para crear una jerarquía. Sólo es recomendable para exportar estrictamente diagramas que tengan una estructura en árbol.
Dibujo vectorial SVG: un estándar abierto de Internet que utiliza XML.
Imagen vectorial PICT: un antiguo formato de Macintosh para gráficos.
Imagen de mapa de bits BMP: un antiguo formato gráfico de Windows.
Estilo de diagrama de OmniGraffle, Plantilla de OmniGraffle, Modelo de OmniGraffle: recursos para OmniGraffle.
Documento XML de Visio®: el formato de archivo basado en XML para la aplicación de creación de diagramas de Microsoft®.
En función del formato de archivo que elija, hay varias opciones disponibles. La mayoría de los formatos presentan la opción Área de exportación, que permite indicar qué parte del documento se desea exportar.
Selección actual: sólo exporta los objetos que están seleccionados.
Todos los objetos: exporta todos los objetos del lienzo salvo las zonas vacías.
Sector: define la zona concreta del lienzo que se exportará.
Lienzo actual: exporta todo el lienzo.
Documento completo: lo exporta todo y crea un archivo de imagen para cada lienzo.
Según el formato elegido, también podrá seleccionar una escala para exportar con un tamaño mayor o menor que el tamaño real, seleccionar una resolución diferente y establecer un determinado nivel de compresión, así como utilizar un fondo transparente.
