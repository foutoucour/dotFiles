Soporte para AppleScript

OmniGraffle cuenta con una amplia compatibilidad con AppleScript; abra el diccionario de OmniGraffle en el Editor de Scripts para obtener más información sobre el modelo de objetos y los comandos de OmniGraffle.
Puede almacenar AppleScripts en la carpeta de scripts de OmniGraffle para poder acceder a ellos fácilmente a través del menú Scripts. También puede adjuntar scripts a objetos individuales de un lienzo mediante el inspector Acción. Al activar la herramienta Explorar y hacer clic en un objeto que tenga un script adjunto, éste se ejecuta.
Si selecciona algunos objetos y elige Copiar como AppleScript en el menú Edición, puede obtener las líneas exactas de AppleScript que se necesitan para crear dichos objetos. Esta es una excelente manera de aprender cómo funciona el soporte para AppleScript de OmniGraffle, o de añadir objetos específicos a sus propios scripts.
