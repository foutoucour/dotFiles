Importación desde otras aplicaciones

OmniGraffle permite utilizar varios formatos de archivo. Puede importar los archivos con formato compatible abriéndolos tal como haría con un archivo cualquiera de OmniGraffle.
Documentos, plantillas y modelos de Microsoft® Visio® (.vsd, .vdx, .vst, .vtx, .vss, .vsx)
Los documentos guardados con el software de creación de diagramas Microsoft® Visio® en su formato basado en XML (¡y ahora incluso en su formato binario original!) pueden leerse con OmniGraffle Pro. Tenga en cuenta que, aunque es posible abrir el formato binario, no se puede exportar en dicho formato; deberá guardarlo como un diagrama de OmniGraffle o exportarlo en formato XML. Recuerde también que los gráficos en formato Microsoft® Enhanced Metafile (EMF) no pueden importarse.
Archivos DOT (.dot)
OmniGraffle entiende un subconjunto del lenguaje DOT empleado por Graphviz. No se trata de una versión completa de DOT, pero puede ser muy útil.
Documentos de Diagram! 2 (.diagram2 y .dpalette2)
Diagram es una aplicación de NeXTSTEP/OPENSTEP similar a OmniGraffle. OmniGraffle puede importar los objetos de un archivo de Diagram pero no las imágenes EPS, los sonidos o los enlaces adjuntos.
Documentos de OmniOutliner (.ooutline y .oo3)
Al abrir un archivo de OmniOutliner, los elementos del esquema se interpretan como un diagrama. En el cuadro de diálogo que aparece puede indicar cómo debe interpretarse el esquema:
Utilice la lista del menú local para elegir un estilo de diagrama.
Utilice la tabla para asignar los objetos del estilo de diagrama a las columnas del esquema. Si desea ver varias columnas de información para cada elemento, utilice un estilo de diagrama que tenga objetos agrupados. También puede asignar una columna a las notas de los objetos.
Elija si desea colocar el esquema en un documento totalmente nuevo o en un nuevo lienzo en el documento abierto.
Proyectos de Xcode (.xcode), proyectos de Project Builder (.pbproj) y archivos framework (.framework)
OmniGraffle emplea las cabeceras Objective-C para construir un diagrama de las relaciones de herencia, protocolo y categoría.
Archivos nib de Interface Builder (.nib)
OmniGraffle representa gráficamente las conexiones del archivo nib para facilitar la visualización.
Archivos de EOModeler (.eomodeld)
OmniGraffle crea un diagrama de las relaciones de base de datos descritas por el archivo.
Estructuras de carpetas
Arrastre una carpeta del Finder y suéltela sobre el icono de la aplicación OmniGraffle para realizar un diagrama del contenido de la carpeta.
