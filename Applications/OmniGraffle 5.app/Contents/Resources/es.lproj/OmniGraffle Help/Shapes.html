Figuras

Una figura es un objeto bidimensional que puede contener texto y puede rellenarse con colores e imágenes. Puede crear figuras mediante la herramienta Figura o la herramienta Lápiz, o bien arrastrarlas desde modelos. También puede usar la herramienta Texto para crear objetos de figura, que son especialmente útiles para contener texto.
Puede seleccionar una figura con la herramienta Selección y después usar los inspectores para editarla. También puede utilizar la herramienta Selección para mover, cambiar el tamaño o girar una figura. Si dispone de formas personalizadas dibujadas con la herramienta Lápiz, puede usar la herramienta Selección para editar sus puntos.
