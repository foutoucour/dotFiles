Colocación de gráficos dentro de una figura mediante el inspector Imagen

Puede utilizar este inspector para colocar una imagen dentro de una figura. (Tenga en cuenta que, en lugar de crear un objeto y después añadir una imagen, puede arrastrar un archivo de imagen desde otra aplicación directamente al lienzo.)
Utilice el menú local Imagen para colocar una imagen en la figura seleccionada. Una vez colocada una imagen, aparece el comando Quitar imagen en el menú.
Al buscar un archivo de imagen, tiene la opción de crear un alias (o enlace) de la imagen en lugar de añadir la imagen al archivo de OmniGraffle. En ese caso, el documento de OmniGraffle dependerá del archivo de imagen original; el alias no funcionará si borra el archivo original o abre el documento en otro ordenador. Si una imagen de un objeto procede de un alias, en el menú local Imagen estarán disponibles algunos comandos adicionales: puede Abrir el archivo de imagen original, Mostrar el archivo original en el Finder o dejar de usar el alias o Incrustar una copia de la imagen en el documento de OmniGraffle.
Otra manera de ajustar una imagen consiste en arrastrar un archivo de imagen desde otra aplicación y soltarlo en el cuadro de imagen o en el propio objeto de figura.
El primer botón muestra la imagen con sus proporciones naturales, con el nivel de zoom y la posición ajustadas en los controles situados debajo. El segundo botón hace que la imagen se estire para ajustarse al objeto. El tercer botón repite indefinidamente la imagen para formar un mosaico que rellena el objeto.
Los desplazamientos de posición están activados si está visualizando la imagen con sus proporciones naturales, como establece el primero de los tres botones de arriba. Puede escribir un porcentaje positivo o negativo, o simplemente arrastrar la imagen dentro del cuadro de imagen para indicar la zona que desee.
Los controles de zoom no están disponibles a menos que esté estirando la imagen para ajustarla a la figura. Puede introducir un porcentaje de zoom o simplemente arrastrar el regulador para hacer la imagen más grande o más pequeña.
Puede hacer transparente la imagen y dejar que el relleno del objeto aparezca disminuyendo el valor del campo de opacidad o arrastrando el regulador correspondiente.
También puede añadir una imagen a un lienzo. Sólo tiene que hacer clic en el lienzo en la barra lateral (de manera que aparezca resaltado en el color seleccionado) y, a continuación, usar el inspector Imagen normalmente.
