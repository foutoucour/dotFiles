編集メニュー

取り消す
OmniGraffle では、ユーザーが書類に行った変更を記憶します。このコマンドを選択し、リストで 1 ステップ戻ります。これは、リストの最後の動作を行っていないのと同じになります。ステップを連続して戻すには、このコマンドを必要な回数だけ続けて選択してください。
やり直す
前回行った操作が [取り消し] であった場合、効果的に [取り消し] を [やり直し] することで、もういちど先に進めて再度実行することができます。
カット
現在の選択項目を取り除き、クリプボードに置きます。これにより、他の場所にペーストできます。
コピー
現在の選択項目をクリップボードにコピーします。
形式を指定してコピー
PDF — 現在の選択項目の PDF コピーをクリップボードに置き、PDF を認識可能な任意のアプリケーションにペーストします。また、ペースト先のアプリケーションが LinkBack（これには、OmniGraffle そのものも含まれます）もサポートしている場合は、OmniGraffle に戻ってペーストしたコンテンツを編集することもできます。
TIFF — 現在の選択項目の TIFF コピーをクリップボードに置き、TIFF を認識可能な任意のアプリケーションにペーストします。
ベクター PICT — 現在の選択項目の ベクトル PICT コピーをクリップボードに置き、ベクトル PICT を認識可能な任意のアプリケーションにペーストします（これは、特に、Microsoft Word にペーストする場合に有用です。Microsoft Word には、OmniGraffle コンテンツを通常の [コピー] コマンドでペーストすると、正しいデータの取得が防止されるという微妙なバグがあります）。
AppleScript — 選択したオブジェクトの AppleScript 表現をクリップボードに置きます。結果をスクリプトエディタまたはほかのオブジェクトの アクションインスペクタにペーストし、オブジェクトを最作成するスクリプトを作成できます。また、出力は、OmniGraffle の AppleScript サポートがどのように動作するかを学習する上で役立ちます。
ペースト
クリップボードのコンテンツを、選択ツールで最後にキャンバス上でクリックした場所に配置します。（または、オブジェクトのテキストを編集する場合は、クリップボードからテキストを挿入します）。画像または一部のテキストがクリップボードにある場合は、図形またはテキストを含む新規図形オブジェクトが作成されます。
ペーストしてスタイルを合わせる
テキストを編集する場合、このコマンドを使用して、テキストをコピーした作成元のアプリケーションのスタイルを持ち込ますに、一部のテキストをペーストできます。かわりに、ユーザーがあたかも実際に入力したように、テキストのスタイルをすでにあるものと一致させます。
複製
現在の選択項目のコピーを作成し、クリップボードを使用せずにキャンバスに配置します。
このコマンドには特別な機能があります:
キャンバス上で何かを選択肢、複製します。
選択ツールを使用して、コピーを移動します。OmniGraffle では、オリジナルに基づきこの距離と方向を取得します。
必要なだけ複製できます。各新規コピーは、前のコピーからの同じ距離と方向性を保ちます。
削除
現在の選択項目を消去します。この実際の面から永久的になくなります（[取り消し] をのぞく）。
すべてを選択
キャンバス上で選択可能なものすべてを選択します。ロックしたレイヤーのオブジェクトは選択できないことを覚えておいてください。オブジェクトのテキストを編集している場合は、この操作により、かわりにオブジェクトのテキストがすべて選択されます。
選択
同様のオブジェクト — キャンバス上にある同じスタイル属性を持つすべてのオブジェクトを、現在選択しているオブジェクトとして選択します。
接続済みオブジェクト — 現在選択しているオブジェクトとラインにより直接的または間接的に接続されているすべての図形と、その間のすべてのラインを選択します。
上位オブジェクト — 接続ラインにより確立された階層に基づき、現在選択しているオブジェクトの上位のオブジェクトをすべて選択します。
下位オブジェクト — 接続ラインにより確立された階層に基づき、現在選択しているオブジェクトの下位のオブジェクトをすべて選択します。
アウトライン編集
このサブメニューのコマンド（および、特にそのキーボードショートカット）のコマンドは、アウトラインビューでの迅速な作業を容易にします。これらは、アウトラインビューが開かれており、その中にアクティブな選択項目がある場合にのみ使用できます。
マウスレス編集
これらのコマンドは、キーボードショートカットを呼び出す場合に最も有用です。マウスに触れなくても、すべての操作を実行できます。
図形を選択 — これらのコマンドは、現在選択している図形を基準にしたその位置に基づき図形を選択します。
図形を作成 — これらのコマンドは、現在選択している図形を基準にして新規図形を作成します。
キャンバス
新規キャンバス — 新しいキャンバスを書類に追加します。
キャンバスを複製 — 現在のキャンバスと同一の新規キャンバスを書類に追加します。
キャンバスを削除 — 現在のキャンバスを完全に除去します。
レイヤー
新規レイヤー — 新しいレイヤーを現在のキャンバスに追加します。
新規共有レイヤー — 新しい共有レイヤーを現在のキャンバスに追加します。
レイヤーを複製 — 選択したレイヤーと同一の新規レイヤーをキャンバスに追加します。
次のレイヤーと統合 — 選択したレイヤーにあるすべてのものを次のレイヤーと統合し、選択したレイヤーを削除します。
レイヤーを削除 — 現在のレイヤーを完全に除去します。 キャンバスの最後のレイヤーは削除できません。
すべてのコピーを削除 — このコマンドは、共有レイヤーを選択した場合にのみ使用できます。これは、書類全体を通じて、任意のキャンバスにある共有レイヤーのすべてのインスタンスを削除します。
レイヤー設定 — このコマンドは、キャンバスサイドバーのレイヤー上にあるクリック可能なアイコンを複製します。可視は、レイヤーのオブジェクトを表示または非表示にします。プリント可能は、レイヤーのオブジェクトをプリント時に含めるかどうかをコントロールします。また、ロック解除は、レイヤーのオブジェクトを選択または編集可能にするかどうかをコントロールします。
レイヤー上のすべてを選択 — 選択したレイヤーのすべてのオブジェクトを選択します。
選択項目をレイヤーに移動 — 選択したすべてのオブジェクトを選択したレイヤーに移動します。
表
行を挿入 — 表のセルを選択している場合、このコマンドを選択してセルの新規行をテーブルに追加します。選択したセルを含む行の直前に新規行が表示されます。
列を挿入 — 表のセルを選択している場合、このコマンドを選択して、セルの新規列をテーブルに追加します。選択したセルを含む行の直前に新規行が表示されます。
行を選択 — 表のセルを選択している場合、このコマンドを選択して、セルを含む行全体を選択します。
列を選択 — 表のセルを選択している場合、このコマンドを選択して、セルを含む列全体を選択します。
磁石
磁石をコピー— 選択したオブジェクトの磁石配置をクリップボードに置くと、異なるオブジェクトにペーストできます。
磁石をペースト — クリップボードに磁石の配列がある場合は、選択したオブジェクトにそれを適用します。既存の磁石は、新規のものに置換されます。
磁石を削除 — 選択したオブジェクトのすべての磁石を排除します。
図形
前面の重なり部分を取り去る — 選択した図形を取り出し、前面の図形を使用して最も背面の図形に穴を開けます。 [配置] メニューの前面/背面コマンドを使用すると、図形を再配置できます。
図形を融合— 選択した２つ以上の図形を融合し、内部のラインを排除して単一の個体を作ります。
重なり部分の図形を作成 — 選択した図形の重なり合う領域から図形を作成します。
検索
検索... — これは、OmniGraffle の検索パネルを呼び出します。特定の文字列を検索するか、または正規表現を使用して文字列のクラス全体を一致させることができます。特定の文字列を検索する場合は、[大文字／小文字を無視] を選択し、大文字または小文字には関係なく文字を一致させます。[語句で区切る] を選択し、単語の一致部分を回避します。正規表現でグループを使用する場合は、表現全体を選択（または置換）するか、グループのいずれか１つを選択します。ポップアップメニューを使用して指定します。[前へ] および [次へ] の目的について、OmniGraffle では、オブジェクトを左から右、上から下という順序で考慮します。
次を検索 — これを選択し、[次へ] ボタンをクリックすると、[検索] パネルに最後に入力したテキストの次の検索を実行します。
前を検索 — これを選択し、[前へ] ボタンをクリックすると、[検索] パネルに最後に入力したテキストの前の検索を実行します。
選択部分を検索 — オブジェクトのテキストの一部を選択してこのコマンドを選ぶと、選択したテキストが [検索] パネルに入力されます。以前に入力されていたものは置換されます。次に、[次を検索] および [前を検索] コマンドを使用して、次または前のテキストを検索します。
選択部分へジャンプ — 選択したオブジェクトが表示されるように、ビューを移動します。
スペルと文法
このサブメニューには、Mac OS X の標準のスペルおよび文法ユーティリティが含まれています。[前へ] および [次へ] の目的について、OmniGraffle では、オブジェクトを左から右、上から下という順序で考慮します。
変数を挿入
図形オブジェクトのテキストを編集しているときに、これらのコマンドのいずれかを選択して、特別なタグを挿入します。テキストの編集が完了すると、オブジェクトそのものの一部の側面、キャンバス、ページ、または書類のを表すようにタグが変更されます。タグが参照しているデータが変更されると、タグが自動的にアップデートされます。べーじ番号などの一部の変数は、共有レイヤーに配置し、次に各キャンバスでそのレイヤーを共有するのに役立ちます。
日付変数のいずれかを使用している場合は、Cocoa の日付書式トークンを使用して日付書式をカスタマイズできます。たとえば、"2005 - 10 - 06 16:00" のような日付を設定する場合は、次のように入力します:
<%date %Y - %m - %d %H:%M %>
キャンバス名 — オブジェクトが配置されているキャンバス名です。
書類の名前 — オブジェクトが配置されている書類名です。
現在の日付 — この瞬間の日時です。これは、ドキュメントがプリントされた時期を確認するのに便利です。
書類の作成日 — Mac OS X ファイルシステムに準拠して書類が最初に作成された日付です。
書類の変更日 — Mac OS X ファイルシステムに準拠し、書類が最後に保存された日付です。
作成者 — Mac OS X ファイルシステムに準拠し、ファイルを作成したユーザーの名前です。
変更者 — Mac OS X ファイルシステムに準拠し、ファイルを最後に保存したユーザーの名前です。
ページ番号 オブジェクトが配置されているページの番号です。キャンバスサイズインスペクタおよびページ設定パネルの設定に基づきます。
総ページ数 — 書類全体のページ数です。これをページ番号タグとともに使用すると、"Page 2 of 5" などの結果が得られます。
X 位置— 基点からオブジェクトへの水平方向の距離です。配置インスペクタと同様です。
Y 位置— 基点からオブジェクトへの垂直方向の距離です。配置インスペクタと同様です。
幅 — オブジェクトの水平方向のサイズです。配置インスペクタと同様です。
高さ — オブジェクトの垂直方向のサイズです。配置インスペクタと同様です。
ラインの長さ— このタグは、若干他とは異なります。タグを含んでいる図形がラインラベルの場合は、ラインの長さを示します。図形オブジェクトがラインに添付されていない場合は役に立ちません。
LinkBack 項目を編集 — LinkBack コンテンツをほかのアプリケーションからペーストしている場合は、オブジェクトを選択肢、このコマンドを使用して元のアプリケーションのコンテンツを開きます。
特殊文字 — 標準の Mac OS X 文字パレットを呼び出します。
