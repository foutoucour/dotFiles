他の形式への書き出し

OmniGraffle ダイアグラムから、複数の異なるタイプのファイルを作成できます。[ファイル] メニューから [書き出し...] を選択します。エクスポートする場合、新規ファイルがディスクに書き込まれ、元の書類はそのまま画面に残されます。もちろん、ダイアグラムのすべての側面がすべてのファイルフォーマットで表現できるわけではありません。
利用できるファイルフォーマットは下記の通りです：
OmniGraffle 書類 — OmniGraffle 書類です。ファイルを読み取り専用（編集不可）にし、他の人のコンピュータ上に正しく表示されるように、ファイル内にリンクされた画像を含めるように選択できます
PDF 形式 — Portable Document Formatの略。これは、すぐれたクロスプラットフォームフォーマットで、ベクターコンポーネントとビットマップコンポーネントの両方をサポートしています。
TIFF 形式 — Tagged Image File Formatの略。このフォーマットは透明性をサポートしています。
PNG 形式 — Portable Network Graphicsの略。Web 上ではよく使用されるフォーマットです。このフォーマットでは、ファイルサイズを縮小しながらも画像の詳細を保持するため、可逆圧縮を使用しています。これは、透明性をサポートしています。
JPEG 形式 — よく使用される圧縮画像フォーマットです。画像の品質を落とすと、ファイルサイズを小さくできます。
EPS 形式 — Encapsulated PostScriptの略。ベクター画像形式です。
HTML イメージマップ — ハイパーテキストファイルおよびJPEG、PNG、または GIF 画像。元の OmniGraffle 書類のURL アクションは、画像をクリックしてそれに準拠できるように、リンクとしてイメージマップにコード化されます。
OmniOutliner 3 — 図形間のコネクションラインを使用して階層を作成し、テキストアウトラインとしてダイアグラムを表現します。これは、厳密なツリーのような構造を持たないダイアグラムには同様には作用しません。
SVG ベクタードロー — Scalable Vector Graphicsの略。XML ベースのインターネットにおけるオープンスタンダードです。
PICT ベクター画像 — レガシー Macintosh グラフィックフォーマット。
BMP ビットマップ画像 — レガシーグラフィックフォーマット。
OmniGraffle ダイアグラムスタイル、OmniGraffle テンプレート、OmniGraffle ステンシル — OmniGraffle 用のリソース。
Visio® XML 書類 — Microsoft® ダイアグラム作成アプリケーション用の XML ベースのファイルフォーマット。
選択するファイルフォーマットにより、異なるオプションがあります。ほとんどのフォーマットには、書類をどの程度書き出すかを選択できる [書き出しの範囲] オプションがあります。
選択部分 — 現在選択されているオブジェクトのみを書き出します。
すべてのオブジェクト — キャンバスのすべてのオブジェクトを書き出しますが、空の領域は書き出しません。
リージョン — 書き出しを行うキャンバス上の詳細なリージョンを指定します。
現在のキャンバス — キャンバス全体を書き出します。
書類全体 — 各キャンバスに画像ファイルを１つ作成し、すべてを書き出します。
また、選択するフォーマットによっては、書き出しのときに、実際のサイズよりも小さい/大きいサイズ、解像度、圧縮レベル、および背景を透明にするかなどを選択できます。
