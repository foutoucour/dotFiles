Creating shapes with the Shape tool

This tool is quite simple; use it to create shape objects. Drag across the canvas until you get the size you want.
Hold Shift while dragging to make a shape of equal width and height (a perfect square, a perfect circle, and so on).
Once you have created a shape, you can select it and change its attributes with the Style and Properties inspectors.
