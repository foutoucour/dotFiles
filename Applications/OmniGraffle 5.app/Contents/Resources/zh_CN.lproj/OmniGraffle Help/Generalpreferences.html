使用通用预置自定 OmniGraffle 的行为方式

通过从 OmniGraffle 菜单选择“预置”，即可访问“通用”预置。
启动
如果您想让 OmniGraffle 记住打开的文稿，请选择“打开上一次退出时打开的文稿”。
当您未让 OmniGraffle 记住打开的文稿，或者上一次退出时没有打开的文稿，您可以选择“若未打开其他任何文稿，则建立新文稿”来开始使用新文稿。
新建文稿
选择“显示模板选择器”，这样每次在建立新的文稿时都会出现一个窗口，以供您选择模板。 选择“使用模板”，如果您希望略过选择器，则请选择预设模板。 或者，您可以一直使用“文件”菜单中的“模板选择器”命令来打开模板选择器。
文字编辑
当您在编辑对象的文字时，请设定您按下 Return 和 Tab 键会进行的动作。 无论选择什么样的方式，您都可以在按住 Option 的同时按下该键来进行相反的动作。
所选内容
小对象将取得较小的句柄，当您选择它们时就会出现；若您确实喜欢小对象，您可以点按“总是使用小的选择句柄”，这样就可以始终看到它们。
当您在没有使用图像或颜色填充的形状对象中间点按时，便可选择对象或位于其后的项。 使用“穿过未填充的对象点按”注记格来显示您偏好哪种方式。
帮助标签
当您移动或调整对象大小时，会出现一个小的帮助标签让您准确得知对象的新位置和大小。 也就是说，它会一直出现，除非您感到厌烦并将其关闭。
备份
当您处理文件时，OmniGraffle 可以定期存储文件的备份；若发生什么意外事件，您还可以恢复您的部分作业。
列表工具条
如果您正在使用 AppleScript 访问 OmniGraffle 对象，或者您喜欢使用唯一 ID ，那么您可以选择“显示对象标识号”将栏位添加到列表工具条。
预置重置按钮
每个预置面板左下方都有一个“重置”按钮。 点按此按钮来将面板上的所有设置回复为预设值。 要重置所有预置面板上的设置，请在点按“重置”按钮的同时按住 Option 键。
