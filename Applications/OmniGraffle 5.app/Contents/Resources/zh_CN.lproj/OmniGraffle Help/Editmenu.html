编辑菜单

还原－OmniGraffle 会记住您对文稿所做的一系列更改。选择此命令可在列表中往后移一步，令文稿变得和您未在列表中执行上一个动作一样。 一直选择它可及时返回之前的操作。
重做－如果您上次所做的动作是还原某些内容，则可以向前进一步并重新再做，以便有效地撤销还原动作。
剪切－移去当前的所选内容，然后将其放在剪贴板上，这样您就可以将其粘贴在其他地方。
拷贝－将当前的所选内容拷贝到剪贴板上。
拷贝为
PDF－将当前所选内容的 PDF 拷贝放置在剪贴板上，以粘贴到任何可识别 PDF 的应用程序。 如果您所粘贴的应用程序还支持 LinkBack（包括 OmniGraffle 本身），您甚至可以返回并编辑在 OmniGraffle 中粘贴的内容。
TIFF－将当前所选内容的 TIFF 拷贝放置在剪贴板上，以粘贴到任何可识别 TIFF 的应用程序。
向量 PICT－将当前所选内容的“向量 PICT”拷贝放置在剪贴板上，以粘贴到任何可识别“向量 PICT”的应用程序（这对粘贴到 Microsoft Word 中尤其有用， 此时会发生一个奇怪的错误，当您使用普通的“拷贝” 命令粘贴 OmniGraffle 内容时，该错误会阻止 Microsoft Word 获得正确的数据）。
AppleScript－将所选对象的 AppleScript 表示放置在剪贴板上。 您可以将结果粘贴到“脚本编辑器”或另一个对象的动作检查器中，以制作可重新建立对象的脚本。 对于研究 OmniGraffle 的 AppleScript 支持如何运作方面，输出的结果也很有用。
粘贴－将版面上剪贴板中的内容放在使用选择工具点按的最后位置。 （或者，您在编辑对象的文本时，可插入来自剪贴板的文字）。 如果剪贴板中有图像或部分文字，则会建立包含该图像或文字的新形状对象。
粘贴并匹配样式－在编辑文本时，您可以使用此命令粘贴部分文本，而无需附带该文本在来源应用程序中的样式。 相反，它会匹配您已有的文本样式，就像您亲自在此输入的一样。
复制－在不使用剪贴板的情况下，拷贝当前的所选内容，并将其放在版面上。
此命令有一项特别的功能
选择版面上的内容然后进行复制
使用选择工具来移动拷贝；OmniGraffle 了解拷贝与原始文件的距离和方向。
您可以根据喜好反复复制，每个新拷贝皆与前一个拷贝的距离和方向相同。
删除－删除当前的所选内容，将其从现有的位置永远清除（除非您执行还原动作）。
全选－选择版面上可以选择的所有内容。请记住，您无法选择锁定层上的对象。 如果您正在编辑对象的文字，这将会选择对象的所有文字。
选择
类似对象－选择版面上的所有对象，这些对象与当前所选的对象有相同的样式属性。
已连接的对象－选择所有形状，这些形状是由到当前所选对象的线条以及所选对象之间的所有线条直接或间接连接而成。
原始对象－选择当前所选对象的所有原始项（以连线建立的层级为基础计算的）。
派生对象－选择当前所选对象的所有派生项（以连线建立的层级为基础计算的）。
大纲－此子菜单中的命令（特别是它们的键盘快捷键）使您能够更轻松地在大纲视图中快速地进行工作。仅当大纲视图打开，且其中有一个活跃的所选内容时，这些命令才可以使用。
无鼠标编辑－这些命令会在您调用其键盘快捷键时变得最为有用；您甚至无需触按鼠标即可执行所有的动作。
选择形状－这些命令会选择一个当前所选的形状位置周围的形状。
建立形状－这些命令会利用您当前所选的形状，在其位置的周围建立新的形状。
版面
新建版面－添加一个新的版面至文稿。
复制版面－添加一个与当前版面相同的新版面至文稿。
删除版面－完全移去当前版面。
层
新建层－添加一个新的层至当前版面。
新建共享层－添加一个新的共享层至当前版面。
复制层－添加一个与当前所选层相同的新层至版面。
向下合并层－将所选层上的所有内容移到下一层，然后删除所选层。
删除层－完全移去当前层。 您无法删除版面中的最后一个层。
删除所有拷贝－仅当选择共享层时，此命令才可用。 它会删除整个文稿中任何版面上共享层的所有实例。
层设置－这些命令可以复制版面工具条中层上的可点按图标。 “可见”命令会显示或隐藏层上的对象，“可打印”命令控制在打印时是否包含层上的对象，而“解锁”命令控制层上的对象是否可选或编辑。
在层内全选－选择所选层上的所有对象。
将所选内容移到层－将所有选定的对象移到所选层。
表格
插入行－在选择表格的某个单元格后，请选择此命令将新的单元格行添加到表格。 新的行会马上出现在包含所选单元格的行之前。
插入栏－在选择表格的某个单元格后，请选择此命令将新的单元格栏添加到表格。 新的行会马上出现在包含所选单元格的行之前。
选择行－在选择表格的某个单元格后，请选择此命令来选择含有该单元格的整行。
选择栏－在选择表格的某个单元格后，请选择此命令来选择含有该单元格的整栏。
磁化物
拷贝磁化物－将所选对象的磁化物排列放置在剪贴板上，这样您就可以将其粘贴到其他对象。
粘贴磁化物－如果剪贴板上已存在磁化物排列，请将其应用到所选对象。 新磁化物会替换掉所有旧的磁化物。
删除磁化物－移去所选对象上的所有磁化物。
形状
形状补集－用所选的形状和前面的形状，将它们合并为一个形状。 您可以使用“排列”菜单中的“移动/发送”指令来重新排序形状。
形状并集－将两个或更多的所选形状合并在一起，这样会清除内线以制作一个的固定的形状。
形状交集－从所选形状的重叠区域建立新形状。
查找
查找－此控制项会呼出 OmniGraffle 的“查找”面板。您可以搜索特定的文字字符串，也可以使用正则表达式来搜索符合整个类型的字符串。 搜索特定字符串时，选择“忽略大小写”可找到匹配但忽略大小写的词；选择“整个词”可避免找到仅匹配词的部分的结果。 若要选择（或替换）整个正则表达式，或成组中的某部分，您可以在正则表达式中使用成组；请使用弹出式菜单来选择指定。 对于“上一个”和“下一个”的含义，OmniGraffle 认为对象的排列顺序是从左到右、从上到下的。
查找下一个－选择此项可查找上一次在“查找”面版中输入的文本所出现的下一个位置，功能和您点按“下一个”按钮一样。
查找上一个－选择此项可查找上一次在“查找”面版中输入的字符串所现的上一个位置，功能和您点按“上一个”按钮一样。
输入选择－选择对象中的部分文字，接着选择此命令将所选文字置入“查找”面板，以替换掉之前的任何内容。 然后您可以使用“查找下一个”和“查找上一个”命令来搜寻文字出现的其他地方。
滚动到所选内容－ 移动视图范围以使所选对象可见。
拼写和语法
此子菜单包含 Mac OS X 的标准拼写和语法实用程序。 对于“上一个”和“下一个”的含义，OmniGraffle 认为对象的排列顺序是从左到右、从上到下的。
插入变量
在编辑形状对象的文本时，请选择其中一种命令插入特殊标签。 当您完成文字编辑后，标签会更改以表示对象本身的高宽比、版面、页面或文稿。 若标签涉及到的数据有所更改，标签会自动自行更新。 将部分变量（例如“页码”）置于共享层上，然后将该层与所有版面共享十分有用。
使用日期变量时，您可以使用 Cocoa 的日期格式令牌。 例如，要得到如“2005 - 10 - 06 16:00”这样的日期，您可以输入以下内容：
<%日期 %年 - %月 - %d %时:%分 %>
版面名称－对象所在版面的名称。
文稿名称－对象所在文稿的名称。
当前日期－日期精确到每分钟，这样很容易就知道文稿打印的准确时间。
文稿创建日期－最初创建文稿的日期（基于 Mac OS X 文件系统）。
文稿修改日期－上一次存储文稿的日期（基于 Mac OS X 文件系统）。
创建者－创建文件的用户名称（基于 Mac OS X 文件系统）。
修改者－上一次存储文件的用户名称（基于 Mac OS X 文件系统）。
页码－对象所在的页数（基于版面尺寸检查器和页面设置面板中的设置）。
页面总数－整个文稿中的页面数。 请将它与“页码”标签搭配使用，以得到如“2 / 5 页”这样的页码。
X 位置－对象到原点的水平距离，正如在几何形状检查器中一样。
Y 位置－对象到原点的垂直距离，正如在几何形状检查器中一样。
宽度－对象的水平尺寸，正如在几何形状检查器中一样。
高度－对象的垂直尺寸，正如在几何形状检查器中一样。
线条长度－此标签有点特别。 如果含有此标签的形状是线条标签，则它会显示该线条的长度。 如果形状对象没有附在线条上，则此标签就不太实用了。
编辑 LinkBack 项－如果您已从另一个应用程序粘贴了 LinkBack 的内容，请选择对象并使用此命令来打开原始应用程序中的内容。
特殊字符 －呼出标准的 Mac OS X 字符调板。
