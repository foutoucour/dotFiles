演示文稿模式

演示文稿模式可从“显示”菜单中取得；它将您的文稿填满整个荧幕，这样您就可以轻松向他人展示文稿。
每个版面都可以像幻灯片一样播放；使用方向键在版面之间进行移动。 您也可以按下 Return 或 Enter 键，或通过点按来移动到下一个版面。 如果您的“演示文稿”预置设置为点按时高亮显示对象，则点按操作便不会前进到下一个版面；您仍旧可以点按版面的空白区域前进。 如果将鼠标光标移动到荧幕底部，则会出现导航界面。您可以使用该界面上的按钮来前进或后退、退出或直接移动到特定版面。
在演示文稿过程中，您可以高亮显示对象来强调它们。 使用“演示文稿”预置来控制高亮显示方式。
要退出演示文稿模式，请按下 Escape 键或点按将鼠标指针移动到荧幕底部时显示的 X 按钮。
在动作检查器中设置的动作会在演示文稿模式下执行。 当您点按已指定动作的对象时，对象会执行动作，就像您已使用浏览工具来点按它一样。
“演示文稿”预置
在演示文稿模式中，您可以高亮显示版面上的对象。
请选择是在将指针指向时还是在点按时高亮显示对象，还是永不高亮显示对象。 您也可以选择显示已在动作检查器中设置动作的对象的标徽。
点按颜色格来选择高亮显示的颜色。 建议您使用 Developer 调板中的selectedControlColor；此颜色将总是与在 Mac OS X 的“外观”预置中所选的高亮颜色相匹配。 使用滑块来控制高亮显示的粗细度。
如果您打开多个窗口，最前面的窗口将用于演示文稿。 若您有两个显示器，您可以在一个屏幕上演示文稿，而在另一个屏幕上编辑文稿。
从"窗口"菜单中选择“新建‘文稿’窗口”（其中“文稿”是指您的文稿标题）。
将窗口拖移到第二个显示器。
点按第二个显示器的窗口确定它位于最前面。
输入演示文稿模式。 第一个显示器常规文稿窗口，第二个显示器则显示演示文稿。
现在您可以在第一个显示器上编辑文稿，在第二个显示器上的演示文稿会自行更新。
