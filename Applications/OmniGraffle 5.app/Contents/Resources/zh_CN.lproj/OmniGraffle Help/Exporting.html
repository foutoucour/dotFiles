输出为不同的格式

您可以从 OmniGraffle 图表制作出几种不同类型的文件。 从“文件”菜单选择“输出”。 当您输出时，新文件就被写入到磁盘，在荧幕上留下原始文稿。 当然，不是图表的所有特征都可以用所有文件形式表现出来。
可用的文件格式有：
OmniGraffle 文稿－一种 OmniGraffle 文稿。 您可以制作只读（即不可编辑的）文件，也可以选择在文件中加入带有链接的图像，以便可在他人电脑上正确显示。
PDF 向量图像－可携式文档格式。 这是精密的跨平台格式，可以良好支持向量组件和点阵。
TIFF 点阵图像－标记图像文件格式。 此格式支持透明度。
PNG 点阵图像－可携式网络图形，Web 的一种流行格式。 在减少文件大小时，此种格式会使用无损压缩以保留图像的细节信息。 它支持透明度。
JPEG 点阵图像－一种流行的压缩图像格式。 您可以降低图像的质量来取得较小的文件。
EPS 向量图像－封装的 PostScript。 这是专业打印中的通用格式。
HTML 图像映射－一种超文本文件和 JPEG、PNG 或 GIF 图像。 原始 OmniGraffle 文稿中的 URL 动作将作为链接编入图像映射中，这样点按图像就可以连接 URL。(（请注意不包括有关线条的动作，但是包括有关线条标签的动作）
OmniOutliner 3－将图表展示为文字大纲，使用形状之间的连线来建立层次。 这同样也无法用于没有严格树的图表。
SVG 向量绘图－一种使用 XML 的开放式互联网标准。
PICT 向量图像－一种传统的 Macintosh 图形格式。
BMP 点阵图像－一种传统的图形格式。
OmniGraffle 图表样式、OmniGraffle 模板 (Template)、OmniGraffle 模板 (Stencil)－OmniGraffle 的资源。
Visio® XML 文稿－基于 XML 的文件格式，专用于 Microsoft® 图表应用程序。
根据选择的文件格式，您可以有不同的选项。 大部分都提供了“输出区域”选项，您可以选择要输出的文稿部分。
当前所选内容－仅输出现在所选的对象。
所有对象－输出版面上的所有对象，但不输出空白区域。
区域－指定要输出版面的精确区域。
当前版面－输出整个版面。
整个文稿－输出所有内容，为每个版面建立一个图像文件。
根据选择的格式，您也可以选择用于输出比实际尺寸、分辨率、压缩级别小或大的比例，以及是否要使背景透明。
