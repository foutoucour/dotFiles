显示菜单

开始演示文稿－输入带有最前面文稿的演示文稿模式。
缩放
放大－ 往上移到下一个最近的缩放比例，让版面看起来更大。
缩小－往下移到下一个最远的缩放比例，让版面看起来更小。
缩放到实际大小－移动到 100% 缩放，这样版面就会根据版面单位显示同实际大小一样的版面。
缩放到所选内容－缩放到使用当前所选对象来填充视图的比例和位置。
上一次缩放－返回到当前缩放比例之前的上一个缩放比例。
保持符合窗口大小－缩放到您在工作时可看见版面上每个对象的比例和位置。
校样设置－使用此子菜单来设置打印或校样装置，以及其指定的描述文件。 默认的校样描述文件为 Mac OS X 的默认校样描述文件（通常为“普通 CMYK 描述文件”），但您可以使用您电脑上的其他打印或校样装置的描述文件。
校样颜色－打开软校样，使用“校样设置”子菜单中所设置的设备。 这会将额外的步骤添加到图像和颜色的显示路径。 首先，每种颜色都将从正确的三种文稿描述文件的其中一种转换为校样描述文件，然后从校样描述文件转换为显示描述文件。 理论上，只要它复制了 Mac OS X 中的打印路径，您就可以在荧幕上看到打印输出文件的整体效果。 尽管这主要取决于您打印机的色彩描述文件的质量。 对于许多用户喷墨打印机，例如，打印机描述文件称可以复制任何颜色（可避免 ColorSync 以及替代在打印机驱动程序中自己的颜色增强功能）。 对于这些打印机，软校样可能无法做出比较明显的更改。
附加项－版面上可能存在您想要查看的几个项（而不是对象本身）；请从此列表中选择某项以进行显示或隐藏。
动作－在动作检查器中指定有动作的任何对象都将获得一个动作标签，其紧靠它的右上角。
方格－这是在方格检查器中定义的方格。 请注意，您可以贴紧方格，而无论该方格是显示还是隐藏的。
参考线－这些参考线为手动参考线，您可以设置它来对齐版面上的对象。
磁化物－一直选择它来查看所有对象的磁化物。
备注－任何带有在备注检查器中设置的备注的对象都将获得一个备注标签，其紧靠它的右上角。
原点－它为十字准线，可表示版面的原点。
分页符号－它们为线条，可表示不同页面之间的边框。
标尺－标尺可帮助您调整版面上对象的大小，并且还包括用于对对象进行一般编辑的检查器条。
工具条－选择工具条的名称以显示或隐藏它。 版面有自己的工具条，而大纲视图和列表视图共享一个工具条。
版面－文稿中版面和层的列表。
大纲－大纲工具条，显示版面上对象的层次。
列表－列表工具条，显示每个层上对象的排序。
显示版面
编辑下一个版面－切换到当前编辑的版面之后的版面。
编辑上一个版面－切换到当前编辑的版面之前的版面。
自定工具条－创建控制项的表单，您可以将这些控制项拖移到工具条，与在其他 Mac OS X 应用程序中一样。
显示/隐藏工具条－设置文稿窗口工具条的可见性。
