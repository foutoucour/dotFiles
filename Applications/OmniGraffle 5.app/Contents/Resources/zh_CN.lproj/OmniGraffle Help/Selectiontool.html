使用选择工具选择、移动和编辑对象

选择工具是最基本的工具，它的外观与普通的 Mac OS X 鼠标指针相似。
选择对象
点按版面上的某个对象来将其选中。
选择对象后，就可以使用“样式”检查器和“属性”检查器。 您可以使用检查器来更改所选对象的所有设置。
要同时选择不同对象，请点按版面的空白区域，然后将选择框拖到对象上。 如果按住 Option 键，则只会选择完全位于矩形内部的对象。 按住 Command 键或 Shift 键点按对象以将其添加到所选内容，或将其从所选内容中移去。
要选择成组、表格或子图中的项目，请先点按一次来选择整个成组，然后再次点按以选择其中的项目。 版面的其余部分会变暗以显示您处于成组编辑模式中。 点按成组外部的任意位置来退出成组编辑模式。
移动对象
要移动对象，请使用选择工具来点选它并将其拖移到新位置。 若您想要移动某根线条，您可能需要将它的结尾点从其依附的对象中分离。 要水平或垂直移动对象直线，拖移时请按住 Shift 键。 要旋转对象，请按住 Command 键并拖移某个选择句柄；您也可以按住 Shift 键将旋转量控制在 15 度。
如果您的 MacBook 具有 multi-touch 功能，则可以采用两只手指的旋转手势来旋转所选对象。
要非常精确地移动所选对象，请按方向键。 按住 Option 或 Shift 来进行较大规模的移动。
如果您已启用智能参考线，当您拖移对象时它们就会出现，帮助您整齐地排列对象以及保持平均的间距。 您开始拖移后，请按住 Command 键以暂时启用或停用智能参考线。
要拖移对象副本而非对象本身时，请在拖移时按住 Option 键。
缩放对象
当您选择可以调整大小的项目时，如形状或成组，便会出现八个选择句柄。 拖移其中某个句柄来更改对象的大小。 调整对象大小时，请按住 Shift 键来保持其比例，以避免造成变形。 调整对象大小时，请按住 Option 键来保持其中心点位置。
如果您的 MacBook 具有 multi-touch 功能，则可以将手指张开来使所选对象变大，或者将手指收合来使其变小。
如果您选择了表格，边缘句柄就会看起来像小方格一样。 您可以拖移它们来更改表格中的行数或栏数。 拖移角部句柄，即可按常规方式调整表格大小。
如果您已启用智能参考线，当您拖移对象时它们就会出现，帮助您均匀地调整对象尺寸。 您开始拖移后，请按住 Command 键以暂时启用或停用智能参考线。
扩展图表
要建立所选对象的子项，请按住 Command 键并点按版面上的空点。 要建立兄弟项，按住 Command ＋ Option 键并点按版面上的空点。 新的形状会使用与原始形状相同的样式。 连接形状的线条会使用已连接到原始形状线条的样式；若没有线条存在，它将使用线条工具的当前样式。
编辑点
当您选择连线或自定形状时，便会显示指定其形状的点；这些点显示为很小的菱形。
任何线条（使用线条工具建立）都有两个点：来源位置上的红点以及目标位置上的绿点。 线条也可能穿过数个蓝色中间点。 拖移其中一个点来移动它；通过拖移来源点或目标点，您可以使线条连接到其他对象或中断与其他对象的连接。
自定形状（使用笔工具建立）和贝塞尔类型线条在每个顶点处都有控制点。 点按控制点来选择它；如果控制点隐藏在选择句柄之后，请按住 Option ＋ Command 键点按以进行选取，或先选择一个不同控制点来隐藏句柄。 若控制点有贝塞尔句柄，当您选择点时就会出现。 拖移其中一个句柄便可同时调整它们，按住 Option 键并拖移可以一次只调整一个。 按住 Shift 键并拖移以 45 的增量来调整句柄角度。 按住 Command 键来拖移点以显示其句柄。 若您将句柄拖移到非常接近点本身，句柄将全部消失。
将点添加到已有线条的预设方式为连按该线条，但您可以在通用预置中进行更改。
要将点添加到自定形状，请沿着形状的笔划连按某处。
要从线条或自定形状删除点，请先选择它，然后按下 Delete 键或从“编辑”菜单选择“删除”。
