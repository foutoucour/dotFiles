Menu OmniGraffle

Informazioni su OmniGraffle — Ottieni informazioni sulla versione di OmniGraffle che stai utilizzando.
Verifica aggiornamenti — Normalmente, OmniGraffle verifica la presenza di versioni aggiornate in base alle opzioni impostate nelle preferenze Aggiornamento. Utilizza questo comando per verificare in questo momento la presenza di aggiornamenti. Se è presente una versione aggiornata del software, ti verrà chiesto di scaricarla.
Preferenze — Apri la finestra delle preferenze OmniGraffle; puoi utilizzare questa finestra per personalizzare le funzionalità dell'applicazione in base alle tue esigenze.
Licenze — Mostra la finestra di dialogo Licenze, nella quale puoi verificare le licenze software di OmniGraffle. Per ulteriori informazioni sul funzionamento della licenza, consulta la sezione Licenze.
Servizi — Il menu Servizi raccoglie un insieme di servizi applicabili a tutto il sistema offerti dalle applicazioni che sono state installate. OmniGraffle interagisce piuttosto bene con Mac OS X, quindi dovresti poter utilizzare i servizi standard dall'applicazione stessa.
Nascondi OmniGraffle — Lasca aperta l'applicazione OmniGraffle, ma metti via temporaneamente le sue finestre mentre lavori con un'altra applicazione.
Nascondi altre — Nascondi tutte le finestre delle altre applicazioni aperte per concentrarti su OmniGraffle.
Mostra tutte — Mostra tutte le finestre delle applicazioni nascoste utilizzando i comandi Nascondi.
Esci da OmniGraffle — Scegli questo comando se vuoi veramente chiudere OmniGraffle. Se dei documenti non registrati sono aperti, ti verrà chiesto di registrare o ignorare le modifiche.
