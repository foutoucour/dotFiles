Aggiungere dati agli oggetti con le impostazioni Nota

Le impostazioni Nota contengono un campo per l'inserimento di testo associato con l'oggetto selezionato e una tabella di dati personalizzati.
La nota può essere formattata come RTF, ciò significa che puoi utilizzare tutti gli stili di font e i colori usati da qualsiasi altra parte in OmniGraffle. Quando posizioni il puntatore sopra un oggetto che ha una nota, il testo della nota apparirà sotto forma di etichetta d'aiuto sopra l'oggetto.
Puoi utilizzare la tabella dei dati personalizzati per conservare le tue informazioni sull'oggetto. I dati personalizzati vengono archiviati come abbinamenti chiave/valore: la Chiave è un'etichetta che identifica il tipo di informazioni da archiviare; il Valore rappresenta l'informazione stessa.
Per esempio, immagina di avere il diagramma di un network di computer e di voler assegnare a ogni componente un codice identificativo. Fai clic sulla riga segnaposto o sul pulsante + per creare un nuovo abbinamento chiave/valore. Nella colonna Chiave, dovrai inserire il codice identificativo e nel campo Valore, dovrai inserire, ad esempio, A1181. Questi dati non influenzano la funzionalità di OmniGraffle; si tratta semplicemente di un metodo per archiviare dati personali relativi a oggetti del diagramma. Per rimuovere una riga di dati, fai clic sul pulsante 'x' sul lato destro della riga.
Inoltre, puoi modificare la nota di un'area di disegno. Fai clic sull'area di disegno nella barra laterale (in modo che appaia evidenziata nel tuo colore di selezione), quindi utilizza normalmente le impostazioni Nota.
Le note possono essere trovate utilizzando la funzionalità di ricerca Spotlight di Mac OS X, nel caso in cui volessi trovare i tuoi diagrammi OmniGraffle contenenti determinate parole.
