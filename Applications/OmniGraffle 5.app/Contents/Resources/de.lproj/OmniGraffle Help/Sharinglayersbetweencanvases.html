Ebenen auf Arbeitsflächen gemeinsam verwenden

Sie können eine Ebene auf mehreren Arbeitsflächen gemeinsam verwenden. Wenn Sie in diesem Fall die Ebene verändern, wirkt sich dies auf jede Arbeitsfläche aus, die diese Ebene verwendet.
Gehen Sie folgendermaßen vor, um eine Ebene auf mehreren Arbeitsflächen gemeinsam zu verwenden:
Wählen Sie "Neue gemeinsame Ebene" aus dem Einblendmenü "Aktion" am unteren Rand der Arbeitsflächen-Seitenleiste.
Die gemeinsame Ebene wird durch eine besondere Farbe und ein Symbol auf der linken Seite hervorgehoben, um Sie auf den besonderen Charakter dieser Ebene hinzuweisen. Die exakte Farbe richtet sind nach der Hervorhebungsfarbe, die in den Mac OS X-Systemeinstellungen festgelegt wurde. Falls Sie z. B. blau als standardmäßige Hervorhebungsfarbe verwenden, sind Ihre gemeinsamen Ebenen orange.
Das Hinzufügen und Bearbeiten von Objekten auf dieser Ebene erfolgt auf die gewohnte Weise.
Wann immer Sie die Ebene gemeinsam verwenden möchten, ziehen Sie sie einfach auf eine andere Arbeitsfläche in der Seitenleiste. Eine weitere Kopie der gemeinsamen Ebene erscheint in dieser Arbeitsfläche. Die Anordnung der gemeinsamen Ebenen kann nach Belieben geändert werden.
Alle weiteren Änderungen, die Sie an der Kopie der Ebene vornehmen, wird auch auf allen anderen Kopien übernommen.
Objekte auf gemeinsamen Ebenen, die entsprechenden Elemente in der Listen-Seitenleiste und die Vorschauen gemeinsamer Ebenen in der Arbeitsflächen-Seitenleiste werden alle in der besonderen Farbe von gemeinsamen Ebenen angezeigt, um Sie daran zu erinnern, dass sich alle Ihre Änderungen auf zahlreiche Arbeitsflächen auswirken können.
Sie können eine Kopie einer gemeinsamen Ebene von einer Arbeitsfläche entfernen, indem Sie sie normal löschen. Wenn Sie die letzte Kope einer gemeinsamen Ebene löschen, wird diese vollständig gelöscht. Es steht auch ein Befel zum Löschen aller Kopien zur Verfügung.
