Einstellungen zur Softwareaktualisierung

OmniGraffle kann über Ihre Internetverbindung automatisch nach neuen und aktualisierten Versionen des Programms suchen; Sie können auf diese Einstellungen über die Option "Programmeinstellungen" im OmniGraffle-Menü und anschließendes Klicken auf die Taste "Aktualisieren" zugreifen.
Wenn das Feld "Nach Aktualisierungen suchen" markiert ist, werden Sie informiert, wenn eine neue Version von OmniGraffle verfügbar ist. Klicken Sie auf das Feld "Jetzt nach Aktualisierungen suchen", um sofort nachzusehen, ob eine neue Version der Software verfügbar ist.
Sie können das Feld Systeminformationen einschließen aktivieren, um einige Informationen zu Ihrer Computerkonfiguration freizugeben, wenn Sie nach Updates suchen. Diese Informationen unterstützen Omni dabei, Entscheidungen dahingehend zu treffen, welche Hardware von unserer Software unterstützt werden soll. Einzelheiten dazu, welche Informationen ermittelt und wie diese verwendet werden, erfahren Sie, wenn Sie auf die Taste "Weitere Infos" in den Programmeinstellungen klicken.
