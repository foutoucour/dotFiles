Tutorial 8: Ebenen

Ebenen enthalten verschiedene Gruppen von Objekten auf einer Arbeitsfläche. Wir werden für das Hintergrundbild eine separate Ebene erzeugen, um es vom eigentlichen Diagramm zu trennen.
Klicken Sie auf die Taste "Arbeitsflächen" in der Symbolleiste, um die Arbeitsflächen-Seitenleiste anzuzeigen. Diese enthält die einzige Arbeitsfläche, die Sie bislang besitzen. Klicken Sie auf das Aufklappdreieck, um ihre Ebenen anzuzeigen.
Zur Zeit gibt es genau eine Ebene. Wir müssen also zunächst eine weitere hinzufügen. Klicken Sie auf die Taste "Ebene hinzufügen" unter der Seitenleiste. Eine neue Ebene wird erstellt. Nennen Sie die neue Ebene "Hintergrund".
Die Reihenfolge der Ebenen ist entscheidend; die Objekte auf den höheren Ebenen erscheinen vor den Objekten auf niedrigeren Ebenen. Da das Hintergrundbild hinter unserem Beziehungsdiagramm liegen soll, müssen Sie die Hintergrundebene ans Ende der Liste ziehen.
Konzentrieren wir uns auf die Hintergrundebene. Die aktive Ebene erkennen Sie im Zusatzfach an einem Bleistiftsymbol links vom Vorschau-Symbol. Ist die Hintergrundebene nicht aktiv, klicken Sie links neben ihr Vorschau-Symbol. Dann erscheint der Bleistift dort.
Wählen Sie ein hübsches Hintergrundbild aus Ihrem Bestand (oder laden Sie eines aus dem Internet herunter). Ziehen Sie die Bilddatei aus einem Finder-Fenster einfach auf einen leeren Bereich der Arbeitsfläche. Dadurch entsteht auf der Hintergrundebene eine neue Form, die Ihre Grafik beinhaltet.
Ziehen Sie diese Form mithilfe der Aktivpunkte in die gewünschte Größe. Wenn Sie dabei die Umschalttaste gedrückt halten, geschieht die Größenänderung proportional. Verwenden Sie das Informationsfenster "Bild", um die Deckkraft der Grafik auf circa 15 % einzustellen.
Sie haben ein attraktives und informatives Diagramm erschaffen. Dabei haben Sie gelernt, wie man mithilfe der Gliederungsansicht Objekte erzeugt, benennt, verknüpft und ihnen Stilattribute zuweist. Sie wissen, wie man Stile auf andere Objekte überträgt und mit mehreren Ebenen arbeitet. Jetzt können Sie Ihrer Kreativität freien Lauf lassen!
