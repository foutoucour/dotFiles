Objekte automatisch mit dem Informationsfenster "Diagrammlayout" anordnen

Mit diesem Informationsfenster können Sie Formen automatisch aufgrund der logischen, durch die Verknüpfungslinien hergestellten, Beziehungen anordnen.
Markieren Sie das Feld Automatisches Layout, wenn OmniGraffle die Objekte auf der Arbeitsfläche anordnen soll, wenn sich die Verknüpfungen zwischen diesen ändern.
Vier Layouttyp-Tasten geben die vier Methoden an, OmniGraffle für Diagrammlayouts verwendet. Dashierarchische Layout erstellt Ebenen gleichwertiger Objekte, die in eine Richtung verlaufen. Kräftebasierte Layouts werden nicht in einer festgelegten Richtung von einem Rand ausgehend aufgebaut, sondern verzweigen halbzufällig ausgehend von einem Mittelpunkt. Beim kreisförmigen Layout wird versucht, Geschwisterformen in einem Kreis um die Elternform anzuordnen. Beim bogenförmigen Layout werden Geschwisterformen in einem Bogen um die Elternform angeordnet.
Bei Verwendung des hierarchischen Layouts:
Die Richtungstasten bestimmen, wo die Objekte der obersten Ebene liegen und wie die darunterliegenden Objekte von diesem Punkt aus angeordnet werden.
Das Feld Rangtrennung steuert, wie weit jede Objektebene von der nächsten entfernt sein sollte.
Das Feld Objekttrennung steuert, wie weit jedes Objekt von anderen Objekten auf der gleichen Ebene entfernt sein sollte.
Mithilfe der Tasten unter Ausgewählter Objektrang können Sie den ausgewählten Objekten einen Rang in der Hierarchie zuweisen. Bei aktivierter Option "Standard" bestimmt OmniGraffle den Rang anhand der vorhandenen Verknüpfungen. Mit der Option "Spitze" werden die Objekte an den Anfang und mit der Option "Ende" an das Ende der Hierarchie gesetzt. Mit der Option "Gleichwertig" wird Objekten der gleiche Rang zugewiesen. Keine dieser Einstellungen ändert die Richtung von Verknüpfungslinien, sodass Sie jederzeit alle Objekte auswählen und mit der Option "Standard" in den normalen Status zurücksetzen können.
Bei Verwendung der anderen Layoutmethoden:
Verknüpfungslinien können gedehnt und gestaucht werden. Ihre durchschnittliche Länge lässt sich durch Ziehen des Schiebereglers Linienlänge einstellen.
In einem kräftebasierten Layout können Sie einzelne Linien auswählen und das Feld "Angepasste Länge" aktivieren, um sie unabhängig von den übrigen Linien anzupassen.
Der Schieberegler Abstoßung bestimmt, wie stark sich die Formen gegenseitig abstoßen. Falls die Linienlänge und die Abstoßung gering genug sind, können sich die Formen möglicherweise auch überlappen.
