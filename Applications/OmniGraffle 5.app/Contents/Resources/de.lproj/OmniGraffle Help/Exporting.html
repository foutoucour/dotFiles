Exportieren in verschiedene Formate

Sie können aus Ihren OmniGraffle-Diagrammen Dateien unterschiedlichen Typs erstellen. Wählen Sie "Exportieren" aus dem Menü "Ablage". Beim Exportieren wird eine neue Datei auf den Datenträger geschrieben. Das angezeigte Originaldokument bleibt dabei unverändert. Es können allerdings nicht alle Aspekte eines Diagramms in allen Dateiformaten wiedergegeben werden.
Die verfügbaren Dateiformate sind:
OmniGraffle-Dokument: Ein Dokument im OmniGraffle-Format. Sie können für die Datei den Schreibschutz (nicht editierbar) aktivieren und verknüpfte Bilder in die Datei aufnehmen, sodass diese nach der Übertragung auf einen anderen Computer korrekt und vollständig angezeigt werden können.
PDF: Abkürzung für "Portable Document Format". Dies ist ein praktisches, plattformübergreifendes Format, das eine gute Unterstützung sowohl von Vektorkomponenten als auch von Bitmap-Bildern bietet.
TIFF: Abkürzung für "Tagged Image File Format". Dieses Format unterstützt transparente Bildbereiche.
PNG: Abkürzung für "Portable Network Graphics", ein für das Internet häufig verwendetes Format. Dieses Format verwendet eine verlustfreie Komprimierung, um die Details eines Bilds beim Verkleinern der Datei zu erhalten. Transparente Bildbereiche werden unterstützt.
JPEG: Ein häufig verwendetes Format für komprimierte Bilder. Sie können die Bildqualität verringern, um kleinere Dateien zu erzielen.
EPS: Abkürzung für "Encapsulated PostScript". Dies ist ein gängiges Format beim professionellen Drucken.
HTML: Eine Hypertextdatei und ein Bild im JPEG-, PNG- oder GIF-Format. URL-Aktionen im OmniGraffle-Originaldokument werden in der HTML-Datei als Verknüpfungen codiert, sodass Sie in das Bild klicken können, um den Aktionen zu folgen. (Hinweis: Aktionen, die auf Linien angewendete werden, sind hier nicht eingeschlossen. Aktionen, die auf Linienetiketten angewendet werden, sind jedoch eingeschlossen.)
OmniOutliner 3: Das Diagramm wird als Gliederung in Textform wiedergegeben, wobei die Hierarchie anhand der Verknüpfungslinien zwischen Formen erkannt wird. Dies führt nur bei Diagrammen mit einer starren Baumstruktur zu zufriedenstellenden Ergebnissen.
SVG-Vektorzeichnung: Ein offener Internet-Standard, der XML verwendet.
PICT Vektorbild: Ein älteres Macintosh-Grafikformat.
BMP Bitmap-Bild: Ein älteres Grafikformat.
OmniGraffle-Diagrammstil, OmniGraffle-Vorlage, OmniGraffle-Schablone: Ressourcen für OmniGraffle.
Visio® XML-Dokument: Das XML-basierte Dateiformat für das Microsoft®-Programm zur Diagrammerstellung.
Abhängig vom ausgewählten Dateiformat sind verschiedene Optionen verfügbar. Die meisten Formate bieten die Option "Exportbereich", mit der Sie auswählen können, wie groß der zu exportierende Bereich des Dokuments sein soll.
Auswahl: Exportieren Sie nur die aktuell ausgewählten Objekte.
Alle Objekte: Exportieren Sie alle Objekte auf der Arbeitsfläche. Die leeren Bereiche werden dabei ignoriert.
Bereich: Legen Sie einen genauen Bereich der Arbeitsfläche fest, der exportiert werden soll.
Arbeitsfläche: Exportieren Sie die gesamte Arbeitsfläche.
Gesamtes Dokument: Exportieren Sie alles und erstellen Sie dabei für jede Arbeitsfläche eine Bilddatei.
Abhängig vom ausgewählten Format können Sie für den Export außerdem eine Skalierung (kleiner oder größer als die aktuelle Größe) auswählen, die Auflösung und die Komprimierungsstärke festlegen sowie angeben, ob der Hintergrund transparent sein soll.
