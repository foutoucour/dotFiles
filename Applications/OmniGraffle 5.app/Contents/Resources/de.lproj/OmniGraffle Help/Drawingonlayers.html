Auf Ebenen zeichnen

Die Ebenen einer Arbeitsfläche werden in der Seitenleiste "Arbeitsflächen" aufgelistet. Um diese zu öffnen, wählen Sie "Arbeitsflächen" aus dem Menü "Darstellung" oder klicken Sie auf die Schaltfläche "Arbeitsflächen" in der Symbolleiste.
Klicken Sie in das Dreiecksymbol neben einer Arbeitsfläche, um deren Ebenen anzuzeigen. Ebenen sind transparent, sodass Sie hinter dem Inhalt der obersten Ebene den Inhalt der zweiten Ebene sehen usw., bis hin zur untersten Ebene.
Jede Ebene in der Liste verfügt über eine kleine Vorschau ihres Inhalts und einen Titel. Wenn Sie die Maus darüber bewegen, sehen Sie außerdem drei Symbole. Die derzeit bearbeitete Ebene ist durch ein Stiftsymbol auf der linken Seite gekennzeichnet. Wenn Sie eine andere Ebene bearbeiten möchten, klicken Sie einfach links neben deren Vorschau. Das Stiftsymbol wird nun an dieser Stelle angezeigt.
Klicken Sie in das Augensymbol , um eine Ebene ein- oder auszublenden. Klicken Sie in das Druckersymbol , um festzulegen, ob eine Ebene beim Drucken der Arbeitsfläche gedruckt wird. Klicken Sie in das Schlosssymbol , um eine Ebenen zu schützen oder ihren Schutz aufzuheben. Durch das Schützen einer Ebene wird verhindert, dass der Inhalt der Ebene geändert wird.
Sie können Ebenen neu anordnen, indem Sie sie in der Liste bewegen. Sie können eine Ebene auch direkt auf eine andere Arbeitsfläche verschieben. Bewegen Sie sie bei gedrückter Wahltaste, um sie stattdessen zu kopieren. Wenn Sie eine Ebene in ein anderes Dokument bewegen, wird sie immer dorthin kopiert.
Zum Hinzufügen einer neuen Ebene klicken Sie auf die Schaltfläche "Neue Ebene" am unteren Rand des Zusatzfachs für Arbeitsflächen.
Zum Löschen einer Ebene wählen Sie diese aus und drücken die Entf-Taste, oder wählen Sie "Ebene löschen" aus dem Einblendmenü "Aktionen" am unteren Rand der Seitenleiste. Jede Arbeitsfläche muss mindestens eine Ebene enthalten. Aus diesem Grund kann die letzte Ebene nicht gelöscht werden.
