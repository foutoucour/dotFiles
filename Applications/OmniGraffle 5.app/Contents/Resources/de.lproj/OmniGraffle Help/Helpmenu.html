Menü "Hilfe"

OmniGraffle-Hilfe: Hiermit öffnen Sie die Online-Hilfe, die Sie gerade lesen. Willkommen! Wir danken Ihnen, dass Sie diese Dokumentation lesen.
Tastaturabkürzungen: Zeigen Sie ein Dokument an, das alle in OmniGraffle verfügbaren Dokumente auflistet, damit Sie weniger auf die Maus angewiesen sind.
Neue Funktionen in 5 / Zusätzliche Funktionen in Pro: Hiermit gelangen Sie zur OmniGraffle-Website, auf der Sie weiterführende Informationen zu den Vorteilen von OmniGraffle Professional 5 erhalten.
Graffletopia: Besuchen Sie die unabhängige Website Graffletopia, auf der OmniGraffle-Schablonen jeglicher Art ausgetauscht werden.
Feedback senden: Wählen Sie diese Option, um eine E-Mail an die Support-Mitarbeiter von OmniGraffle zu senden. Auf diese Weise können Sie am besten mit uns Kontakt aufnehmen. Ihre Versionsnummer und Lizenzinformationen werden automatisch in das Feld "Betreff" eingetragen, wodurch es für uns einfacher wird, Sie schnell und gezielt zu unterstützen.
