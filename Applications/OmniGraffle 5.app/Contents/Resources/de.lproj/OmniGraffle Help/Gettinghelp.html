E-mail-Support
Falls Sie mal nicht mehr weiter wissen, eine gute Idee für die nächste Version von OmniGraffle haben oder uns einfach mitteilen möchten, wie Sie zurecht kommen, können Sie uns jederzeit eine E-Mail senden. Wählen Sie "Feedback senden" aus dem Menü "Hilfe", um eine bereits an uns adressierte Mail aufzurufen, oder schreiben Sie einfach an omnigraffle@omnigroup.com. Wir bemühen uns um einen erstklassigen Support. Sie können sich also darauf verlassen, dass Sie eine Antwort von einem unserer Mitarbeiter erhalten.
