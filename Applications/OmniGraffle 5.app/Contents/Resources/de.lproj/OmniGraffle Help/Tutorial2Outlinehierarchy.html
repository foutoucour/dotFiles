Tutorial 2: Gliederungshierarchie

Das Erzeugen von Formen per Tastatur ist sehr praktisch. Spannend wird es, wenn eine Gliederungshierarchie erzeugt werden soll. Alle bisher erzeugten Figuren befinden sich auf einer Ebene. Im nächsten Schritt wollen wir weitere Figuren eine Ebene darunter erzeugen.
Die nächsten beiden Mitwirkenden sind Soldaten, die unter Don Pedros Kommando stehen, deshalb werden wir sie unter ihm anordnen. Klicken Sie links in der Gliederung auf "Don Pedro", um dieses Element auszuwählen. Drücken Sie dann die Eingabetaste, um ein neues Element unter ihm zu erzeugen. Benennen Sie es "Benedikt".
Drücken Sie die Tabulatortaste: In der Gliederung wird Benedikt eingerückt, und die dazugehörende Figur auf der Arbeitsfläche erhält eine Verknüpfungslinie von Don Pedro aus. Drücken Sie, während Benedikt weiterhin ausgewählt ist, die Eingabetaste, um ein weiteres Element auf derselben Ebene zu erzeugen, und nennen Sie es "Claudio". Jetzt gibt es zwei Mitwirkende, die zu Don Pedro gehören.
Verwenden Sie nun dieselbe Technik, um Beatrice und Hero unter Leonato anzuordnen, sie sind seine Nichte und seine Tochter. Im Diagramm erscheinen sie auf derselben Ebene wie Benedikt und Claudio, sind aber mit Leonato anstelle von Don Pedro verknüpft.
