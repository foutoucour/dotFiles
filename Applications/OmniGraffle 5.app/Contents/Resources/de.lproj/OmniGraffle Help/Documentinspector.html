Dokumentenweite Einstellungen mit dem Informationsfenster "Dokumente" steuern

Das Einblendmenü unter "Dateiformatoptionen" bestimmt, ob Ihr Dokument als flache Datei oder als einDateipaket gespeichert wird. Eine flache Datei ist eine einzelne Datei auf dem Datenträger, in die alle angehängten Bilder eingebettet sind. Ein Dateipaket ist im Grund ein Ordner, der den Anschein einer einzelnen Datei erweckt, in dem sich die angehängten Bilder befinden. Technisch versierte Anwender können in bestimmten Situationen eine Option oder die andere wählen, meist genügt es jedoch, sich auf die Einstellung Automatisch zu verlassen.
Normalerweise sind OmniGraffle-Dokumente Textdateien, die auf einer Eigenschaftenliste basieren. Falls die Option "Beim Sichern komprimieren" aktiviert ist, wird Ihre Datei statt dessen in einem Binärformat gespeichert, das weniger Speicherplatz auf dem Datenträger einnimmt, dessen Bestandteile jedoch nicht von Skripten oder Texteditoren gelesen werden können. Falls Sie nicht vorhaben, OmniGraffle-Dokumente mit einem Texteditor zu öffnen, können Sie die Einstellung zum Komprimieren der Dateien wählen.
Sie können Seitenränder festlegen oder einfach die Druckerränder verwenden, wie sie vom Druckertreiber definiert sind (oder in Ihren eigenen Einstellungen unter Seitenformat).
