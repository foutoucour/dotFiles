Menü "Ablage"

Neu: Falls Sie eine Standardvorlage haben, wird mit diesem Befehl ein neues Dokument basierend auf dieser Vorlage erstellt. Falls Sie keine Standardvorlage haben, wird die Vorlagenauswahl geöffnet.
Neue Ressource: Erstellen Sie eine neue Schablone, Vorlage oder einen Diagrammstil für OmniGraffle, ausgehend von der Vorlage Ihrer Wahl.
Vorlagenauswahl: Öffnen Sie die Vorlagenauswahl, in der Sie angeben können, welche Vorlage Sie für ein neues Dokument verwenden möchten.
Öffnen: Suchen Sie nach dem Dokument, das geöffnet werden soll. Wählen Sie ein beliebiges OmniGraffle-Dokument oder eine Datei in einem der Formate, die von OmniGraffle importiert werden können, aus.
Letzte Dokumente: Öffnen Sie ein Dokument, an dem Sie vor Kurzem gearbeitet haben. Wählen Sie "Liste löschen" , um die Liste der zuletzt geöffneten Dokumente zu löschen, damit z. B. niemand merkt, dass Sie die Datei "Überraschungsparty.graffle" bearbeitet haben.
Schließen: Schließen Sie das oberste geöffnete Fenster. Sind nicht gesicherte Änderungen vorhanden, werden Sie gefragt, ob Sie diese sichern oder verwerfen möchten.
Sichern: Sichern Sie Ihre Arbeit im obersten Dokument auf dem Datenträger. Wenn Sie das Dokument bereits einmal gesichert haben, wird die alte Version durch die neue Version ersetzt. Wenn Sie das Dokument zum ersten Mal sichern, werden Sie aufgefordert, den gewünschten Speicherort auf dem Datenträger auszuwählen und einen Dateinamen einzugeben.
Sichern unter: Sichern Sie das oberste Dokument unter einem neuen Namen auf dem Datenträger. Somit erhalten Sie eine neue, mit einem anderen Namen versehene Version der Datei, wobei die vorhandene, ältere Datei unverändert bleibt.
Alles sichern: Schreiben Sie alle Ihre Arbeiten in allen geöffneten Dokumenten in einem Schritt auf den Datenträger.
Zuletzt gesichert: Wenn Ihre Änderungen nicht wunschgemäß ausgefallen sind, wählen Sie diese Option, um die letzte auf dem Datenträger gesicherte Version wiederherzustellen. Dies ist mit einer Reihe von Widerrufen-Vorgängen vergleichbar. Beim Öffnen einer Datei, die einige Änderungen in einer automatischen Sicherungskopie enthält, können Sie diesen Befehl verwenden, um zur letzten manuell gesicherten Version zurückzukehren.
Bild platzieren: Suchen Sie nach einer Bilddatei. Auf der Arbeitsfläche wird ein Formobjekt mit dem Bild an der Stelle erstellt, in die Sie zuletzt mit dem Auswahlwerkzeug geklickt haben.
Exportieren: Hiermit wird aus einem bestimmten Teil der Daten in Ihrem OmniGraffle-Dokument eine neue Datei erstellt, und zwar mit dem von Ihnen ausgewählten Dateityp. Exportierte Dateien geben den Inhalt des OmniGraffle-Dokuments möglichst genau in einem anderen Format wieder, sie sind jedoch keine exakten Kopien. Im Abschnitt Exportieren erhalten Sie weitere Informationen.
Seitenformat: Das Mac OS X-Standardfenster "Papierformat" wird angezeigt. Dieses Fenster enthält mehrere Unterfenster mit Einstellungen für die Darstellung Ihres Dokuments auf Druckseiten.
Drucken: Das Mac OS X-Standardfenster "Drucken" wird angezeigt. Zusätzlich zu den Einstellungen im Fenster für das Papierformat gibt es eine Vielzahl von Optionen speziell für Druckvorgänge, die Sie hier festlegen können.
Arbeitsfläche(n) drucken: Das Fenster "Drucken" wird angezeigt, es bezieht sich jedoch nur auf die ausgewählten Arbeitsflächen.
