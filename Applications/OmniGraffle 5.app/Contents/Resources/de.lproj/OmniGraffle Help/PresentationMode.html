Präsentationsmodus

Sie können den Präsentationsmodus im Menü "Darstellung" auswählen, um Ihr Dokument bildschirmfüllend zu präsentieren.
Jede Arbeitsfläche verhält sich dabei wie die Folie einer Präsentation. Mit den Pfeiltasten können Sie zwischen den einzelnen Arbeitsflächen wechseln. Sie können auch durch Drücken der Eingabetaste oder der Nummernblock-Eingabetaste bzw. durch Klicken zur nächsten Arbeitsfläche gelangen. Wenn Sie in den Programmeinstellungen "Präsentation" die Option "Objekte durch Klicken hervorheben" aktiviert haben, wird beim Klicken in ein Objekt nicht zur nächsten Arbeitsfläche gewechselt. In diesem Fall können Sie in einen leeren Bereich der Arbeitsfläche klicken, um die nächste Arbeitsfläche anzuzeigen. Wenn Sie die Maus an den unteren Rand des Bildschirms bewegen, erscheint eine Navigationseiblendung mit Tasten zum Vorwärts- oder Rückwärtsgehen, zum Beenden und zum direkten Wechsel zu einer bestimmten Arbeitsfläche.
Sie können Objekte hervorheben, um während der Präsentation besondere Aufmerksamkeit darauf zu lenken. Verwenden Sie die Programmeinstellung "Präsentation", um festzulegen, wie die Hervorhebung erfolgt.
Zum Verlassen des Präsentationsmodus drücken Sie die Esc-Taste oder klicken auf das "X", das angezeigt wird, wenn Sie die Maus zum unteren Bildschirmrand bewegen.
Im Informationsfenster "Aktion" eingerichtete Aktionen können im Präsentationsmodus verwendet werden. Wenn Sie in ein Objekt klicken, dem eine Aktion zugewiesen wurde, wird diese Aktion in gleicher Weise ausgeführt wie beim Klicken mit dem Aktionenwerkzeug.
Programmeinstellungen "Präsentation"
Im Präsentationsmodus können Sie Objekte auf der Arbeitsfläche hervorheben.
Wählen Sie aus, ob Sie Objekte durch Zeigen oder durch Klicken bzw. nie hervorheben möchten. Sie können auch ein Symbol bei Objekten anzeigen, für die im Informationsfenster "Aktion" Aktionen eingerichtet wurden.
Klicken Sie in das Farbfeld, um eine Farbe für die Hervorhebung auszuwählen. Wenn Sie "selectedControlColor" aus der Palette "Entwickler" verwenden, können Sie sicherstellen, dass die Hervorhebungsfarbe immer der Auswahlfarbe entspricht, die in der Mac OS X-Systemeinstellung "Erscheinungsbild" ausgewählt wurde. Verwenden Sie den Schieberegler, um die Breite der Hervorhebung festzulegen.
Sind mehrere Fenster geöffnet, wird das vorderste Fenster für die Präsentation verwendet. Arbeiten Sie mit zwei Bildschirmen, so können Sie auf einem Bildschirm die Präsentation anzeigen und auf dem anderen das Dokument bearbeiten.
Wählen Sie "Neues Fenster für "Dokument"" (wobei "Dokument" für Ihren Dokumentnamen steht) aus dem Menü "Fenster".
Bewegen Sie ein Fenster zum zweiten Bildschirm.
Klicken Sie in das Fenster auf dem zweiten Bildschirm, sodass es ganz vorne angezeigt wird.
Öffnen Sie den Präsentationsmodus. Auf dem zweiten Bildschirm wird die Präsentation angezeigt, während auf Ihrem ersten Bildschirm weiterhin das standardmäßige Dokumentfenster zu sehen ist.
Sie können nun das Dokument auf dem ersten Bildschirm bearbeiten. Die Präsentation auf dem zweiten Bildschirm aktualisiert sich daraufhin selbst.
