Grundlagen: Objekte auswählen, ihre Größe verändern und Objekte bewegen

Wenn Sie ein Objekt auswählen möchten, achten Sie darauf, dass das Auswahlwerkzeug () aktiviert ist. Klicken Sie dann in das Objekt. Das Objekt wird hervorgehoben. Wenn seine Größe verändert werden kann, erscheinen acht Aktivpunkte an den Kanten und Ecken. Sobald Sie ein Objekt ausgewählt haben, können Sie es mithilfe der Informationsfenster oder mit Menübefehlen bearbeiten.
Wenn Sie weitere Objekte auswählen möchten, ohne dabei bereits ausgewählte Objekte zu deaktivieren, drücken Sie beim Auswählen die Befehlstaste.
Wenn Sie mehrere nebeneinander liegende Objekte gleichzeitig auswählen möchten, klicken Sie in einen freien Bereich auf der Arbeitsfläche und zeichnen Sie einen Auswahlrahmen um die Objekte, die Sie auswählen möchten. Wenn Sie beim Ziehen die Optionstaste gedrückt halten, erhalten Sie nur die Objekte, die vollständig in das Rechteck passen.
Um die Größe eines Objekts zu verändern, müssen Sie dieses zunächst auswählen. Bewegen Sie dann die Aktivpunkte für die Größenanpassung (die kleinen Vierecke, die angezeigt werden, wenn ein Objekt ausgewählt wird), bis das Objekt die gewünschte Größe hat. Automatische Hilfslinien werden angezeigt und helfen Ihnen dabei, Objekte gleichmäßig zu vergrößern bzw. zu verkleinern.
Wenn Sie ein Objekt bewegen möchten, verwenden Sie das Auswahlwerkzeug. Klicken Sie damit in das Objekt, halten Sie die Maustaste gedrückt und bewegen Sie es an die gewünschte Stelle. Automatische Hilfslinien werden angezeigt und helfen Ihnen dabei, Objekte gleichmäßig auszurichten.
