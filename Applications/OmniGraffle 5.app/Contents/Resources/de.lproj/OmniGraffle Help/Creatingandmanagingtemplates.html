Vorlagen erstellen und verwalten

Eine Vorlage ist ein Dokument, das als Grundlage für neue Dokumente verwendet wird. Auf der in OmniGraffle enthaltenen Vorlage "1cm = 1m" sind beispielsweise die Einheiten für Zeichnen in einem großen Maßstab eingestellt. Andere Vorlagen sind für Diagrammstile eingerichtet, die für bestimmte Arten von Zeichnungen geeignet sind. OmniGraffle umfasst mehrere Vorlagen. Sie können auch eigene Vorlagen erstellen, wenn Sie bestimmte Anordnungen der Arbeitsfläche häufiger verwenden.
In der Vorlagenauswahl können Sie eine Vorlage für Ihr neues Dokument auswählen. Sie können auch eine neue Vorlage erstellen, eine Ihrer eigenen Vorlagen bearbeiten oder eine Vorlage als Standard festlegen. Sobald Sie eine Standardvorlage festgelegt haben, umgeht OmniGraffle die Vorlagenauswahl und verwendet diese Vorlage für alle neu erstellten Dokumente. Falls Sie im Einzelfall eine andere Vorlage verwenden möchten, können Sie weiterhin auf die Vorlagenauswahl über das Menü "Ablage" zugreifen.
Beim Erstellen einer OmniGraffle-Vorlage werden alle dazugehörigen Einstellungen bewahrt, von der Arbeitsflächengröße und dem Rasterlayout, über die Ebenen und die Sichtbarkeit der Extras, bis hin zu den automatischen Layouteinstellungen. Zum Erstellen einer neuen Vorlage wählen Sie "Neue Ressource ▸ Neue Vorlage" im Menü Ablage und wählen dann eine bestehende Vorlage als Ausgangsdatei. Richten Sie das Dokument genau Ihren Wünschen entsprechend ein, und speichern Sie es. Der Standardspeicherort ist Ihr Ordner Application Support, in dem OmniGraffle nach Ihren Vorlagen sucht. Sobald eine Vorlage in diesem Ordner gespeichert wurde, ist sie in der Vorlagenauswahl verfügar.
