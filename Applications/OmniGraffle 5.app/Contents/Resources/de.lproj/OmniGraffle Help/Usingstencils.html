Mithilfe von Schablonen häufig verwendete Objekte speichern und gemeinsam verwenden

Eine Schablone ist ein Satz nützlicher Objekte, die Sie in Ihre Diagramme ziehen können. Zur Anzeige von Schablonen wählen Sie "Schablonen" aus dem Menü "Fenster".
Klicken Sie auf diese Schaltflächen, um die Objekte so zu sehen, wie sie auf der Schablone erscheinen bzw. jedes Objekt in seinem eigenen Feld.
Verwenden Sie das Einblendmeü "Aktion", um Schablonen zu bearbeiten, die Schablonenliste anzupassen oder neue Schablonen zu erstellen. Im Folgenden erhalten Sie weitere Informationen zu diesen Befehlen.
Geben Sie einen beliebigen Text in das Suchfeld ein, um nach Schablonen zu suchen, deren Namen übereinstimmen oder die Objekte enthalten, deren Text oder Notizen übereinstimmen. Sie können sogar online nach Schablonen anderer Anwender suchen. Weitere Informationen zur Suche nach Schablonen erhalten Sie weiter unten.
Wählen Sie eine Schablone aus der Liste aus, um ihren Inhalt anzuzeigen. wählen Sie einen gesamten Ordner aus, um den Inhalt aller darin enthaltenen Schablonen anzuzeigen.
Die ausgewählten Schablonen erscheinen hier. Ziehen Sie Objekte von hier in Ihr Dokument. Falls Sie eine Schablone geöffnet haben, die noch nicht installiert ist, klicken Sie auf die Taste "Installieren" in der Titelleiste, um diese in Ihrem Ordner zu speichern.
Bei der Auswahl eines Objekts werden einige Informationen zu diesem, einschließlich Notizen und angepassten Daten, hier auf der Seite mit Metadaten angezeigt.
Falls Sie jedoch kein ganzes Objekt auswählen möchten, sondern lediglich einen oder einige seiner Stile, ziehen Sie einen Stil aus der Stilleiste auf ein Objekt in Ihrem Dokument.
Klicken Sie auf diese Taste, um das Fenster mit Metadaten anzuzeigen oder auszublenden.
Wenn Sie ein Objekt auf der Arbeitsfläche platzieren möchten, bewegen Sie es einfach von der Schablone an eine beliebige Position. Es wird eine Kopie des Objekts erstellt. Das Original verbleibt auf der Schablone. Sie können also beliebig viele Kopien erstellen. Für die Auswahl stehen dieselben Funktionen wie auf der Arbeitsfläche zur Verfügung. Sie können also z.B. bei gedrückter Befehlstaste klicken oder einen Auswahlrahmen zeichnen, um mehrere Objekte auszuwählen.
Die Schablonenliste ist in verschiedene Bereiche aufgeteilt.
Ihre Schablonenbibliothek enthält alle Schablonen, die Sie installiert haben, egal ob diese in OmniGraffle integriert sind oder im Ordner Application Support vorliegen.
Unter Zuletzt verwendete Schablonen sind die letzten fünf Schablonen aufgelistet, die Sie sich angeschaut haben. Wenn Sie eine noch nicht installierte Schablone öffnen, erscheint diese in der Liste der zuletzt verwendeten Schablonen. Sie können auf die Taste Installieren neben dem Titel klicken, um sie am passenden Speicherort in Ihrem persönlichen Ordner zu installieren.
Der Bereich Schablonen-Favoriten ist vollständig anpassbar. Sie können beliebige installierte Schablonen in diesen Bereich ziehen. Wählen Sie Neuer Schablonen-Favoriten-Ordner aus dem Einblendmenü "Aktion", um einen Ordner zur Gruppierung Ihrer bevorzugten Schablonen zu erstellen.
Zum Suchen nach Schablonen nehmen Sie eine Eingabe im Suchfeld über der Schablonenliste vor. OmniGraffle durchsucht die Titel Ihrer Schablonen, den Text der Objekte auf jeder Schablone sowie die Notizen und angepassten Daten. Übereinstimmende Schablonen werden im Bereich "Suchergebnisse" der Schablonenliste aufgeführt. Wenn ein Objekt auf einer Schablone übereinstimmt, erscheinen normalerweise alle Objekte auf dieser Schablone in den Suchergebnissen. Sie können jedoch die Filter-Einstellung im Einblendmenü des Suchfelds aktivieren. In diesem Fall werden die Objekte, die nicht mit Ihrer Eingabe übereinstimmen, ausgeblendet.
Um die Online-Bibliothek von OmniGraffle-Schablonen zu durchsuchen, klicken Sie auf das Einblendmenü des Suchfelds und wählen Graffletopia. Das ist der kostenlose Service von Patrick Crowley, bei dem jeder Benutzer nützliche Schablonen bereitstellen und verwenden kann. Beim Durchsuchen von Graffletopia erscheinen Schablonen, deren Namen oder Beschreibungen übereinstimmen, in der Liste der Suchergebnisse. Klicken Sie auf eine Schablone, um eine Vorschau und eine Beschreibung anzuzeigen, und dann auf die Taste "Download", um eine Kopie dieser Schablone herunterzuladen. Öffnen Sie die Datei im Finder. Anschließend können Sie die Schablone normal verwenden oder installieren. Sie können auch auf Graffletopia zugreifen, indem Sie in OmniGraffle in das Menü "Hilfe" gehen oder mit Ihrem Browser http://graffletopia.com öffnen.
Um eine neue Schablone zu erstellen, wählen Sie "Neue Ressource ▸ Schablone" aus dem Menü "Ablage". Sie können das daraufhin angezeigte Dokument auf die gleiche Weise wie jedes gewöhnliche OmniGraffle-Diagramm bearbeiten. Während Sie arbeiten, wird eine Vorschau der Schablone im Fenster "Schablone" angezeigt. Sobald die Schablone Ihren Vorstellungen entspricht, wählen Sie "Sichern" aus dem Menü "Ablage". Der standardmäßige Speicherort ist der Unterordner Application Support Ihres persönlichen Library-Ordners.
Zum Bearbeiten einer Schablone wählen Sie diese in der Liste aus und wählen dann "Schablone bearbeiten" (oder "Schablone kopieren und bearbeiten" für integrierte Schablonen) aus dem Einblendmenü "Aktion". Anschließend bearbeiten und sichern Sie die Schablone wie ein gewöhnliches OmniGraffle-Dokument.
