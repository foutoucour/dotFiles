Menü "Darstellung"

Präsentation starten: Rufen Sie den Präsentationsmodus mit dem vordersten Dokument auf.
Zoomen
Vergrößern: Wählen Sie den nächsthöheren Vergrößerungsfaktor, damit die Arbeitsfläche größer aussieht.
Verkleinern: Wählen Sie den nächstniedrigen Vergrößerungsfaktor, damit die Arbeitsfläche kleiner aussieht.
Tatsächliche Größe: Wählen Sie den Vergrößerungsfaktor 100%, sodass die Arbeitsfläche den Arbeitsflächeneinheiten entspricht.
Auswahl zoomen: Zoomen Sie auf eine Ebene und Position, die die Ansicht mit den aktuell ausgewählten Objekten füllt.
Vorheriger Zoom: Kehren Sie zum letzten Zoomfaktor vor dem aktuellen zurück.
An Fenstergröße anpassen: Zoomen Sie auf eine Ebene und Position, bei der jedes Objekt auf der Arbeitsfläche während der Arbeit sichtbar bleibt.
Proof konfigurieren: Verwenden Sie dieses Untermenü, um einen Drucker oder ein anderes Gerät und das zugehörige Profil für die Proof-Ausgabe auszuwählen. Das Standardprofil zum Proofen ist das Standard-Proof-Profil von Mac OS X (üblicherweise das Profil "Allgemeines CMYK-Profil"). Sie können aber auch das Profil eines beliebigen Druck- oder Proof-Geräts wählen, das auf Ihrem System verfügbar ist.
Proof-Farben: Aktivieren Sie die Option "Soft-Proofing" mit dem Gerät, das Sie im Untermenü "Proof konfigurieren" ausgewählt haben. Hiermit wird ein zusätzlicher Schritt zur Farbverarbeitung hinzugefügt. Zuerst wird jede Farbe aus einem der drei Dokumentprofile in das Proof-Profil und anschließend aus dem Proof-Profil in das Anzeigeprofil konvertiert. Auf diese Weise können Sie theoretisch auf dem Bildschirm sehen, wie Ihre gedruckte Ausgabe aussehen wird, da der Druckpfad von Mac OS X nachvollzogen wird. Die Qualität des Ergebnisses hängt jedoch stark von der Qualität des Farbprofils Ihres Druckers ab. Bei vielen Tintenstrahldruckern aus dem Consumer-Bereich wird für das Druckerprofil ausgewiesen, dass hiermit jede beliebige Farbe reproduziert werden kann (als Umgehung von ColorSync zur Verwendung herstellereigener Farboptimierungen im Druckertreiber). Bei diesen Druckern sind möglicherweise kaum Unterschiede beim Soft-Proofing zu erkennen.
Extras: Auf der Arbeitsfläche stehen Ihnen verschiedene Elemente zur Verfügung, die sich ein- bzw. ausblenden lassen. Wählen Sie die gewünschte Option aus dieser Liste, um sie ein- oder auszublenden.
Aktionen: Jedes Objekt, dem im Informationsfenster "Aktion" eine Aktion zugewiesen ist, wird in der oberen rechten Ecke durch ein Aktionssymbol gekennzeichnet.
Raster: Dies ist das im Informationsfenster "Raster" definierte Raster. Sie können die Ausrichtung am Raster verwenden, egal ob dieses sichtbar ist oder nicht.
Hilfslinien: Dies sind die Hilfslinien, die Sie einrichten, um Objekte auf der Arbeitsfläche auszurichten.
Magnete: Wählen Sie diese Option, damit die Magnete aller Objekte permanent sichtbar sind.
Notizen: Jedes Objekt mit einer Notiz im Informationsfenster "Notiz" wird in der oberen rechten Ecke durch ein Notizsymbol gekennzeichnet.
Ursprung: Dies ist das Fadenkreuz, das auf der Arbeitsfläche den Ursprung kennzeichnet.
Seitenumbrüche: Dies sind die Linien, die die Grenzen zwischen einzelnen Seiten darstellen.
Lineale: Die Lineale helfen Ihnen, die Größe der Objekte auf der Arbeitsfläche festzulegen. Sie schließen eine Informationsleiste für häufig verwendete Bearbeitungsfunktionen ein.
Seitenleisten: Wählen Sie den Namen einer Seitenleiste aus, um diese anzuzeigen oder auszublenden. Die Arbeitsflächen haben ihre eigene Seitenleiste, die Gliederungsansicht und die Listenansicht teilen sich eine.
Arbeitsflächen: Eine Liste der Arbeitsflächen und Ebenen in dem Dokument.
Gliederung: die Gliederungs-Seitenleiste, die die Hierarchie der Objekte auf der Arbeitsfläche zeigt.
Liste: Die Listen-Seitenleiste, die die Anordnung der Objekte auf jeder Ebene zeigt.
Arbeitsfläche anzeigen
Nächste Arbeitsfläche bearbeiten: Wechseln Sie zu der Arbeitsfläche, die auf die aktuell bearbeitete folgt.
Vorherige Arbeitsfläche bearbeiten: Wechseln Sie zu der Arbeitsfläche, die der aktuell bearbeiteten vorangeht.
Symbolleiste anpassen: Wählen Sie diesen Befehl aus, damit verschiedene Steueroptionen angezeigt werden, die Sie in Ihre Symbolleiste bewegen können, wie bei anderen Mac OS X-Programmen auch.
Symbolleiste anzeigen/ausblenden: Legen Sie die Sichtbarkeit der Symbolleiste im Dokumentfenster fest.
