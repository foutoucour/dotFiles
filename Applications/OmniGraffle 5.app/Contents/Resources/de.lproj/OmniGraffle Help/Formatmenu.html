Menü "Format"

Schriften einblenden: Öffnet das Standard-Schriftartenfenster von Mac OS X.
Farben einblenden: Öffnet das Standard-Farbenfenster von Mac OS X.
Diagrammstil wählen: Öffnet die Diagrammstilauswahl, sodass Sie auf alle Objekte auf der Arbeitsfläche einen konsistenten Stil anwenden können.
Diagrammstil erneut anwenden: Wendet den aktuellen Diagrammstil erneut an, damit Objekte mit einem anderen Stil, die seit dem letzten Anwenden dieses Befehls hinzugekommen sind, angeglichen werden.
Stil auf ausgewählte Objekte anwenden: Wendet den aktuellen Diagrammstil auf die ausgewählten Objekte an.
Schriftart: Hierdurch wird das standardmäßige Mac OS X-Schriftenuntermenü eingeblendet. Es enthält Befehle zum Festlegen einer Schriftart und eines Stils für den ausgewählten Text.
Text: Dieses Untermenü enthält Befehle zum Ändern der Position und des Verhaltens des ausgewählten Texts. Die meisten dieser Funktionen stehen auch über das Informationsfenster "Text" zur Verfügung.
Stil kopieren: Kopiert die Stile des ausgewählten Objekts in die Zwischenablage. Von dort können sie in andere Objekte eingesetzt werden.
Stil einsetzen: Befinden sich Objektstile in der Zwischenablage, wenden Sie sie mit diesem Befehl auf die ausgewählten Objekte an.
Farbprofile: Wenn das Farbmanagement in der Programmeinstellung "ColorSync" aktiviert ist, können Sie die Farbprofile des Dokuments zuweisen oder umwandeln.
Profile zuweisen: Durch die Zuweisung eines Farbprofils werden die Farbdaten im Dokument nicht geändert. Es wird lediglich die Interpretation der Farbdaten geändert, wenn sie an einen Bildschirm oder einen Drucker gesendet werden. Das heißt, dass die Farben, die Sie auf dem Bildschirm oder der gedruckten Seite sehen, möglicherweise anders angezeigt bzw. gedruckt werden, wenn Sie ein anderes Profil zuweisen.
Durch In Profile konvertieren werden die Farbdaten im Dokument tatsächlich geändert, sodass das Dokument gleich aussieht, auch wenn ein anderes Profil verwendet wird. Dabei sollte kein Farbunterschied erkennbar sein. Dies dient dazu, Farben auch auf unterschiedlichen Geräten immer gleich darzustellen.
