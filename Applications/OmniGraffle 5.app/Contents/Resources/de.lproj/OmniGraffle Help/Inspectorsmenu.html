Menü "Informationen"

Informationsfenster anzeigen/ausblenden: Zeigt die aktuelle Anordnung von Informationsfenstern an oder blendet sie aus.
Informationsfenster auf ursprüngliche Position zurücksetzen: Setzen Sie alle Informationsfenster auf ihre Anfangsposition zurück, die beim allerersten Starten von OmniGraffle verwendet wurde.
Stil: Blenden Sie das gewünschte Informationsfenster "Stil" ein oder aus.
Eigenschaften: Blenden Sie das gewünschte Informationsfenster "Eigenschaften" ein oder aus.
Arbeitsfläche: Blenden Sie das gewünschte Informationsfenster "Arbeitsfläche" ein oder aus.
Dokument: Blenden Sie das gewünschte Informationsfenster "Dokument" ein oder aus.
Farben anzeigen/ausblenden: Öffnen oder schließen Sie das Mac OS X-Farbfenster. Diesen Schritt können Sie auch über ein Farbfeld in einem der Informationsfenster ausführen. Sie können Farbproben aus diesem Bereich auf Füllungen, Linientypen oder Text von Objekten bzw. auf Farbfelder in den Informationsfenstern bewegen.
Schriftarten anzeigen/ausblenden: Öffnen oder schließen Sie das Mac OS X-Schriftartenfenster. Diesen Schritt können Sie auch die Schriftart-Schaltfläche im Informationsfenster "Text" ausführen. Ist ein Objekt auf der Arbeitsfläche ausgewählt, wird die hier ausgewählte Schrift auf dieses Objekt angewendet.
