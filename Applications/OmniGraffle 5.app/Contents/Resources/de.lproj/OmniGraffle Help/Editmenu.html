Menü "Bearbeiten"

Widerrufen: OmniGraffle speichert eine Liste der Änderungen, die an einem Dokument vorgenommen werden. Wählen Sie diesen Befehl, um in der Liste einen Schritt zurück zu gehen, so als hätten Sie die letzte Aktion in der Liste nicht ausgeführt. Bei wiederholtem Auswählen dieses Befehls werden Ihre letzten Änderungen nacheinander widerrufen.
Wiederholen: Wenn Sie als letzte Aktion einen Schritt widerrufen haben, können Sie mit diesem Befehl wieder einen Schritt vorwärts gehen und die entsprechende Aktion erneut ausführen. Auf diese Weise wird der Befehl "Widerrufen" rückgängig gemacht.
Ausschneiden: Entfernen Sie das aktuell ausgewählte Objekt und legen Sie es in die Zwischenablage, um es an anderer Stelle wieder einzusetzen.
Kopieren: Kopieren Sie die aktuelle Auswahl in die Zwischenablage.
Kopieren als
PDF: Legen Sie eine PDF-Kopie der aktuellen Auswahl in die Zwischenablage, um sie in ein Programm einzusetzen, das das PDF-Format lesen kann. Falls die Anwendung, in die Sie die Auswahl einsetzen, ebenfalls LinkBack unterstützt (hierzu gehört auch OmniGraffle selbst), können Sie sogar von dort aus zurückgehen und den eingesetzten Inhalt mit OmniGraffle bearbeiten.
TIFF: Legen Sie eine TIFF-Kopie der aktuellen Auswahl in die Zwischenablage, um sie in ein Programm einzusetzen, das das TIFF-Format lesen kann.
Vektor PICT: Legen Sie eine Vektor PICT-Kopie der aktuellen Auswahl in die Zwischenablage, um sie in ein Programm einzusetzen, das das Vektor PICT-Format lesen kann (Dies ist besonders nützlich für das Einsetzen in Microsoft Word, da dieses Programm einen merkwürdigen Fehler aufweist, aufgrund dessen bei Verwendung des normalen Kopieren-Befehls für OmniGraffle-Inhalt nicht die korrekten Daten eingefügt werden).
AppleScript: Legen Sie eine AppleScript-Version der ausgewählten Objekte in die Zwischenablage. Sie können das Ergebnis in den Skripteditor oder in das Informationsfenster "Aktionen" eines anderen Objekts einsetzen, um ein Skript zu erstellen, mit dem sich das Objekt erstellen lässt. Anhand des Ergebnisses lässt sich außerdem leicht erkennen, wie die AppleScript-Unterstützung von OmniGraffle funktioniert.
Einsetzen: Setzen Sie den Inhalt der Zwischenablage an der letzten Stelle, an der Sie mit dem Auswahlwerkzeug geklickt haben, auf der Arbeitsfläche ein. (oder setzen Sie Text aus der Zwischenablage ein, wenn Sie gerade den Text eines Objekts bearbeiten). Wenn sich ein Bild oder Text in der Zwischenablage befindet, wird ein neues Formobjekt erstellt, das dieses Bild oder den Text enthält.
Einsetzen und Stil anpassen: Sie können beim Bearbeiten von Text mithilfe dieses Befehls Text einsetzen, ohne gleichzeitig den Stil zu übernehmen, den dieser Text in dem Programm, aus dem Sie ihn kopiert haben, hatte. Statt dessen wird der Stil des bereits vorhandenen Texts übernommen, als ob Sie den Text an dieser Stelle selbst eingetippt hätten.
Duplizieren: Erstellen Sie eine Kopie der aktuellen Auswahl und platzieren Sie diese auf der Arbeitsfläche. Die Zwischenablage wird hierbei nicht verwendet.
Dieser Befehl hat eine spezielle Funktion:
Treffen Sie eine bestimmte Auswahl auf der Arbeitsfläche und duplizieren Sie sie.
Bewegen Sie die Kopie mithilfe des Auswahlwerkzeugs. OmniGraffle merkt sich Abstand und Richtung der Kopie im Verhältnis zu Original.
Duplizieren Sie, so oft Sie wollen. Jede neue Kopie wird mit dem gleichen Abstand gegenüber der vorherigen Kopie und in gleicher Richtung platziert.
Löschen: Entfernen Sie die aktuelle Auswahl endgültig von der Arbeitsfläche (es sei denn, Sie wählen "Widerrufen").
Alles auswählen: Damit wählen Sie alle auswählbaren Objekte auf der Arbeitsfläche aus. Denken Sie daran, dass auf geschützten Ebenen keine Objekte ausgewählt werden können. Wenn Sie den Text eines Objekts bearbeiten, wird hierbei stattdessen der gesamte Text des Objekts ausgewählt.
Auswählen
Ähnliche Objekte: Wählen Sie alle Objekt auf der Arbeitsfläche aus, die die gleichen Stilattribute aufweisen wie die aktuell ausgewählten Objekte.
Verknüpfte Objekte: Wählen Sie alle Formen aus, die direkt oder indirekt durch Linien mit den aktuell ausgewählten Objekten verknüpft sind, sowie alle Verknüpfungslinien dazwischen.
Vorfahrenknoten: Wählen Sie alle Vorfahrenknoten der aktuell ausgewählten Objekte basierend auf der von den Verknüpfungslinien festgelegten Hierarchie aus.
Nachfahrenknoten: Wählen Sie alle Nachahrenknoten der aktuell ausgewählten Objekte basierend auf der von den Verknüpfungslinien festgelegten Hierarchie aus.
Gliederung: Dank der Befehle in diesem Untermenü (und insbesondere ihrer Kurzbefehle) wird das schnelle Arbeiten in der Gliederungsdarstellung noch einfacher. Die Befehle sind nur verfügbar, wenn die Gliederungsdarstellung geöffnet und darin eine aktive Auswahl vorhanden ist.
Ohne Maus arbeiten: Diese Befehle sind besonders nützlich, wenn Sie sie über ihre Kurzbefehle verwenden. Auf diese Weise können Sie eine Vielzahl von Schritten ausführen, ohne dafür die Maus zu benötigen.
Form auswählen: Mit diesen Befehlen wird eine Form auf Basis ihrer relativen Position zur aktuell ausgewählten Form ausgewählt.
Form erstellen: Diese Befehle erstellen eine neue Form relativ zur aktuell ausgewählten Form.
Arbeitsflächen
Neue Arbeitsfläche: Fügen Sie eine neue Arbeitsfläche zum Dokument hinzu.
Arbeitsfläche duplizieren: Fügen Sie eine neue Arbeitsfläche zum Dokument hinzu, die mit der aktuellen Arbeitsfläche identisch ist.
Arbeitsfläche löschen: Löschen Sie die aktuelle Arbeitsfläche vollständig.
Ebenen
Neue Ebene: Fügen Sie eine neue Ebene zur aktuellen Arbeitsfläche hinzu.
Neue gemeinsame Ebene: Fügen Sie eine neue gemeinsame Ebene zur aktuellen Arbeitsfläche hinzu.
Ebene duplizieren: Fügen Sie zu der Arbeitsfläche eine neue Ebene hinzu, die mit der aktuellen Ebene identisch ist.
Ebenen zusammenfügen: Verschieben Sie alles auf der ausgewählten Ebene auf die nächste Ebene, und löschen Sie dann die ausgewählte Ebene.
Ebene löschen: Löschen Sie die aktuelle Ebene vollständig. Die letzte Ebene auf der Arbeitsfläche kann nicht gelöscht werden.
Alle Kopien löschen: Dieser Befehl ist nur bei Auswahl einer gemeinsamen Ebene verfügbar. Damit werden alle Instanzen der gemeinsamen Ebene auf beliebigen Arbeitsflächen im gesamten Dokument gelöscht.
Ebeneneinstellungen: Diese Befehle haben die gleiche Funktion wie die klickbaren Symbole einer Ebene in der Arbeitsflächen-Seitenleiste. Sichtbar zeigt oder verbirgt die Objekte auf der Ebene, Druckbar steuert, ob die Objekte auf der Ebene beim Drucken eingeschlossen werden sollen oder nicht, und Nicht gesperrt steuert, ob die Objekte auf der Ebene ausgewählt bzw. bearbeitet werden können.
Alles in Ebene auswählen: Wählen Sie alle Objekte auf der aktuellen Arbeitsfläche aus.
Auswahl zu Ebene bewegen: Bewegen Sie alle ausgewählten Objekte auf die ausgewählte Ebene.
Tabellen
Zeile einfügen: Ist eine Zelle einer Tabelle ausgewählt, fügen Sie mit diesem Befehl eine neue Zeile mit Zellen zur Tabelle hinzu. Die neue Zeile wird direkt vor der Zeile eingefügt, in der sich die ausgewählte Zelle befindet.
Spalte einfügen: Ist eine Zelle einer Tabelle ausgewählt, fügen Sie mit diesem Befehl eine neue Spalte mit Zellen zur Tabelle hinzu. Die neue Zeile wird direkt vor der Zeile eingefügt, in der sich die ausgewählte Zelle befindet.
Zeile auswählen: Ist eine Zelle einer Tabelle ausgewählt, wählen Sie mit diesem Befehl die gesamte Zeile aus, in der sich die Zelle befindet.
Spalte auswählen: Ist eine Zelle einer Tabelle ausgewählt, wählen Sie mit diesem Befehl die gesamte Spalte aus, in der sich die Zelle befindet.
Magnete
Magnete kopieren: Legen Sie dieMagnet-Anordnung des ausgewählten Objekts in die Zwischenablage, damit Sie sie in einem anderen Objekt einsetzen können.
Magnete einsetzen: Befindet sich eine Magnetanordnung in der Zwischenablage, wenden Sie sie auf das ausgewählte Objekt an. Alle vorhandenen Magnete werden dabei durch die neuen Magnete ersetzt.
Magnete löschen: Entfernen Sie alle Magnete von den ausgewählten Objekten.
Formen
Formen ausstanzen: Verwenden Sie die oben liegenden Formen der ausgewählten Formen, um ein Loch in die unterste Form zu stanzen. Sie können die Reihenfolge der Formen mithilfe der Befehle "Nach vorne/Nach hinten" im Menü "Anordnen" ändern.
Formen verbinden: Erstellen Sie eine neue Form aus zwei oder mehr ausgewählten Formen, wobei die internen Linien entfernt werden, um eine einzige deckende Form zu erzielen.
Formen teilen: Erstellen Sie aus den überlappenden Bereichen der ausgewählten Formen eine neue Form.
Suchen
Suchen: Hiermit wird das Suchfenster von OmniGraffle angezeigt. Sie können nach einer bestimmten Zeichenfolge suchen oder einen regulären Ausdruck verwenden, um nach einer ganzen Klasse von Zeichenfolgen zu suchen. Bei der Suche nach einer bestimmten Zeichenfolge markieren Sie das Feld "Groß/klein", wenn die Groß- bzw. Kleinschreibung des Suchbegriffs nicht beachtet werden soll. Mit der Option "Ganzes Wort" können Sie vermeiden, dass Ergebnisse angezeigt werden, in denen der Suchbegriff nur als Teil eines Wortes vorkommt. Wenn Sie Gruppen in einem regulären Ausdruck verwenden, kann es vorkommen, dass Sie den ganzen Ausdruck oder nur eine der Gruppen auswählen (oder ersetzen) möchten. Treffen Sie die gewünschte Auswahl mithilfe des Einblendmenüs. Für die Optionen "Rückwärts" und "Vorwärts" definiert OmniGraffle die Reihenfolge von Objekten als "von links nach rechts" und "von oben nach unten".
Weitersuchen (vorwärts): Mit dieser Option können Sie nach dem nächsten Vorkommen des zuletzt in das Suchfenster eingegebenen Suchbegriffs suchen, so als hätten Sie im Suchfenster in die Taste "Weiter" geklickt.
Weitersuchen (rückwärts): Mit dieser Option können Sie nach dem vorherigen Vorkommen des zuletzt in das Suchfenster eingegebenen Suchbegriffs suchen, so als hätten Sie im Suchfenster in die Taste "Rückwärts" geklickt.
Auswahl eingeben: Wählen Sie Text in einem Objekt aus und kopieren Sie den ausgewählten Text dann durch Auswahl dieses Befehls in das Suchfenster. Zuvor eingegebene Suchbegriffe werden dadurch überschrieben. Sie können dann mithilfe der Befehle "Weitersuchen (vorwärts)" und "Weitersuchen (rückwärts)" weitere Vorkommen des Texts suchen.
Auswahl anzeigen: Bewegen Sie die Darstellung so, dass die ausgewählten Objekte zu sehen sind.
Rechtschreibung und Grammatik
Dieses Untermenü enthält standardmäßige Mac OS X-Rechtschreib- und Grammatikfunktionen. Für die Optionen "Rückwärts" und "Vorwärts" definiert OmniGraffle die Reihenfolge von Objekten als "von links nach rechts" und "von oben nach unten".
Variable einfügen
Wählen Sie bei der Bearbeitung des Texts eines Formobjekts einen dieser Befehle aus, um einen speziellen Platzhalter einzufügen. Nach Beendigung der Textbearbeitung ändert sich die Anzeige dieses Felds und enthält eine bestimmte Information zu dem zugehörigen Objekt, wie z.B. die Arbeitsfläche, die Seite oder das Dokument. Wenn Sie das Bezugsobjekt des Informationsfelds ändern, werden die Informationen im Feld automatisch aktualisiert. Manche Variablen, wie etwa die Seitennummer, sind nützlich zum Einfügen einer gemeinsamen Ebene und zum gemeinsamen Verwenden dieser Ebene mit allen Arbeitsflächen.
Wenn Sie Datumsvariablen verwenden, können Sie das Datumsformat mithilfe der Datenformatierungsausdrücke ändern. Um beispielsweise ein Datumsformat wie "2005 - 10 - 06 16:00" zu erhalten, müssten Sie folgendes eingeben:
<%date %Y - %m - %d %H:%M %>
Name der Arbeitsfläche: Der Name der Arbeitsfläche, in der sich das Objekt befindet.
Dokumentname: Der Name des Dokuments, in dem sich das Objekt befindet.
Aktuelles Datum: Das aktuelle Datum. Dies ist praktisch, um zu wissen, wann ein Dokument gedruckt wurde.
Erstellungsdatum: Das Datum, an dem das Dokument gemäß dem Mac OS X-Dateisystem ursprünglich erstellt wurde.
Änderungsdatum: Das Datum, an dem das Dokument gemäß dem Mac OS X-Dateisystem zuletzt gesichert wurde.
Erstellt von: Der Name des Benutzers, der die Datei gemäß dem Mac OS X-Dateisystem erstellt hat.
Geändert von: Der Name des Benutzers, der die Datei gemäß dem Mac OS X-Dateisystem zuletzt gesichert hat.
Seitenzahl: Die Nummer der Seite, auf der sich das Objekt befindet, basierend auf den Einstellungen im Informationsfenster "Arbeitsflächengröße" und der SeiteSeitenformat.
Seiten gesamt: Die Gesamtanzahl der Seiten in dem Dokument. Kombiniert mit "Seitenzahl" erhalten Sie Angaben wie z.B. "Seite 2 von 5".
X-Position: Der horizontale Abstand des Objekts vom Ursprung, ebenso wie im Informationsfenster "Größe und Position".
Y-Position: Der vertikale Abstand des Objekts vom Ursprung, ebenso wie im Informationsfenster "Größe und Position".
Breite: Die horizontale Größe des Objekts, ebenso wie im Informationsfenster "Größe und Position".
Höhe: Die vertikale Größe des Objekts, ebenso wie im Informationsfenster "Größe und Position".
Linienlänge: Dieses Informationsfeld hat eine besondere Eigenschaft. Falls die Form, die das Informationsfeld enthält, ein Linienetikett ist, wird die Länge der Linie angezeigt. Es ist von keinem großen Nutzen, wenn es nicht an eine Linie angehängt ist.
LinkBack-Element bearbeiten: Wenn Sie LinkBack-Inhalte aus einem anderen Programm eingesetzt haben, wählen Sie das Objekt aus und verwenden Sie diesen Befehl, um den Inhalt im Originalprogramm zu öffnen.
Sonderzeichen: Rufen Sie die Standard-Mac OS X-Zeichenpalette auf.
