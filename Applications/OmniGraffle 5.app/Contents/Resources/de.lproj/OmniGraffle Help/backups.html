Automatisch Sicherungskopien von Dokumenten sichern

Wenn Sie die Sicherungsfunktion unter Allgemeine Einstellungen aktivieren, erstellt OmniGraffle eine Sicherungskopie jedes Dokuments, während Sie daran arbeiten. Im Falle eines System- oder Stromausfalls bleibt die Sicherungskopie auf dem Datenträger erhalten. Wenn Sie das Dokument erneut öffnen, wird die zuletzt automatisch gesicherte Version als ein Dokument mit nicht gesicherten Änderungen angezeigt. Sie können weiterhin mit der automatisch gesicherten Version arbeiten oder die Option Zurück zur letzten Version aus dem Menü "Ablage" wählen, um zur manuell gesicherten Version zurückzukehren.
Wenn Sie das Dokument schließen, ohne es vorher zu sichern, wird die Sicherungskopie verworfen.
