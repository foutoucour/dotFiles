Objekte mit dem Informationsfenster "Aktion" interaktiv machen

In diesem Informationsfenster können Sie die Aktion festlegen, die ausgeführt werden soll, wenn mit dem Aktionenwerkzeug in das ausgewählte Objekt geklickt wird.
Die Standardeinstellung ist "Keine Aktion". In diesem Fall löst das Klicken in ein Objekt keine Aktion aus.
Wenn Sie "URL-Adresse öffnen" oder "Datei öffnen" auswählen, werden zusätzlich zwei Tasten und ein Textfeld angezeigt. Um auf Ihren Volumes nach einer Datei zu suchen, geben Sie eine URL oder einen Dateipfad in dieses Feld ein, bewegen eine Datei in diesen Bereich oder klicken auf die Taste "Datei auswählen". Beachten Sie bitte, dass Dateipfade relativ sind: Sie beginnen in dem Ordner, der das Dokument enthält, das Sie bearbeiten, also nicht auf der obersten Ebene Ihrer Festplatte. Klicken Sie in die Taste "Öffnen", um die Datei bzw. die URL-Adresse im Textfeld zu öffnen. Wird in das betreffende Objekt geklickt, wird die Datei bzw. URL-Adresse geöffnet.
Wenn Sie "Skript ausführen" auswählen, wird ein Textfeld zur Eingabe eines AppleScript-Skripts angezeigt. Bei der Ausführung des Skripts wird "self" als Verweis auf das Objekt interpretiert, in das geklickt wurde. Klicken Sie in "Syntax prüfen", um sicherzustellen, dass Ihr AppleScript-Skript keine Fehler enthält. Klicken Sie anschließend in "Skript ausführen", um das Skript zu testen. Wird in das betreffende Objekt geklickt, wird das Skript gestartet.
Wenn Sie "Zu anderer Position wechseln" auswählen, wird ein weiteres Einblendmenü angezeigt, in dem Sie auswählen können, zu welcher Position im aktuellen Dokument gewechselt werden soll. Sie können zu einer bestimmten oder zur vorherigen bzw. nächsten Arbeitsfläche wechseln oder zu einem bestimmten Punkt oder Objekt auf einer beliebigen Arbeitsfläche. Für einige dieser Optionen steht eine Miniaturdarstellung der Arbeitsfläche zur Verfügung, in der Sie durch Klicken oder Bewegen festlegen können, welches Objekt hervorgehoben, auf welchen Punkt zentriert oder an welche Stelle gezoomt werden soll.
Durch Auswahl von "Ebenen anzeigen oder ausblenden" können Sie angeben, ob Ebenen der aktuellen Arbeitsfläche angezeigt oder ausgeblendet werden sollen bzw. die Einstellung umschalten.
