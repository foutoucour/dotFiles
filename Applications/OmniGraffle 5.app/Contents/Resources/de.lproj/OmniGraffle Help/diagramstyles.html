Diagrammstile verwenden

Diagrammstile sind OmniGraffle-Dateien, die ein Erscheinungsbild definieren, das auf eine Gliederung oder ein Diagramm angewendet werden kann. Es gibt nahezu unendlich viele Möglichkeiten zum Darstellen identischer Daten. Mit Diagrammstilen werden die Darstellungsformen von Daten definiert.
Die wichtigsten Situationen, in der Diagrammstile zum Einsatz kommen, sind beim Erstellen einer Gliederung in der Gliederungsdarstellung von OmniGraffle (mit dem Befehl "Diagrammstil wählen" im Menü "Format") oder beim Importieren einer OmniOutliner-Datei (mit dem Dialogfenster "Import"). In diesen Fällen wandelt der Diagrammstil Textobjekte in Grafiken um. OmniGraffle wird mit mehreren Diagrammstilen geliefert. Probieren Sie verschiedene Diagrammstile aus, indem Sie sie auf eine Gliederungsdarstellung anwenden und sich deren Wirkung ansehen.
Beim Anwenden eines Diagrammstils werden die Objekte der Gliederung an die Einstellungen für Formen, Verknüpfungslinien und automatischen Layout des Diagrammstils angepasst. Objekte auf allen Ebenen der Gliederung werden zu Formobjekten, die Objekten auf der gleichen Ebene im Diagramm entsprechen. Wenn die Gliederung mehr Ebenen besitzt als der Diagrammstil, verwenden die weiter unten liegenden Objekte den Stil der untersten Ebene des Diagrammstils.
Zum Erstellen Ihres eigenen Diagrammstils wählen Sie "Neue Ressource ▸ Neuer Diagrammstil" aus dem Menü "Ablage", und wählen Sie anschließend eine Vorlage als Ausgangspunkt aus. Richten Sie ein Diagramm mit einer Arbeitsfläche und einer Ebene ein, das einige verknüpfte Objekte enthält und gestalten Sie es Ihren Wünschen entsprechend. Ein Diagrammstil sollte eine starre Baumstruktur aufweisen, d. h., die Verknüpfungslinien sollten eine Hierarchie ohne Rückkopplungen bilden. Speichern Sie es, wenn es Ihren Wünschen entspricht. Der Standardspeicherort ist Ihr Ordner Application Support, in dem OmniGraffle nach Ihren Diagrammstilen sucht. Sobald ein Diagrammstil in diesem Ordner gespeichert wurde, wird er in der Diagrammstilauswahl verfügar.
Wenn Sie OmniOutliner-Dateien mit mehreren Spalten importieren wollen, können Sie einen Diagrammstil mit einer Gruppe von Formen statt nur einer Form erstellen, um jedes Objekt darzustellen. In diesem Fall empfiehlt es sich, jedes Objekt mit einem Textetikett zu versehen, damit Sie diese beim Öffnen der OmniOutliner-Datei den Spalten zuordnen können.
