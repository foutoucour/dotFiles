Menü "Anordnen"

"Ganz nach vorne/ Schrittweise nach vorne": Objekte auf einer Ebene haben eine ganz bestimmte Reihenfolge: neu erstellte Objekte erscheinen vor bereits bestehenden Objekten. Wählen Sie "Ganz nach vorne", um die ausgewählten Objekte vor alle anderen Objekte auf der Ebene zu stellen. Halten Sie die Wahltaste gedrückt, wenn stattdessen der Befehl "Schrittweise nach vorne" verwendet werden soll. Auf diese Weise werden die ausgewählten Objekte jeweils einen Schritt nach vorne bewegt.
"Ganz nach hinten/ Schrittweise nach hinten": Funktioniert genauso wie "Ganz nach vorne", nur dass die ausgewählten Objekte hinter die anderen Objekte auf der Ebene gestellt werden. Halten Sie die Wahltaste gedrückt, wenn stattdessen der Befehl "Schrittweise nach hinten" verwendet werden soll. Auf diese Weise werden die ausgewählten Objekte jeweils einen Schritt nach hinten bewegt.
Schützen: Schützt die ausgewählten Objekte. Ein geschütztes Objekt kann weder bewegt noch bearbeitet werden. Wenn Sie versuchen, ein geschütztes Objekt auszuwählen, werden anstelle der Aktivpunkte X-Zeichen angezeigt. Die einzige Aktion, die Sie für ein geschütztes Objekt ausführen können, ist die Aufhebung des Schutzes. Ausnahmen bilden weiterhin die Möglichkeiten, das Etikett eines Objekts und seine Position in der Hierarchie der Gliederungsdarstellung zu ändern. Das Schützen ist nützlich, wenn Sie unabsichtliche Änderungen an Objekten verhindern wollen, die bereits optimal sind.
Schutz aufheben: Heben Sie für die ausgewählten Objekte den Schutz auf, damit sie bewegt oder bearbeitet werden können.
Gruppieren: Verwenden Sie diesen Befehl mit mindestens zwei ausgewählten Objekten, um sie in eine Gruppe zusammenzufassen, sodass sie sich wie ein einzelnes Objekt verhalten.
Gruppierung löschen: Wenn Sie eine Gruppe ausgewählt haben, können Sie die Gruppierung der einzelnen Objekte mit diesem Befehl aufheben.
Tabelle erstellen: Wählen Sie ein Objekt aus und wandeln Sie es mit diesem Befehl in eine 1x1-Tabelle um, zu der Sie anschließend Zeilen und Spalten hinzufügen können.
Als Unterdiagramm gruppieren: Wandeln Sie die ausgewählten Objekte in ein eigenständiges Unterdiagramm um, das Sie anschließend zu einem einzelnen Objekt reduzieren können.
Unterdiagramm reduzieren: Ändern Sie das ausgewählte Unterdiagramm zu einem einzelnen Objekt, das alle Verbindungslinien hinein oder hinaus behält.
Unterdiagramm erweitern: Stellt ein reduziertes Unterdiagramm wieder auf seine volle Größe um.
Hilfslinien
OmniGraffle stellt automatische und manuelle Hilfslinien zur Verfügung. Automatische Hilfslinien sind dünne blaue Linien, die angezeigt werden, wenn Sie Objekte bewegen, vergrößern, verkleinern oder drehen. Sie unterstützen Sie dabei, Objekte auszurichten und in einem gleichmäßigen Abstand zu positionieren.
Manuelle Hilfslinien sind arbeitsflächenüberspannende magentafarbene Linien, die Sie selbst anlegen. Sie können so viele dieser Hilfslinien einfügen, wie Sie möchten. Sie können an ihnen Objekte ausrichten und sie über das Menü "Darstellung" ein- oder ausblenden.
Automatische Hilfslinien für Ausrichtung aktivieren/deaktivieren: Diese Hilfslinien erleichtern das Ausrichten der Kanten oder Mittelpunkte von Objekten. Außerdem unterstützen sie Sie dabei, die Rotation von Objekten anzugleichen, wenn Sie bei gedrückter Befehlstaste die Aktivpunkte eines Objekts bewegen.
Automatische Hilfslinien für Abstand aktivieren/deaktivieren: Diese Hilfslinien erleichtern das Setzen gleichmäßiger Abstände zwischen Objekten.
Hilfslinien schützen/freigeben: So lange manuelle Hilfslinien geschützt sind, können sie nicht bewegt werden.
Neue horizontale/vertikale Hilfslinie: Mit dieser Option erhält die Arbeitsfläche eine neue Hilfslinie. Wenn die Hilfslinien nicht geschützt sind, können Sie das Auswahlwerkzeug verwenden, um sie nach Belieben an eine neue Position zu bewegen. Wenn Sie ein Objekt nah an eine Hilfslinie bewegen, werden seine Kanten oder sein Mittelpunkt automatisch an der Hilfslinie ausgerichtet. Sie können auch neue Hilfslinien aus den vertikalen und horizontalen Linealen ziehen.
Manuelle Hilfslinien löschen: Löschen Sie alle manuellen Hilfslinien von der Arbeitsfläche. Falls Sie sie vorübergehend ausblenden möchten, können Sie dies über das Menü "Darstellung" tun.
Ausrichten: Diese Befehle haben die Funktionalität der Schaltflächen im Informationsfenster "Ausrichtung", um die Objekte an den Kanten und Mittelpunkten auszurichten oder einen gleichmäßigen Abstand zwischen Objekten einzustellen.
Raster: "Automatisch am Raster ausrichten" und " Objekte am Raster ausrichten " funktionieren hier auf die gleiche Weise wie im Informationsfenster "Raster". Mit der Option " Kanten an Raster" wird die Größe von Objekten so angepasst, dass deren Kanten sich an den nächstgelegenen Rasterlinien ausrichten.
Diagrammlayout: Mit diesen Befehlen wird das Layout aller Objekte bzw. aller ausgewählten Objekte festgelegt, und zwar gemäß den Einstellungen im Informationsfenster "Diagrammlayout".
Größe: Wenn mehrere Objekte ausgewählt sind, können Sie für sie "Gleiche Höhe", "Gleiche Breite" oder beides auswählen. Mit der Option "Höhe = Breite" wird ein Objekt so angepasst, dass es ebenso hoch wie breit ist. Bei Objekten mit einem Bild, das im Informationsfenster "Bild" angezeigt wird, können Sie mithilfe der Option "Größe an Bild anpassen" die Objektgröße so ändern, dass das Bild in Originalgröße angezeigt wird.
