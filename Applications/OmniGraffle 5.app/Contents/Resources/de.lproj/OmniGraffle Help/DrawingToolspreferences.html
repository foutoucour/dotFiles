Ändern, wie Werkzeuge mit den Voreinstellungen für Zeichenwerkzeuge zusammenarbeiten

Einstellungen für Zeichenwerkzeuge stehen auf dem zweiten Voreinstellungsfenster zur Verfügung und können durch Auswahl von "Voreinstellungen" aus dem OmniGraffle-Menü aufgerufen werden.
Sie können Werkzeuge in der Liste verschieben, um ihre Reihenfolge in der Palette zu ändern. Verschieben Sie die Trennlinie, um zu ändern, wo die Erweiterungsschaltfläche in der Symbolleiste angezeigt wird. Verschieben Sie sie zum Ende der Liste, um die Schaltfläche ganz auszublenden.
Wählen Sie den Kurzbefehl eines Werkzeugs durch Doppelklicken aus, um diesen zu ändern. Ein Kurzbefehl aktiviert ein Werkzeug lediglich so lange Sie die Taste gedrückt halten. Wenn Sie die Taste loslassen, wird wieder das zuvor aktive Werkzeug aktiviert.
Details eines Werkzeugs anzeigen
Standardmäßig müssen Sie zum Anzeigen von Informationen zu Werkzeugen bei gedrückter Wahltaste in das Werkzeug klicken. Sie können auch festlegen, dass hierfür nur einfache Mausklicks erforderlich sind.
Tool aktiviert lassen
Standardmäßig können Sie ein Werkzeug mit einem Mausklick zeitweise aktivieren und mit zwei Mausklicks dauerhaft aktivieren. Sie können einstellen, dass Werkzeuge schon mit einem Mausklick dauerhaft aktiviert werden oder auch jegliche Werkzeugauswahl nur als zeitweise Auswahl definieren.
Linienerstellung
Wenn Sie eine Linie erstellen, wird die Linie durch Klicken in ein Objekt immer mit diesem Objekt verknüpft und endet dort. Sie können festlegen, ob beim Klicken in einen leeren Bereich der Arbeitsfläche die Linie dort beendet oder einfach ein Punkt hinzugefügt werden soll, sodass Sie mit dem Erstellen der Linie fortfahren können. Legen Sie in dieser Einstellung fest, welche Aktion ausgeführt werden soll.
Linienbearbeitung
Sie können festlegen, ob beim Auswählen einer Linie durch Doppelklicken ein neuer Punkt oder ein Etikett zur Linie hinzugefügt werden soll. Wählen Sie eine der Möglichkeiten aus. Wenn Sie bei gedrückter Wahltaste auf eine Linie doppelklicken, wird die andere Aktion ausgeführt.
