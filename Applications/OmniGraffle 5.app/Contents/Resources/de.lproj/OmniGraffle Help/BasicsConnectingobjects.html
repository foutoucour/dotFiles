Grundlagen: Objekte mit Linien verknüpfen

Sie können zwei beliebige Objekte mit dem Linienwerkzeug () verknüpfen.
Klicken Sie in das Linienwerkzeug. (Abhängig vom gewählten Stil sieht dieses Werkzeug unter Umständen anders aus.)
Klicken Sie in das erste Objekt, das verknüpft werden soll. Dieses Objekt wird zur "Quelle" der Linie.
Klicken Sie in das zweite Objekt, das verknüpft werden soll. Dieses Objekt wird zum "Ziel" der Linie.
Sobald Objekte durch eine Linie verknüpft sind, bleiben sie verknüpft, egal, wie sehr sie geändert oder bewegt werden. Folglich bleibt der logische Aufbau Ihres Diagramms erhalten, unabhängig davon, wie Sie es optisch gestalten. Sie können eine Linie von einem Objekt trennen, indem Sie die Linie auswählen und dann einen der Endpunkte ziehen.
