Daten mit dem Informationsfenster "Notizen" zu Objekten hinzufügen

Das Informationsfenster "Notizen" enthält ein Feld zur Eingabe von Text, der dem ausgewählten Objekt zugewiesen werden soll, sowie eine Tabelle mit angepassten Daten.
Die Notiz kann als formatierter Text (RTF) angezeigt werden, sodass Sie alle in OmniGraffle verfügbaren Schriftstile und Farben verwenden können. Wenn Sie den Mauszeiger über ein Objekt bewegen, dem eine Notiz zugewiesen ist, wird diese Notiz als Zusatzinformation über dem Objekt angezeigt.
In der Tabelle mit den angepassten Daten können Sie Ihre eigenen Informationen zum Objekt eintragen. Angepasste Daten werden als Schlüssel/Wert-Paare gespeichert: der Schlüssel ist eine Bezeichnung für die Art der gespeicherten Daten, und der Wert gibt die Daten selbst an.
Angenommen, Sie haben ein Diagramm eines Computernetzwerks und wollen jeder Komponente eine Modellnummer zuweisen. Klicken Sie auf die Platzhalterzeile oder das Plus-Zeichen, um ein neues Schlüssel/Wert-Paar zu erstellen. Geben Sie in die Spalte "Schlüssel" "Teilenummer" und in das Feld "Wert" die Eingabe A1181 ein. Diese Daten haben keinen Einfluss auf die Arbeitsweise von OmniGraffle. Sie dienen lediglich zum Speichern beliebiger Daten über Objekte in Ihrem Diagramm. Zum Löschen einer Zeile mit Daten klicken Sie auf das 'x' auf der rechten Seite der Zeile.
Sie können auch die Notiz der Arbeitsfläche selbst bearbeiten. Klicken Sie hierzu einfach auf die Arbeitsfläche in der Seitenleiste (um diesen in der ausgewählten Farbe hervorzuheben) und verwenden Sie dann das Informationsfenster "Notiz" wie gewohnt.
Notizen können mit der Mac OS X-Suchfunktion "Spotlight" gefunden werden, so dass Sie damit nach OmniGraffle-Diagrammen suchen können, die bestimmte Wörter enthalten.
