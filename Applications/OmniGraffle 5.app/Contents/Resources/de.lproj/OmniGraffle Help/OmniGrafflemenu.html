Menü "OmniGraffle"

Über OmniGraffle: Weitere Informationen zu der verwendeten Version von OmniGraffle.
Aktualisierungen suchen: Normalerweise sucht OmniGraffle selbstständig nach aktualisierten Versionen des Programms, basierend auf den Aktualisierungseinstellungen. Wählen Sie diesen Befehl aus, um sofort nach Aktualisierungen zu suchen. Wenn eine aktualisierte Version der Software verfügbar ist, werden Sie zum Herunterladen dieser Version aufgefordert.
Einstellungen: Öffnen Sie das OmniGraffle Fenster "Einstellungen" und passen Sie das Programm damit an Ihre Anforderungen an.
Lizenzen: Zeigen Sie das Dialogfenster "Lizenzen" an, in dem Sie Ihre OmniGraffle-Softwarelizenzen überprüfen können. Weitere Informationen zur Funktionsweise von Lizenzen erhalten Sie im Abschnitt Lizenzierung.
Dienste: Das Menü "Dienste" ist eine Sammlung systemweiter Dienste, die von Ihren installierten Programmen bereitgestellt werden. OmniGraffle ist sehr gut mit Mac OS X kompatibel, sodass Sie Ihre Standarddienste im Normalfall von diesem Programm aus verwenden können.
OmniGraffle ausblenden: Lassen Sie OmniGraffle geöffnet, blenden Sie jedoch die dazugehörigen Fenster zeitweise aus, während Sie in einem anderen Programm arbeiten.
Andere ausblenden: Blenden Sie alle Fenster anderer geöffneter Programme aus, sodass Sie sich auf OmniGraffle konzentrieren können.
Alle anzeigen: Zeigen Sie alle Fenster aller Programme an, die Sie zuvor mit den Befehlen zum Ausblenden ausgeblendet haben.
OmniGraffle beenden: Wählen Sie diese Option, wenn Sie OmniGraffle beenden möchten. Sind nicht gesicherte Dokumente geöffnet, werden Sie aufgefordert, Ihre Änderungen zu sichern.
