Mit den allemeinen Programmeinstellungen die Funktionsweise von OmniGraffle ändern

Die allgemeinen Programmeinstellungen können durch Auswahl von Einstellungen im OmniGraffle-Menü aufgerufen werden.
Start
Wählen Sie Dokumente öffnen, die beim letzten Beenden offen waren, wenn sich OmniGraffle die zuletzt geöffneten Dokumente "merken" soll.
Wählen Sie "Neues Dokument erstellen, wenn noch keines geöffnet ist", um mit einem neuen Dokument zu beginnen, wenn OmniGraffle keine zuletzt geöffneten Dokumente anzeigen soll oder vor dem letztmaligen Schließen keine Dokumente geöffnet waren.
Neue Dokumente
Wählen Sie Vorlagenauswahl verwenden, wenn jedesmal, wenn Sie ein neues Dokument erstellen, ein Fenster zur Auswahl einer Vorlage angezeigt werden soll. Wählen Sie Vorlage verwenden und wählen Sie Ihre Standardvorlage, wenn Sie das Auswahlfenster umgehen möchten. Durch Auswahl von Vorlagenauswahl aus dem Menü Ablage können Sie in jedem Fall ein Auswahlfenster öffnen.
Text
Legen Sie fest, welche Funktionen mit der Eingabetaste und der Tabulatortaste ausgeführt werden sollen, wenn Sie den Text eines Objekts bearbeiten. Unabhängig davon, für welche Möglichkeit Sie sich entscheiden, können Sie beim Drücken dieser Tasten die Wahltaste gedrückt halten, um die andere Aktion auszuführen.
Auswahl
Kleine Objekte erhalten kleinere Versionen der Aktivpunkte, die bei Auswahl der Objekte angezeigt werden. Falls Ihnen die kleinen Aktivpunkte gut gefallen, können sie die Option Immer kleine Aktivpunkte verwenden wählen, dann werden immer die kleinen Aktivpunkte angezeigt.
Beim Klicken in ein Objekt, das nicht mit einem Bild oder einer Farbe ausgefüllt ist, gibt es zwei Möglichkeiten: das Objekt oder das darunter liegende Element wird ausgewählt. Mit der Option Bei Objekten ohne Füllfarbe darunter liegende Objekte durch Hindurchklicken auswählen können Sie festlegen, was ausgewählt werden soll.
Zusatzinfos
Wenn Sie ein Objekt bewegen oder vergrößern bzw. verkleinern, werden in einem kleinen Feld Zusatzinformationen mit der genauen neuen Position und Größe des Objekts angezeigt. Das heißt, solange Sie diese Zusatzinfos hier nicht deaktivieren, da Sie sie als störend empfinden.
Sichern
OmniGraffle kann in regelmäßigen Abständen eine Sicherungskopie Ihrer Dateien anlegen, während Sie damit arbeiten. Sollte es zu einem Ausfall kommen, können Sie Ihre Arbeit dadurch teilweise wieder herstellen.
Listen-Seitenleiste
Wenn Sie mit AppleScript auf OmniGraffle zugreifen oder aus einem anderen Grund eindeutige IDs benötigen, können Sie durch Auswahl von Objektidentifizierungsnummern anzeigen eine Spalte zur Listen-Seitenleiste hinzufügen.
Taste "Zurücksetzen" in den Programmeinstellungen
In jedem der Bereiche für die Programmeinstellungen befindet sich unten links die Taste "Zurücksetzen". Wenn Sie in diese Taste klicken, werden alle Einstellungen für den betreffenden Bereich auf die Standardwerte zurückgesetzt. Wenn Sie beim Klicken in die Taste "Zurücksetzen" die Wahltaste gedrückt halten, werden alle Einstellungen für alle Bereiche zurückgesetzt.
