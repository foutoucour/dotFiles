Support für AppleScript

OmniGraffle bietet umfassenden Support für AppleScript. Für weitere Informationen zum Objektmodell und zu den Befehlen von OmniGraffle öffnen Sie das OmniGraffle-Wörterbuch im Script Editor.
Sie können AppleScript-Skripte im Skripteordner von OmniGraffle speichern, sodass über das Menü "Skripte" besonders einfach darauf zugegriffen werden kann. Mithilfe des Informationsfensters "Aktion" können Sie Skripte auch an einzelne Objekte auf einer Arbeitsfläche anhängen. Wenn Sie das Aktionenwerkzeug aktivieren und auf ein Objekt mit angehängtem Skript klicken, wird das Skript ausgeführt.
Wenn Sie einige Objekte auswählen und dann "Kopieren als AppleScript" aus dem Menü "Bearbeiten" wählen, erhalten Sie ein Skript für AppleScript, das genau diese Objekte erstellt. Auf diese Weise können Sie ganz leicht lernen, wie der Support für AppleScript in OmniGraffle funktioniert oder spezifische Objekte zu Ihren eigenen Skripten hinzufügen.
