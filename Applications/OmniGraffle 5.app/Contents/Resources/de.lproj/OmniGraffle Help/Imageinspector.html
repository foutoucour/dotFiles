Grafiken mit dem Informationsfenster "Bild" in eine Form einfügen

Mithilfe dieses Informationsfensters können Sie ein Bild in einem Formobjekt platzieren. (Anstatt ein Objekt zu erstellen und anschließend ein Bild hinzuzufügen, können Sie auch einfach eine Bilddatei aus einem anderen Programm direkt auf die Arbeitsfläche bewegen.)
Verwenden Sie das Einblendmenü "Bild", um ein Bild in die ausgewählte Form einzufügen. Sobald ein Bild eingefügt wurde, erscheint in dem Menü die Option Bild entfernen.
Wenn Sie nach einer Bilddatei suchen, können Sie eine Aliasdatei (eine Veknüpfung) erstellen, die auf das Bild verweist, anstatt das Bild selbst zur OmniGraffle-Datei hinzuzufügen. In diesem Fall verwendet das OmniGraffle-Dokument die ursprüngliche Bilddatei als Grundlage. Die Aliasdatei funktioniert nicht, wenn Sie die Bilddatei löschen oder das Dokument auf einem anderen Computer öffnen. Wenn das Bild eines Objekts aus einem Alias stammt, werden im Einblendmenü "Bild" einige weitere Befehle verfügbar: Sie können die Original-Bilddatei öffnen, die Original-Bilddatei im Finder einblenden oder nicht länger das Alias verwenden, sondern statt dessen eine Kopie des Bilds im OmniGraffle-Dokument einbetten.
Sie können ein Bild auch einfügen, indem Sie eine Bilddatei aus einer anderen Anwendung ziehen und in der Bildquelle oder auf dem Formobjekt selbst ablegen.
Die erste Schaltfläche zeigt das Bild mit seinem normalen Seitenverhältnis an, wobei Zoomebene und Position mit den Steuerelementen darunter festgelegt werden. Die zweite Schaltfläche dehnt das Bild, damit es in das Objekt passt. Die dritte Schaltfläche kachelt das Objekt unendlich über dem Objekt.
Die Positionierungs-Offsets sind aktiviert, wenn Sie das Bild mithilfe der ersten der drei Schaltflächen mit dem normalen Seitenverhältnis anzeigen. Sie können einen positiven oder negativen Prozentsatz eingeben oder das Bild einfach auf eine andere Stelle in der Bildquelle bewegen, um den gewünschten Bereich anzugeben.
Die Zoom-Steuerungen sind verfügbar, wenn das Bild nicht in die Form passend gedehnt wird. Sie können einen Zoom-Prozentsatz eingeben oder den Schieber bewegen, um das Bild größer oder kleiner zu machen.
Sie können das Bild transparent machen und die Füllfarbe des Objekts durchscheinen lassen, indem Sie den Wert des Felds Deckkraft reduzieren oder den Deckkraftregler verschieben.
Sie können auch ein Bild zu einer Arbeitsfläche hinzufügen. Klicken Sie hierzu einfach auf die Arbeitsfläche in der Seitenleiste (um diese in der ausgewählten Farbe hervorzuheben) und verwenden Sie dann das Informationsfenster "Bild" wie gewohnt.
