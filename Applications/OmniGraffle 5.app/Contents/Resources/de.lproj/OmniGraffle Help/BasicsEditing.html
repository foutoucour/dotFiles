Grundlagen: Objekte bearbeiten

Jedes Attribut eines Objekts kann mithilfe der Informationsfenster "Stil" und "Eigenschaften" geändert werden: ihre Größe, Form, Farbe, Position usw. Bei Auswahl eines Objekts werden die entsprechenden Informationsfenster verfügbar. Verwenden Sie das Informationsfenster für das zu ändernde Attribut. Wenn Sie die Einstellungen im Informationsfenster ändern, verändert sich das Objekt auf der Arbeitsfläche.
Gehen Sie wie folgt vor, wenn Sie z.B. die Farbe in einem Formobjekt (die Füllfarbe) ändern wollen:
Wählen Sie das Objekt aus.
Wenn das Informationsfenster "Füllfarben" nicht angezeigt wird, wählen Sie "Füllfarben" aus dem Menü "Informationsfenster" aus. Klicken Sie in das Farbfeld oben rechts im Informationsfenster "Füllfarben".
Verwenden Sie die Farbpalette, um eine neue Füllfarbe für das Objekt zu wählen.
Das Objekt erhält eine neue Füllfarbe! Juhu!!
