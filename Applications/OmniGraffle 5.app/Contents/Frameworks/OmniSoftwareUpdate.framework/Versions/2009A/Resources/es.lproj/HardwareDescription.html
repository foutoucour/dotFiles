Cuando ${APP} busca versiones nuevas o actualizaciones de software, puede incluir opcionalmente información básica sobre la configuración de su sistema. Gracias a esta información, podemos saber qué sistemas utilizan nuestros usuarios y determinar cuáles de ellos necesitan un mayor soporte.

Omni no revelará nunca información sobre la configuración de los ordenadores de los usuarios, sino sólo datos estadísticos sobre las configuraciones estudiadas. Le garantizamos el máximo respeto hacia su privacidad.

Si prefiere no enviarnos información sobre su sistema, sólo tiene que anular la opción correspondiente en el panel Actualización de las preferencias de esta aplicación.

Las estadísticas que realizamos se hacen públicas para que también otros desarrolladores puedan beneficiarse de esta información. Si desea ver la información que publicamos, visite la página web de actualización de software de Omni.

Los datos siguientes se envían siempre junto con la consulta:

Versión de OmniSoftwareUpdate ${OSU_VER}
ID de la aplicación ${OSU_APP_ID}
Versión de la aplicación ${OSU_APP_VER}
Pista de actualización ${OSU_TRACK}
Pistas de actualización visibles ${OSU_VISIBLE_TRACKS}

Si está activada la opción correspondiente en el panel Actualización de las preferencias de la aplicación, se enviarán además los datos siguientes:

${OTHERVARS}
UUID ${uuid}
Una secuencia de caracteres aleatoria que identifica de forma anónima su ordenador. Este identificador nos permite mantener actualizadas nuestras bases de datos cuando cambie la configuración de su ordenador. Este dato no está relacionado con su nombre ni con ninguna otra información personal.
Tipo de licencia ${license-type}
La clave de su licencia no se enviará nunca.
Versión de OS ${os}
Idioma preferido ${lang}
Ubicación general Latitud: ${LATITUDE} Longitud: ${LONGITUDE-DMS}
Esta ubicación general está basada en la ciudad más cercana que seleccionó en la aplicación Preferencias del Sistema al ajustar la zona horaria durante la configuración del ordenador. Puede ver exactamente la ubicación que se envía en MapQuest.
Modelo de ordenador ${machine_name} (${hw-model})
Unidades de CPU ${ncpu}
Tipo de CPU ${cpu_type} (${cpu})
Velocidad de la CPU ${cpuhz}
Velocidad de bus ${bushz}
Memoria ${mem}
Configuración de pantalla ${DISPLAYS}
Tarjetas de vídeo ${VIDEO}
