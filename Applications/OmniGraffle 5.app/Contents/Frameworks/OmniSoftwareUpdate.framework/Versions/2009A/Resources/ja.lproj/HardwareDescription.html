${APP} の新しいバージョンまたはアップデートを確認する際、ご使用マシンのシステム構成に関する基本的な情報の一部を送信することができます。私たちはこのデータを使用し、ユーザのみなさまがどのようなシステム構成を使用しているかといった情報を収集し、どのようなシステム構成を最優先にサポートするべきかを判断する材料にさせていただいております。

Omni Group は、個人のコンピュータ構成に関する情報を決して漏洩いたしません — 収集した情報は統計的に処理されるだけです。私たちは、誠実にユーザのみなさまのプライバシーを尊重いたします。

システム情報は送信しなくても問題ありません ー システム構成に関する情報の送信を希望しない場合は、本アプリケーションの Omni ソフトウェアアップデート環境設定画面でチェックボックスをオフにしてください。

ここで収集される情報はまた他の開発者にとっても貴重です。私たちは統計処理を行いこれを公開しています。Omni Software Update ページへアクセスすると、公開している情報をご覧いただけます。

以下の情報はクエリとともに常に送信されます：

OmniSoftwareUpdate バージョン ${OSU_VER}
アプリケーション ID ${OSU_APP_ID}
アプリケーションのバージョン ${OSU_APP_VER}
アップデートトラック ${OSU_TRPDFtd>
表示できるアップデートトラック ${OSU_VISIBLE_TRACKS}

「アップデート」環境設定で送信が有効になっている場合、以下の追加情報がクエリとともに送信されます：

${OTHERVARS}
UUID ${uuid}
このランダムな文字列は匿名性を保ちながらあなたのコンピュータを識別するものです。これにより、あなたのシステム構成が変更された場合、私たちのデータベースに更新されます。これは個人名をはじめとする個人情報に何ら結びつくものではありません。
ライセンス種類 ${license-type}
ライセンスキーそのものは一切送信されません。
OS のバージョン ${os}
優先する言語 ${lang}
大まかな場所 緯度: ${LATITUDE-DMS} 経度: ${LONGITUDE-DMS}
「大まかな場所」とは、コンピュータのセットアップ時に設定した時間帯に従って、「システム環境設定」で主な都市の中から選択された最も近い都市の情報に基づきます。送信される場所の情報はMapQuestで確認できます。
コンピュータのモデル ${machine_name} (${hw-model})
CPU 数 ${ncpu}
CPU タイプ ${cpu_type} (${cpu})
CPU 速度 ${cpuhz}
Bus 速度 ${bushz}
メモリ ${mem}
ディスプレイ設定 ${DISPLAYS}
ネットワーク速度 ${qt_netspeed}
ビデオカード ${VIDEO}
OpenGL 情報 ${OPENGL}
起動回数と累計起動時間
現在のバージョンすべてのバージョン
${RUNTIME-Hours}${runmin}${trunmin}
${RUNTIME-Launches}${nrun}${tnrun}
${RUNTIME-Crashes}${ndie}${tndie}
