Lorsque ${APP} recherche des nouvelles versions et des mises à jour, il peut inclure certaines informations générales à propos de votre configuration système. Nous utilisons ces données pour déterminer quels sont les systèmes utilisés par nos clients et par conséquent pour quels systèmes il est le plus important de fournir une assistance.

Omni ne publiera jamais des informations sur la configuration informatique d'un individu.Il ne fournira que des informations à caractère statistique sur l'ensemble des configurations. Nous respectons entièrement la confidentialité de vos données personnelles.

Si vous préférez ne pas envoyer vos informations système, ce n'est pas un problème.Il vous suffit de ne pas activer la case de la sous-fenêtre des préférences de la mise à jour des logiciels Omni dans cette application.

Nous publions les statistiques afin de permettre à d'autres développeurs de bénéficier également de ces connaissances. Vous pouvez visualiser les informations que nous publions en vous rendant sur la page de mise à jour des logiciels Omni.

Les renseignements suivants sont toujours envoyés avec la requête :

Version de la mise à jour des logiciels Omni ${OSU_VER}
Identifiant de l'application ${OSU_APP_ID}
Version de l'application ${OSU_APP_VER}
Piste de mise à jour ${OSU_TRACK}
Pistes de mise à jour visibles ${OSU_VISIBLE_TRACKS}

Si l'option correspondante est activée dans les préférences de mise à jour, les informations suivantes supplémentaires seront envoyées avec la requête :

${OTHERVARS}
UUID ${uuid}
Chaîne aléatoire qui identifie de manière anonyme votre ordinateur, ce qui nous permet de maintenir à jour notre base de données lorsque votre configuration système change. Cette chaîne n'est pas associée à votre nom ou à d'autres informations personnelles.
Type de licence ${license-type}
Votre clé de licence elle-même ne sera jamais envoyée.
Version OS ${os}
Langue préférée ${lang}
Position générale Latitude : ${LATITUDE-DMS} Longitude : ${LONGITUDE-DMS}
Cette position générale est basée sur la grande ville la plus proche sélectionnée par l'application des Préférences Système lorsque vous avez choisi un fuseau horaire en configurant votre ordinateur. Vous pouvez visualiser la position exacte rapportée sur le site Web suivant :MapQuest.
Modèle d'ordinateur ${machine_name} (${hw-model})
Nombre de processeurs ${ncpu}
Type de processeur ${cpu_type} (${cpu})
Vitesse de processeur ${cpuhz}
Vitesse de bus ${bushz}
Mémoire ${mem}
